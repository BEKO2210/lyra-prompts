## 🎯 Was ist Lyra?

**Lyra** ist ein bewährtes Framework für präzise, wirksame AI-Prompts. Jeder Prompt in dieser Sammlung folgt der **4-D Methodology**:

```
┌─────────────┐  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
│  1. ROLLE   │→ │ 2. KONTEXT  │→ │  3. AUFGABE │→ │ 4. AUSGABE  │
│   (Wer?)    │  │   (Was?)    │  │  (Wie?)     │  │  (Ergebnis) │
└─────────────┘  └─────────────┘  └─────────────┘  └─────────────┘
```

> 💡 **Kein Raten mehr.** Keine vagen Anfragen. Nur präzise Ergebnisse.

👉 **[Jetzt auf der Website ausprobieren](https://beko2210.github.io/lyra-prompts/)**

### Für Anfänger

1. **Besuche die [Lyra Prompts Website](https://beko2210.github.io/lyra-prompts/)**

## 📋 Prompt-Struktur

Jeder Prompt ist eine Markdown-Datei mit **Frontmatter**:

```yaml
---
id: "#001"
titel: "Rezept anpassen"
kategorie: "Alltag & Leben"
unterkategorie: "Küche & Kochen"
tags: ["kochen", "rezept", "meal-prep"]
erstellt: "2026-02-20"
plattformen: [ChatGPT, Claude]
---
```

**Inhalt:**
- 📝 Der komplette Prompt (kopierbar)
- 💡 Anwendungsbeispiele
- 🔄 Variationen für verschiedene Situationen
- ⚙️ Parameter (für Midjourney, etc.)

## 🤝 Mitmachen

Hast du einen guten Prompt? **Füge ihn hinzu!**

### Schritt-für-Schritt

1. **Fork** das Repository
2. **Lies [CREDITS.md](./CREDITS.md)** – prüfe ob ein ähnlicher Prompt existiert
3. **Erstelle eine neue Datei** in der passenden Kategorie
4. **Folge dem Template:**

```
---
id: "#XXX"
titel: "Dein Prompt-Titel"
kategorie: "Kategorie Name"
unterkategorie: "Unterkategorie"
tags: ["tag1", "tag2", "tag3"]
erstellt: "YYYY-MM-DD"
plattformen: [ChatGPT, Claude]
---

## Prompt


Rolle: ...
Kontext: ...
Aufgabe: ...
Ausgabe: ...
```

## Anwendung
...

## Variationen
...

```

5. **Trage den Prompt in CREDITS.md ein**
6. **Sende einen Pull Request**
