## <img src="https://media.giphy.com/media/j2pOGeGYKe2xCCKwfi/giphy.gif" width="30"> Quick Start

<details>
<summary><b>For Beginners — Click to expand</b></summary>
<br/>

1. **Visit the [Lyra Prompts Website](https://beko2210.github.io/lyra-prompts/)**
2. **Choose a category** that fits your needs
3. **Open a prompt** — each has a copyable template
4. **Fill in the [PLACEHOLDERS]** with your details
5. **Send to your AI** (ChatGPT, Claude, Gemini, etc.)
6. **Get professional results** instantly

</details>

<details>
<summary><b>For Developers — Click to expand</b></summary>
<br/>

```bash
# Clone the repository
git clone https://github.com/BEKO2210/lyra-prompts.git

# Browse prompts by category
ls kategorien/

# Search for specific topics
grep -r "tags:.*marketing" kategorien/

# Search for Pro prompts
ls kategorien/pro/
```

</details>

## <img src="https://media.giphy.com/media/cj87CxfRtrUifF3Riz/giphy.gif" width="25"> Prompt Structure

Every prompt is a Markdown file with **YAML frontmatter**:

```yaml
---
id: "#001"
titel: "Recipe Adaptation"
kategorie: "Everyday & Life"
unterkategorie: "Cooking"
tags: ["cooking", "recipe", "meal-prep"]
erstellt: "2026-02-20"
plattformen: [ChatGPT, Claude, Gemini]
---
```

**Each prompt contains:**

| Section | Description |
|:---|:---|
| 📝 **Prompt** | The complete, copy-ready prompt template |
| 💡 **Application** | Real-world use cases and examples |
| 🔄 **Variations** | Alternative versions for different scenarios |
| ⚙️ **Parameters** | Platform-specific settings (Midjourney, etc.) |

## <img src="https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif" width="25"> Contributing

Contributions are welcome! Here's how to add your prompts:

<details>
<summary><b>Contribution Guide — Click to expand</b></summary>
<br/>

1. **Fork** this repository
2. **Read [CREDITS.md](./CREDITS.md)** — check if a similar prompt already exists
3. **Create a new file** in the matching category folder under `kategorien/`
4. **Follow the Lyra 4-D template:**

```markdown
---
id: "#NEXT_ID"
titel: "Your Prompt Title"
kategorie: "Category Name"
unterkategorie: "Subcategory"
tags: ["tag1", "tag2", "tag3"]
erstellt: "YYYY-MM-DD"
plattformen: [ChatGPT, Claude]
---

## Prompt

Role: [Expert persona]
Context: [Situation + placeholders]
Task: [Precise instructions]
Output: [Desired format]

## Application
[Explain use cases]

## Variations
- Variation 1
- Variation 2
```

5. **Register in CREDITS.md** — assign the next available ID
6. **Submit a Pull Request**

</details>
