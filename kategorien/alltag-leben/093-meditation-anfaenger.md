# Meditation für Anfänger

## Prompt

```
Rolle: Zertifizierter Meditationslehrer
Kontext: Ich möchte mit Meditation beginnen
Meine Erfahrung: [NOCH NIE / WENIG / UNREGELMÄßIG]
Mein Ziel: [STRESS ABBAU / BESSERER SCHLAF / FOKUS / ACHTSAMKEIT]
Verfügbare Zeit: [MINUTEN PRO TAG]
Bevorzugte Zeit: [MORGENS / MITTAGS / ABENDS]
Räumliche Situation: [EIGENER RAUM / GETEILTER RAUM / UNTERWEGS]
Aufgabe: Erstelle einen Einstiegsplan für Meditation

Ausgabe:
1. Die richtige Technik für mein Ziel:
   - Atemmeditation
   - Body Scan
   - Fokussierte Aufmerksamkeit
   - Loving-Kindness
2. Schritt-für-Schritt Anleitung:
   - Körperhaltung
   - Atemtechnik
   - Was mit Gedanken machen?
   - Wie lange?
3. 14-Tage-Starter-Programm:
   - Tag 1-3: 5 Minuten
   - Tag 4-7: 10 Minuten
   - Tag 8-14: 15 Minuten
4. Häufige Anfängerprobleme:
   - "Ich kann nicht aufhören zu denken"
   - "Mir wird unruhig"
   - "Ich habe keine Zeit"
5. App-Empfehlungen (kostenlos/günstig)
6. Fortschritts-Tracking
7. Wann Meditationskurs/Klosterbesuch sinnvoll?
```

## Beispiel

**Input:** "Noch nie meditiert, Ziel: Stress abbauen, 10 Minuten/Tag, morgens, eigener Raum"

**Output:**
- Technik: Atemmeditation (einfachster Einstieg)
- Haltung: Auf Stuhl sitzend, Füße auf Boden
- 14-Tage-Plan mit steigender Dauer
- App-Tipp: Insight Timer (kostenlos)

## Variationen

- Für gestresste Eltern: "Mikro-Meditationen zwischendurch"
- Für Sportler: "Meditation für bessere Performance"
- Für Schichtarbeiter: "Meditation bei unregelmäßigen Zeiten"
- Für Skeptiker: "Meditation ohne Esoterik"
