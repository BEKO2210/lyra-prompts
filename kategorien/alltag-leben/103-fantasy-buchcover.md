# Fantasy Buchcover Design

## Prompt

```
Erstelle ein episches Fantasy-Buchcover im Stil großer Verlagshäuser.

GENRE:
- [HIGH FANTASY / DARK FANTASY / URBAN FANTASY / EPIC FANTASY]
- Zielgruppe: [YA / ADULT]
- Ton: [DUNKEL / HOFFNUNGSVOLL / MYSTERIÖS / ACTION-GELADEN]

HAUPTELEMENTE:
- Protagonist: [BESCHREIBUNG]
- Setting: [WELT/ORT]
- Zentrales Symbol/Objekt: [SCHWERT / AMULETT / STADT / KREATUR]
- Magie-System-Visualisierung: [FEUER / EIS / LICHT / SCHATTEN]

KOMPOSITION:
- [ZENTRALER CHARAKTER / LANDSCHAFT / SYMBOL]
- Perspektive: [FRONTAL / VON UNTEN / ÜBERSICHT]
- Tiefe: [VORDERGRUND / HINTERGRUND ELEMENTE]

FARBSCHEMA:
- Primär: [FARBE]
- Akzent: [FARBE]
- Kontrast: [HELL/DUNKEL]

STIL:
- Digitale Malerei mit fotorealistischen Texturen
- Dramatisches Licht (Rim-Light, volumetrisch)
- Partikel-Effekte (Magie, Staub, Feuer)
- Tiefe Schärfentiefe
- Filmische Komposition

TECHNISCHE SPEZIFIKATION:
- Format: Buchcover-Portrait (2:3 Ratio)
- 8K Auflösung
- Platz für Titel-Text (obere/mitlere Zone frei)
- Autorenname-Platz (unterer Bereich)
- Spine-Bereich berücksichtigen

Referenz-Stil: [WIE BEI: Brandon Sanderson, Sarah J. Maas, George R.R. Martin]
```

## Anwendung

**Für:** Self-Publishing-Autoren, Verlage, Konzeptkunst

**Beispiel-Output:** Dunkler Krieger mit leuchtendem Schwert, im Hintergrund brennende Stadt, Drachen-Silhouette am Himmel

## Variationen

- Für Romance-Fantasy: "Romantisches Fantasy-Cover mit Paar"
- Für Dark Fantasy: "Gruseliges Cover mit Schatten-Kreaturen"
- Für Urban Fantasy: "Moderne Stadt mit magischen Elementen"
- Für Serie: "Konsistente Cover-Reihe designen"
