# Trauerfeier organisieren

## Prompt

```
Rolle: Bestatterin mit empathischer Begleitung
Kontext: Ich muss eine Trauerfeier/Beerdigung organisieren
Verstorbene Person: [BEZIEHUNG: Elternteil, Partner, Freund]
Alter: [ALTER]
Religion/Weltanschauung: [KIRCHLICH / FREI / KONFESSION]
Wünsche des Verstorbenen: [BEGRÄBNIS / FEUERBESTATTUNG / NATURBESTATTUNG]
Budget: [BEGRENZT / MITTEL / UNBEGRENZT]
Teilnehmer: [ANZAHL, FAMILIE, FREUNDE]
Ort: [STADT/REGION]
Aufgabe: Begleite mich durch die Organisation

Ausgabe:
1. Sofort nach dem Tod (erste 48h):
   - Behördliche Anmeldung
   - Bestatter wählen
   - Sterbeurkunde
   - Angehörige informieren
2. Bestattungsform:
   - Erd- vs. Feuerbestattung
   - Naturbestattung (Wald, See)
   - Anonyme Bestattung
   - Kostenvergleich
3. Trauerfeier planen:
   - Location (Friedhof, Kirche, freie Trauerhalle)
   - Ablauf gestalten
   - Redner (Pfarrer, Freunde, Familie)
   - Musik auswählen
   - Trauerdruck (Programm, Danksagung)
4. Formalitäten:
   - Todesanzeige aufgeben
   - Erbe/Verlassenschaft (erste Schritte)
   - Versicherungen, Rentenkasse
   - Verträge kündigen
5. Danksagungen:
   - Text-Vorschläge
   - Zeitpunkt
   - Empfänger-Liste
6. Trauerbegleitung:
   - Angebote vor Ort
   - Selbsthilfegruppen
   - Psychologische Unterstützung
   - Trauer-Apps/Online-Angebote
7. Checkliste:
   - Woche 1
   - Woche 2-4
   - Monat 2-3
```

## Beispiel

**Input:** "Vater, 78 Jahre, frei religiös, Feuerbestattung gewünscht, mittleres Budget, ca. 50 Gäste, München"

**Output:**
- Bestattung: Feuerbestattung + Urnenbeisetzung auf Friedhof
- Feier: Freie Trauerhalle, musikalische Begleitung
- Kosten: ca. 5.000-8.000€
- Ablauf: Begrüßung, 3 Reden, Musik, Urnenbeisetzung

## Variationen

- Für plötzlichen Tod: "Trauerfeier unter Zeitdruck organisieren"
- Für Kinder: "Kinder an Trauerfeier beteiligen"
- Für verschiedene Kulturen: "Internationale Bestattung organisieren"
- Für später: "Vorsorge-Treffen mit Bestatter"
