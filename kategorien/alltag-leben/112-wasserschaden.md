# Wasserschaden im Haus/Auto

## Prompt

```
Rolle: Schadensregulierer mit Versicherungsexpertise
Kontext: Ich habe einen Wasserschaden
Betroffen: [WOHNUNG / HAUS / AUTO]
Ursache: [ROHRBRUCH / ÜBERSCHWEMMUNG / REGEN / MASCHINENSCHADEN]
Ausmaß: [KLEIN / MITTEL / GROSS]
Versicherung: [VORHANDEN / UNKLAR / KEINE]
Wann passiert: [GERADE / VOR STUNDEN / VOR TAGEN]
Aufgabe: Sofortmaßnahmen und Schadensabwicklung

Ausgabe:
1. Sofortmaßnahmen (in Reihenfolge):
   - Strom abschalten (wenn nötig)
   - Wasser abstellen
   - Wasser bergen/sofort entfernen
   - Belüftung einrichten
   - Fotos dokumentieren
2. Schadensdokumentation:
   - Was fotografieren?
   - Was notieren?
   - Zeugen/Beweise sichern
   - Wertgegenstände erfassen
3. Versicherung:
   - Welche Versicherung zuständig? (Hausrat, Gebäude, Kfz)
   - Wann melden? (sofort!)
   - Was angeben?
   - Selbstbeteiligung prüfen
4. Fachfirmen:
   - Wann Trocknungsfirma nötig?
   - Wann Wasserschaden-Sanierung?
   - Kostenvoranschläge einholen
   - Sachverständiger wann sinnvoll?
5. Mietrecht (bei Mietwohnung):
   - Vermieter informieren
   - Mietminderung prüfen
   - Wer zahlt was?
   - Ersatzwohnung Anspruch?
6. Timeline:
   - Tag 1-3: Sofortmaßnahmen, Meldung
   - Woche 1-2: Gutachten, Kostenvoranschlag
   - Woche 3+: Sanierung
7. Tipps:
   - Typische Versicherungs-Tricks vermeiden
   - Wann Anwalt?
   - Wann zahlt die Versicherung nicht?
```

## Beispiel

**Input:** "Eigentumswohnung, Waschmaschine gelaufen, mittlerer Schaden (Wohnzimmer nass), Hausratversicherung vorhanden, vor 2 Stunden"

**Output:**
- Sofort: Wasser abstellen, Fotos machen, Lüfter aufstellen
- Versicherung: Heute noch melden
- Trocknungsfirma: Ja, bei diesem Ausmaß empfohlen
- Kosten: Wahrscheinlich über Selbstbeteiligung

## Variationen

- Für Mieter: "Wasserschaden in Mietwohnung - wer zahlt?"
- Für Nachbar: "Wasserschaden vom Nachbarn"
- Für Keller: "Keller überflutet - was tun?"
- Für Auto: "Wasserschaden im Auto (Durchfahrt, offenes Fenster)"
