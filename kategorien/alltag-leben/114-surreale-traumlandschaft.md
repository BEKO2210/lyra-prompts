# Surreale Traumlandschaft

## Prompt

```
Erstelle eine surreale Traumlandschaft wie aus einem Lucid Dream.

ATMOSPHÄRE:
- [VERTRÄUMT / UNHEIMLICH / WUNDERVOLL / MELANCHOLISCH]
- Tageszeit: [EWIGER SONNENUNTERGANG / DÄMMERUNG / NACHT MIT ZWEI MONDEN]
- Physik: [VERZERRT / SCHWERGELOS / FLÜSSIG / KRISTALLIN]

ELEMENTE:
- Landschaft: [SCHWIMMENDE INSELN / SPIEGEL-WÜSTE / BIO-LUMINISZENTER WALD]
- Himmel: [MEER AUS WOLKEN / AURORA BOREALIS / GALAXIE SICHTBAR]
- Architektur: [SCHWEBENDE TEMPLE / UNMÖGLICHE GEOMETRIE / ORGANISCHE STRUKTUREN]
- Lebewesen: [RIESIGE KREATUREN / ABSTRAKTE FORMEN / LEUCHTENDE WESEN]

FARBSCHEMA:
- Dominant: [FARBE]
- Akzente: [KONTRASTFARBEN]
- Licht: [GOLDEN / SILBERN / REGENBOGEN / MONDLICHT]

STIL:
- Hyperrealistisch aber physikalisch unmöglich
- Tiefe Perspektive mit unmöglichen Tiefen
- Fließende Übergänge zwischen Elementen
- Symbolische Details versteckt in der Landschaft

REFERENZEN:
- Salvador Dalí (schmelzende Uhren, lange Schatten)
- René Magritte (Wolken, unmögliche Objekte)
- Zdzisław Beksiński (düstere Schönheit)
- Hayao Miyazaki (wundersame Welten)

TECHNISCH:
- 8K, hyperdetailliert
- Volumetrisches Licht
- Atmosphärische Perspektive
- Tiefe Schärfentiefe mit Fokus auf [HAUPTELEMENT]
```

## Anwendung

**Für:** Konzeptkunst, Album-Cover, Buchillustration, Wallpaper

**Beispiel-Output:** Schwimmende Inseln verbunden durch Wasserfälle in den Himmel, riesige leuchtende Qualle als Sonne, spiegelnder Ozean darunter

## Variationen

- Für Portraits: "Surreales Portrait mit Traum-Elementen"
- Für Architektur: "Unmögliche Architektur im Escher-Stil"
- Für Natur: "Surreale Natur mit riesigen Skalen (kleine Menschen, riesige Blumen)"
- Für Dark Art: "Düstere surreale Alptraum-Landschaft"
