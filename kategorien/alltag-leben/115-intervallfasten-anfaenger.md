# Intervallfasten für Anfänger

## Prompt

```
Rolle: Ernährungsberaterin mit Spezialisierung auf Intervallfasten
Kontext: Ich möchte mit Intervallfasten beginnen
Ziel: [ABNEHMEN / GESUNDHEIT / FOKUS / ENERGIE]
Aktuelle Ernährung: [3 MAHLZEITEN / SNACKER / UNREGELMÄßIG]
Arbeitszeiten: [FRÜH / SPÄT / SCHICHT / FLEXIBEL]
Sport: [KEINER / MORGENS / ABENDS / VARIABEL]
Gesundheit: [KEINE BESCHWERDEN / DIABETES / MEDIZINISCHE EINSCHRÄNKUNGEN]
Aufgabe: Erstelle einen sicheren Einstiegsplan

Ausgabe:
1. Fasten-Methode wählen:
   - 16:8 (16h fasten, 8h Essfenster)
   - 14:10 (sanfter Einstieg)
   - 5:2 (2 Tage reduziert essen)
   - OMAD (One Meal A Day)
   - Welche passt zu meinem Leben?
2. Einstiegsplan:
   - Woche 1-2: Gewöhnungsphase
   - Woche 3-4: Reguläres Fasten
   - Woche 5+: Optimierung
3. Essensfenster-Planung:
   - Frühstück überspringen vs. Abendessen ausfallen lassen
   - Was bei Frühdienst?
   - Was bei Abendsport?
   - Soziale Events managen
4. Was darf ich während Fasten?
   - Getränke (Kaffee, Tee, Wasser)
   - Supplements
   - Kaugummi?
   - Zero-Drinks?
5. Ernährung im Essensfenster:
   - Was essen für Sättigung?
   - Makro-Verteilung
   - Mahlzeiten-Anzahl
   - Beispiel-Tagespläne
6. Herausforderungen:
   - Hunger-Attacken (was tun?)
   - Kopfschmerzen am Anfang
   - Schlaf-Probleme
   - Sozialer Druck
7. Tracking:
   - Was messe ich?
   - Wann Erfolge erwarten?
   - Wann Methode wechseln?

WARNUNG: Bei Medikamenten oder Diabetes vorher mit Arzt sprechen!
```

## Beispiel

**Input:** "Abnehmen, 3 Mahlzeiten, Bürojob 9-17, Sport abends, keine Beschwerden"

**Output:**
- Methode: 16:8 mit Essensfenster 12-20 Uhr
- Plan: Woche 1 = 14:10, dann 16:8
- Morgens: Schwarzer Kaffee, Wasser
- Abends nach Sport: Hauptmahlzeit

## Variationen

- Für Frauen: "Intervallfasten für Frauen (zyklusangepasst)"
- Für Sportler: "Intervallfasten mit Muskelaufbau kombinieren"
- Für 50+: "Intervallfasten ab 50"
- Für gestörtes Essverhalten: "Intervallfasten nach Essstörung"
