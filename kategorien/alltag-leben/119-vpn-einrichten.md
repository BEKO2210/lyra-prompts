# VPN einrichten und nutzen

## Prompt

```
Rolle: IT-Sicherheitsexperte für Privatanwender
Kontext: Ich möchte ein VPN nutzen
Nutzerprofil: [PRIVAT / BERUFLICH / BEIDES]
Hauptzweck: [STREAMING / SICHERHEIT / GEO-BLOCKING / TORRENTING]
Geräte: [PC / SMARTPHONE / SMART-TV / ROUTER / ALLE]
Technik-Level: [ANFÄNGER / FORTGESCHRITTEN]
Budget: [KOSTENLOS / BIS 5€/MONAT / BIS 10€/MONAT]
Aufgabe: Finde das richtige VPN und richte es ein

Ausgabe:
1. VPN-Auswahl:
   - Anbieter-Vergleich (NordVPN, Surfshark, Mullvad, etc.)
   - Was ist wichtig? (No-Logs, Geschwindigkeit, Server)
   - Was ist Marketing-Gimmick?
   - Anbieter vermeiden (rote Flaggen)
2. Einrichtung pro Gerät:
   - Windows/Mac
   - iOS/Android
   - Smart-TV (Android TV, Apple TV, Fire Stick)
   - Router (VPN auf Router-Ebene)
3. Nutzungsszenarien:
   - Sicheres WLAN (Café, Flughafen)
   - Streaming (Netflix, BBC, etc.)
   - Geoblocking umgehen
   - Torrenting (sicher?)
4. Sicherheit:
   - Was schützt ein VPN wirklich?
   - Was schützt es NICHT?
   - Kill-Switch einrichten
   - DNS-Leaks vermeiden
5. Protokolle erklärt:
   - WireGuard (schnell, modern)
   - OpenVPN (bewährt)
   - IKEv2 (mobil gut)
   - Wann welches?
6. Troubleshooting:
   - Langsame Verbindung
   - Seiten blockieren VPN
   - Streaming funktioniert nicht
   - Verbindungsabbrüche
7. Alternativen:
   - Wann reicht ein DNS-Service?
   - Tor vs. VPN
   - Eigenes VPN (VPS)
```

## Beispiel

**Input:** "Privat, Streaming + Sicherheit, PC + Smartphone, Anfänger, bis 5€/Monat"

**Output:**
- Empfehlung: Surfshark (günstig, unbegrenzte Geräte)
- Einrichtung: App installieren, Anmelden, Server wählen
- Streaming: US-Netflix über US-Server
- Sicherheit: Kill-Switch aktivieren

## Variationen

- Für Unternehmen: "Business-VPN einrichten"
- Für Reisen: "VPN für China/Iran (Zensur umgehen)"
- Für Experten: "Eigenen VPN-Server aufsetzen"
- Für kostenlos: "Kostenlose VPNs sicher nutzen"
