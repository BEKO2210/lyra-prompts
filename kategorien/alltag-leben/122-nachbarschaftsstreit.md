# Nachbarschaftsstreit schlichten

## Prompt

```
Rolle: Mediatorin für Nachbarschaftskonflikte
Kontext: Ich habe einen Konflikt mit meinem Nachbarn/meiner Nachbarin
Art des Konflikts: [LÄRM / GRENZE / PFLANZEN / PARKEN / SONSTIGES]
Dauer: [SEIT WANN?]
Intensität: [GERING / MITTEL / HOCH / ESKALIERT]
Vorheriger Kontakt: [NOCH KEINER / GESPROCHEN / GESCHRIEBEN / BEHÖRDE]
Wohnform: [MIETE / EIGENTUM / REIHENHAUS / WOHNUNG]
Aufgabe: Finde eine konstruktive Lösung

Ausgabe:
1. Situationsanalyse:
   - Meine Rechte (Mietrecht, Nachbarrecht)
   - Gültige Beschwerdegründe
   - Was ist rechtlich relevant?
   - Was ist "normal" vs. "übertrieben"?
2. Selbstcheck:
   - Ist meine Erwartung realistisch?
   - Gibt es Kompromiss-Raum?
   - Was will ich erreichen?
   - Was bin ich bereit zu akzeptieren?
3. Gesprächsvorbereitung:
   - Richtiger Zeitpunkt
   - Gesprächstechnik (Ich-Botschaften)
   - Lösungsvorschläge mitbringen
   - Was bei Ablehnung?
4. Eskalationsstufen:
   - Stufe 1: Persönliches Gespräch
   - Stufe 2: Schriftliche Nachricht
   - Stufe 3: Vermieter/Eigentümergemeinschaft
   - Stufe 4: Schlichtungsstelle
   - Stufe 5: Anwalt/Gericht
5. Dokumentation:
   - Was aufschreiben?
   - Beweise sammeln (Lautstärke, Fotos)
   - Zeugen
6. Schlichtungsangebote:
   - Lokale Schlichtungsstellen
   - Mediation
   - Rechtsschutzversicherung nutzen
7. Worst Case:
   - Wann umziehen?
   - Kosten/Nutzen abwägen
   - Psychische Belastung managen
```

## Beispiel

**Input:** "Lautstärke (Musik bis 24h), seit 3 Monaten, mittel, einmal gesprochen (ohne Erfolg), Mietwohnung"

**Output:**
- Recht: Ruhezeiten 22-6/7h, Sonntag/Feiertag ganztägig
- Gespräch: Nochmal versuchen, konkrete Ruhezeiten vorschlagen
- Dokumentation: Tagebuch führen
- Eskalation: Vermieter → Ordnungsamt (wenn nötig)

## Variationen

- Für Eigentum: "Grundstücksstreit mit Nachbarn"
- Für Garten: "Hecke/Baum-Streit schlichten"
- Für Parken: "Parkplatz-Konflikt lösen"
- Für Tierhalter: "Hundekläffer/Nachbarskatze"
