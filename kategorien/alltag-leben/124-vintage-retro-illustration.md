# Vintage Retro Illustration

## Prompt

```
Erstelle eine Vintage Retro Illustration im Stil einer bestimmten Ära.

ÄRA:
- [1950er / 1960er / 1970er / 1980er / ART DECO / MID-CENTURY]

MOTIV:
- [REISEPOSTER / WERBUNG / MAGAZIN COVER / POSTKARTE / ALBUM COVER]

HAUPTELEMENTE:
- [BESCHREIBUNG DES MOTIVS: Auto, Frau, Landschaft, Produkt]
- Stimmung: [OPTIMISTISCH / NOSTALGISCH / ABENTEUERLICH / LUXURIÖS]

STIL-MERKMALE:
- Farbpalette: [TYPISCH FÜR DIE ÄRA]
  - 50er: Pastellfarben, Türkis, Pink
  - 60er: Psychedelisch, Orange, Gelb
  - 70er: Erdtöne, Braun, Orange, Grün
  - Art Deco: Gold, Schwarz, Creme, Marine
- Typografie-Stil: [SERIF / SANS-SERIF / HANDLETTERING]
- Texturen: [HALFTONE / SCREEN PRINT / PAPIERSTRUKTUR]
- Linienführung: [CLEAN / ORGANIC / GEOMETRISCH]

TECHNIK:
- Vektor-Illustration Stil
- Leichte Textur-Overlays für Authentizität
- Subtile Aging-Effekte (Flecken, Falten)
- Perioden-typische Komposition

REFERENZEN:
- 50er: Norman Rockwell, Travel Posters
- 60er: psychedelische Konzertplakate
- 70er: NASA-Graphics, Umweltplakate
- Art Deco: Cassandre, WPA Posters

TECHNISCH:
- Hochauflösend für Print
- CMYK-freundliche Farben
- Platz für Text/Logo
```

## Anwendung

**Für:** Event-Poster, Bar/Restaurant-Deko, Marken-Relaunch, Einladungen

**Beispiel-Output:** 1950er Travel Poster für Hawaii mit DC-3 Flugzeug, Palmen, Pin-up Girl, Pastellfarben, leichter Halftone-Textur

## Variationen

- Für spezifische Themen: "Vintage Space Poster im 60er-Jahre Stil"
- Für Werbung: "Retro Produktwerbung wie in den 50ern"
- Für Musik: "Vinyl-Cover im 70er Funk-Stil"
- Für Getränke: "Art Deco Cocktail-Poster"
