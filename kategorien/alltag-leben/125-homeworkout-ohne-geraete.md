# Homeworkout-Routine ohne Geräte

## Prompt

```
Rolle: Personal Trainer für Home-Workouts
Kontext: Ich möchte zu Hause trainieren ohne Geräte
Fitness-Level: [ANFÄNGER / FORTGESCHRITTEN / SPORTLICH]
Ziele: [ABNEHMEN / MUSKELAUFBAU / AUSDAUER / BEWEGUNG]
Verfügbare Zeit: [MINUTEN PRO SESSION]
Häufigkeit: [X MAL PRO WOCHE]
Einschränkungen: [VERLETZUNGEN / RÜCKEN / GELENKE]
Platz: [ENG / WOHNZIMMER / TERRASSE]
Aufgabe: Erstelle eine effektive Home-Workout-Routine

Ausgabe:
1. Aufwärmung (5-10 Min):
   - Gelenk-Mobilisation
   - Dynamisches Dehnen
   - Herzfrequenz hochfahren
2. Hauptteil:
   - Übungen nach Muskelgruppen
   - Sets und Wiederholungen
   - Progression (schwerer machen)
   - Alternative Übungen bei Einschränkungen
3. Übungs-Bibliothek:
   - Push (Brust, Schultern, Trizeps)
   - Pull (Rücken, Bizeps)
   - Legs (Beine, Gesäß)
   - Core (Bauch, Rumpf)
   - Cardio (Herz-Kreislauf)
4. Trainingspläne:
   - Ganzkörper (für Anfänger)
   - Split (für Fortgeschrittene)
   - HIIT (für Ausdauer)
   - Mobility (für Beweglichkeit)
5. Intensität steigern:
   - Ohne Gewichte schwerer trainieren
   - Zeit unter Spannung
   - Supersätze
   - Drop-Sets
6. Abkühlung:
   - Statisches Dehnen
   - Atemübungen
7. Fortschritt tracken:
   - Was messe ich?
   - Wann Routine wechseln?
```

## Beispiel

**Input:** "Anfänger, Bewegung + leichter Muskelaufbau, 30 Min, 3x/Woche, keine Einschränkungen, Wohnzimmer"

**Output:**
- Aufwärmung: Jumping Jacks, Armkreisen, Hüftkreisen
- Workout: Kniebeugen, Liegestütze (an Wand), Planke, Ausfallschritte
- 3 Runden, 45 Sekunden pro Übung
- Cool-down: Dehnen

## Variationen

- Für kleine Räume: "Workout auf 2qm"
- Für Büro: "Büro-Workout am Schreibtisch"
- Für Reisen: "Hotelzimmer-Workout"
- Für 50+: "Senioren-Workout zu Hause"
