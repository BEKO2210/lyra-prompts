# Selbstständigkeit gründen (Gewerbe)

## Prompt

```
Rolle: Gründungsberaterin mit 100+ erfolgreichen Gründungen
Kontext: Ich möchte mich selbstständig machen (Gewerbe)
Branche: [DIENSTLEISTUNG / HANDEL / PRODUKTION / IT / HANDWERK]
Produkt/Dienstleistung: [BESCHREIBUNG]
Startkapital: [VERFÜGBAR / KREDIT BENÖTIGT]
Ziel: [HAUPTERWERB / NEBENTÄTIGKEIT]
Risikobereitschaft: [KONSERVATIV / MODERAT / HOCH]
Aufgabe: Begleite mich durch die Gründung

Ausgabe:
1. Vorbereitung (1-3 Monate vorher):
   - Businessplan erstellen
   - Markt- und Wettbewerbsanalyse
   - Zielgruppe definieren
   - Finanzplanung (Break-Even)
   - Namen finden und prüfen
2. Formalitäten:
   - Gewerbe anmelden (wo? was brauche ich?)
   - Steuernummer beantragen
   - Kammern (IHK, HWK)
   - Versicherungen (Betriebshaftpflicht, Berufsunfähigkeit)
3. Finanzen:
   - Geschäftskonto eröffnen
   - Buchhaltungssystem wählen
   - Steuerberater finden (ja/nein?)
   - Förderungen prüfen (Gründungszuschuss, etc.)
4. Aufbau:
   - Website/Social Media
   - Erste Kunden akquirieren
   - Lieferanten/Partner finden
   - Büro/Coworking/Homeoffice
5. Marketing:
   - Positionierung
   - Preisgestaltung
   - Vertriebskanäle
   - Netzwerk aufbauen
6. Risiken:
   - Worst-Case-Plan
   - Rückkehr zum Angestelltenverhältnis
   - Einkommenslücken überbrücken
7. Timeline:
   - Monat 1-3: Vorbereitung
   - Monat 4: Anmeldung, Start
   - Monat 6-12: Aufbau
```

## Beispiel

**Input:** "IT-Beratung, Webentwicklung für KMUs, 10.000€ Startkapital, Haupterwerb, moderat"

**Output:**
- Businessplan: Lean Canvas ausfüllen
- Gewerbe: Gewerbeamt, 20-50€ Gebühr
- Versicherungen: Betriebshaftpflicht (ca. 300€/Jahr)
- Marketing: LinkedIn, lokale Netzwerk-Events
- Break-Even: Bei 3.000€/Monat nach 6 Monaten

## Variationen

- Für Handwerk: "Handwerksbetrieb gründen (Meisterpflicht?)"
- Für Online-Shop: "E-Commerce Business gründen"
- Für Beratung: "Freiberuflich vs. Gewerblich"
- Für nebenberuflich: "Von nebenberuflich zu hauptberuflich"
