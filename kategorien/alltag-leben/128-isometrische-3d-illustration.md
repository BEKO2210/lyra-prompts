# Isometrische 3D Illustration

## Prompt

```
Erstelle eine isometrische 3D-Illustration.

THEMA:
- [SMART CITY / BÜRO / DATENCENTER / ÖKOSYSTEM / FABRIK / WOHNUNG]

HAUPTELEMENTE:
- [LISTE DER WICHTIGSTEN OBJEKTE]
- Detail-Level: [MINIMAL / DETAILLIERT / HYPER-DETAILLIERT]
- Stil: [TECHNISCH / SPIELERISCH / REALISTISCH / FLACH]

FARBSCHEMA:
- Primär: [FARBE]
- Sekundär: [FARBE]
- Akzent: [FARBE]
- Hintergrund: [HELL / DUNKEL / FARBE]

PERSPEKTIVE:
- Isometrisch 30°
- Kamera: [BIRD'S EYE / LEICHT ANGEWINKELT]
- Zoom-Level: [ÜBERSICHT / DETAIL]

ELEMENTE ZEIGEN:
- Schnittbild (Innenansicht von Gebäuden)
- Datenflüsse (Linien zwischen Elementen)
- Menschen/Avatare (ja/nein)
- Bewegung/Animation (Pfeile, Partikel)

STIL:
- Cleane Kanten
- Sanfte Schatten
- Subtile Reflexionen
- Einheitliche Lichtquelle

TECHNISCH:
- Vektor-geeignet
- Hohe Auflösung
- Einzelne Elemente unterscheidbar
- Für Infografiken geeignet

REFERENZEN: Apple Design, Stripe Illustrations, Tech-Startup Style
```

## Anwendung

**Für:** Tech-Illustrationen, Infografiken, Landing Pages, Präsentationen

**Beispiel-Output:** Isometrisches Smart City mit verbundenen Gebäuden, Datenflüssen, Elektroautos, Solarpanels, winzigen Menschen

## Variationen

- Für Prozesse: "Isometrischer Workflow/Prozess"
- Für Architektur: "Isometrisches Haus mit Inneneinrichtung"
- Für Natur: "Isometrisches Ökosystem Wald"
- Für Tech: "Isometrisches Blockchain-Netzwerk"
