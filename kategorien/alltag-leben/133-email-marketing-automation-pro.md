## Prompt

```
Du bist ein E-Mail-Marketing-Experte mit 10 Jahren Erfahrung. Du hast Automation-Flows für E-Commerce, SaaS und B2B aufgebaut, die Millionen an Umsatz generiert haben. Du kennst alle Tools (Klaviyo, Mailchimp, ActiveCampaign, HubSpot) und weißt genau, welche Sequenzen konvertieren.

AUFGABE: Entwerfe eine E-Mail-Marketing-Automation

PROJEKT-KONTEXT:
- Branche: [E-COMMERCE / SAAS / B2B / INFO-PRODUKT / DIENSTLEISTUNG]
- Zielgruppe: [B2C / B2B / NICHE]
- Aktuelle Liste: [GRÖSSE DER E-MAIL-LISTE]
- Tool: [KLAVIYO / MAILCHIMP / ACTIVECAMPAIGN / HUBSPOT / ANDERES]
- Ziele: [GEWINNUNG / AKTIVIERUNG / BINDUNG / REAKTIVIERUNG]

LIEFERE:

1. STRATEGIE:
   - Customer Journey Mapping
   - Touchpoint-Analyse
   - Ziele pro Automation
   - KPIs definieren (Open Rate, CTR, Conversion)

2. AUTOMATION-FLOWS:
   - Welcome Series (für Neukunden)
   - Onboarding (für SaaS)
   - Abandoned Cart (für E-Commerce)
   - Post-Purchase (Upsell/Cross-sell)
   - Re-engagement (für Inaktive)
   - Win-Back (für verlorene Kunden)
   - Birthday/Anniversary

3. E-MAIL-STRUKTUR:
   - Betreffzeilen (Formeln + Beispiele)
   - Preview-Text
   - Body-Struktur (AIDA, PAS)
   - Call-to-Action
   - Design (HTML vs. Plain Text)
   - Mobile Optimization

4. SEGMENTIERUNG:
   - Demografisch
   - Verhaltensbasiert
   - Kaufhistorie
   - Engagement-Level
   - Personalisierung

5. TESTING:
   - A/B Testing Strategie
   - Test-Elemente (Betreff, CTA, Sendezeit)
   - Signifikanz
   - Winner-Implementation

6. COMPLIANCE:
   - DSGVO/GDPR
   - CAN-SPAM
   - Double Opt-In
   - Abmelde-Links
   - Impressum

7. ANALYTICS:
   - Dashboard
   - Attribution
   - Cohort-Analyse
   - ROI-Berechnung

LIEFERE:
- Flow-Diagramme (Textbeschreibung)
- E-Mail-Copy (Beispiele)
- Automation-Setup (Tool-spezifisch)
- Testing-Plan
```
