# Minimalistisches Logo Design

## Prompt

```
Erstelle ein minimalistisches Logo-Design.

MARKE:
- Name: [FIRMENNAME]
- Branche: [BRANCHE]
- Werte: [3-5 KERNWERTE]
- Zielgruppe: [WER?]
- Persönlichkeit: [PROFESSIONELL / SPIELERISCH / LUXURIÖS / TECHNISCH]

STIL:
- [WORTMARKE / BILDMARKE / KOMBINATION]
- [GEOMETRISCH / ORGANISCH / ABSTRAKT / FIGURATIV]
- Minimalismus-Level: [EXTREM / BALANCIERT]

FARBSCHEMA:
- Primärfarbe: [FARBE]
- Sekundärfarbe: [FARBE / KEINE]
- Hintergrund: [HELL / DUNKEL / BEIDES]

ELEMENTE:
- Symbol/Ikonographie: [BESCHREIBUNG]
- Typografie: [SERIF / SANS-SERIF / CUSTOM]
- Negative Space nutzen? [JA / NEIN]

ANFORDERUNGEN:
- Skalierbar (favicon bis Billboard)
- Einfarbig funktioniert
- Erkennbar in klein
- Zeitlos (nicht trendy)

REFERENZEN:
- Apple, Nike, FedEx (Pfeil im Negative Space)
- Airbnb, Uber, Slack
- [EIGENE REFERENZEN]

TECHNISCH:
- Vektor-basiert
- CMYK + RGB
- Mit und ohne Claim
- Variationen für verschiedene Hintergründe
```

## Anwendung

**Für:** Startups, Rebranding, persönliche Marke, Produkte

**Beispiel-Output:** Abstraktes geometrisches Symbol aus Anfangsbuchstaben, cleane Sans-Serif Typografie, funktioniert in Schwarzweiß

## Variationen

- Für Tech: "Tech-Startup Logo (geometrisch, futuristisch)"
- Für Luxus: "Luxus-Brand Logo (serif, elegant)"
- Für Food: "Food-Brand Logo (organisch, appetitlich)"
- Für Personal Brand: "Persönliches Logo/Monogramm"
