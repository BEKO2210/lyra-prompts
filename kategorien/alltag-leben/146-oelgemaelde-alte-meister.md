# Ölgemälde im Stil der Alten Meister

## Prompt

```
Erstelle ein Ölgemälde im Stil der Alten Meister.

EPOCHE:
- [RENAISSANCE / BAROCK / ROKOKO / NIEDERLÄNDISCHES GOLDENES ZEITALTER]

MEISTER-STIL:
- [REMBRANDT (Licht/Schatten) / VERMEER (Intimes Licht) / LEONARDO (Sfumato) / RUBENS (Dynamik)]

MOTIV:
- [PORTRAIT / STILLLEBEN / HISTORIE / LANDSCHAFT / MYTHOLOGIE]
- Hauptfigur: [BESCHREIBUNG]
- Szene: [BESCHREIBUNG]

TECHNIK:
- Chiaroscuro (dramatischer Licht-Kontrast)
- Sfumato (weiche Übergänge)
- Impasto (pastose Farbauftrag - ja/nein?)
- Glazing (Lasuren)

FARBSCHEMA:
- Erdtöne: Umbra, Ocker, Siena
- Tiefes Rot, Königsblau, Gold
- Warme Schatten
- Reiche, gesättigte Farben

BELEUCHTUNG:
- [SEITLICHES LICHT / VON OBEN / KERZENlicht]
- Dramatische Schatten
- Helle Akzente auf Haut/Stoff

TEXTUR:
- Sichtbare Pinselstriche
- Leinwand-Struktur
- Alte Craquelé-Effekte (subtil)
- Patina des Alters

REFERENZEN:
- Rembrandt: "Die Nachtwache", Selbstportraits
- Vermeer: "Das Mädchen mit dem Perlenohrgehänge"
- Leonardo: "Mona Lisa", "Das Abendmahl"

TECHNISCH:
- Museum-Qualität
- Großformat
- Authentische Material-Darstellung
```

## Anwendung

**Für:** Kunst-Prints, historische Illustrationen, elegante Portraits

**Beispiel-Output:** Portrait einer Frau im Stil Vermeers, seitliches Fensterlicht, Perlenohrgehänge, dunkler Hintergrund, weiche Sfumato-Übergänge

## Variationen

- Für Portraits: "Barock-Portrait mit Kragen und Spitze"
- Für Stillleben: "Niederländisches Stillleben mit Blumen"
- Für Mythologie: "Mythologische Szene im Rubens-Stil"
- Für Moderne: "Zeitgenössisches Portrait im Alten Meister-Stil"
