# Präsentationstechnik verbessern

## Prompt

```
Rolle: Präsentationscoach für Führungskräfte
Kontext: Ich möchte meine Präsentationstechnik verbessern
Aktuelles Level: [ANFÄNGER / FORTGESCHRITTEN / ERFAHREN]
Häufigkeit: [WIE OFT PRÄSENTIERE ICH?]
Publikum: [KLEIN / GROSS / INTERN / EXTERN]
Themen: [TECHNISCH / VERKAUF / STRATEGIE / MOTIVATION]
Schwächen: [NERVOSITÄT / FOLIEN / STIMME / KÖRPERSPRACHE]
Aufgabe: Erstelle einen Trainingsplan

Ausgabe:
1. Vorbereitung:
   - Zielgruppen-Analyse
   - Kernbotschaft definieren (1 Satz)
   - Struktur (Anfang, Mitte, Ende)
   - Storytelling-Elemente
2. Folien-Design:
   - Weniger ist mehr
   - Bilder vs. Text
   - Corporate Design
   - Tools (PowerPoint, Keynote, Prezi)
3. Stimme:
   - Lautstärke und Tempo
   - Pausen (die Macht der Stille)
   - Betonung
   - Stimmband-Gesundheit
4. Körpersprache:
   - Blickkontakt
   - Gestik
   - Bewegung im Raum
   - Auftritt und Abgang
5. Nervosität managen:
   - Atemtechniken
   - Visualisierung
   - Vorbereitung
   - Erfahrung sammeln
6. Interaktion:
   - Fragen stellen
   - Auf Fragen antworten
   - Schwierige Zuhörer
   - Q&A vorbereiten
7. Übungsplan:
   - Solo-Übungen
   - Vor Publikum üben
   - Video-Analyse
   - Feedback einholen
```

## Beispiel

**Input:** "Fortgeschritten, 1x/Monat, externe Kunden, Verkauf, Nervosität bei großem Publikum"

**Output:**
- Vorbereitung: 3-Kernbotschaften-Regel
- Nervosität: Box-Breathing vor dem Vortrag
- Übung: Toastmasters besuchen oder interne Sessions
- Folien: 10-20-30 Regel (10 Folien, 20 Min, 30pt Schrift)

## Variationen

- Für Online: "Online-Präsentationen meistern"
- Für Pitch: "Investoren-Pitch perfektionieren"
- Für TED-Stil: "TED-Talk Stil lernen"
- Für Panel: "Panel-Diskussion führen"
