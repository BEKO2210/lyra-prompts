# Discord-Server aufbauen

## Prompt

```
Rolle: Discord-Community-Manager mit mehreren 10k+ Servern
Kontext: Ich möchte einen Discord-Server erstellen
Zweck: [GAMING / LERNEN / BUSINESS / HOBBY / SUPPORT]
Zielgruppe: [WER SOLL BEITRETEN?]
Größe: [KLEIN / MITTEL / GROSS]
Moderation: [ICH ALLEIN / TEAM / BOTS]
Monetarisierung: [KEINE / PREMIUM-ROLLEN / SPENDEN]
Aufgabe: Erstelle einen professionellen Discord-Server

Ausgabe:
1. Server-Struktur:
   - Kategorien planen
   - Text-Channel (Information, Chat, Support)
   - Voice-Channel (Gaming, Musik, AFK)
   - Private Bereiche (Premium, Mods)
2. Rollen-System:
   - Admin/Mod-Rollen
   - User-Level (XP-System)
   - Premium/Supporter-Rollen
   - Self-Assignable Roles (Interessen)
3. Bot-Integration:
   - MEE6 (Leveling, Moderation)
   - Dyno (Custom Commands)
   - Carl-bot (Reaction Roles)
   - Musik-Bots
   - Ticket-Bot (Support)
4. Regeln und Moderation:
   - Server-Regeln formulieren
   - Automod einrichten
   - Verwarnungssystem
   - Ban/Timeout-Richtlinien
5. Onboarding:
   - Willkommens-Nachricht
   - Verification-System
   - Einführungs-Channel
   - FAQ
6. Community-Aufbau:
   - Erste Mitglieder gewinnen
   - Events planen
   - Engagement fördern
   - Partnerschaften
7. Wachstum:
   - Server-Listing (Disboard, etc.)
   - Social Media Promotion
   - Incentives für Einladungen
   - Qualität vs. Quantität
```

## Beispiel

**Input:** "Gaming (Coop-Spiele), 18-35 Jahre, mittel, ich + 2 Mods, keine Monetarisierung"

**Output:**
- Struktur: Willkommen, Allgemein, Spiele-Kategorien, Voice
- Rollen: Admin, Mod, Veteran, Member
- Bots: MEE6 für XP, Rhythm für Musik
- Regeln: Kein Spam, respektvoller Umgang

## Variationen

- Für Business: "Business-Discord für Kunden"
- Für Lernen: "Lern-Community Discord"
- Für Support: "Support-Server für Produkt"
- Für große Communities: "Discord-Server mit 10k+ Mitgliedern"
