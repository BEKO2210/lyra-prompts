# Notfall-Apotheke für Zuhause

## Prompt

```
Rolle: Apothekerin mit Notfallmedizin-Erfahrung
Kontext: Ich möchte eine gut ausgestattete Hausapotheke haben
Haushalt: [ANZAHL PERSONEN, ALTER]
Besonderheiten: [KINDER / BABY / SCHWANGER / CHRONISCH KRANKE / HAUSTIERE]
Vorerkrankungen: [ALLERGIEN / HERZ / ASTHMA / SONSTIGES]
Region: [STADT / LAND / BERGE]
Aufgabe: Erstelle eine Hausapotheke-Checkliste

Ausgabe:
1. Basis-Ausstattung:
   - Schmerzmittel (Paracetamol, Ibuprofen, ASS)
   - Erkältungsmittel
   - Verdauung (Magen-Darm)
   - Wundversorgung (Pflaster, Verbände)
   - Haut (Cremes, Sonnenschutz)
2. Spezifisch für Haushalt:
   - Kinder-Medikamente (Dosierung!)
   - Baby-Bedarf (Fieberthermometer, Nasensauger)
   - Chronische Erkrankungen
   - Allergie-Medikamente
3. Erste-Hilfe:
   - Verbandskasten DIN 13169
   - Erste-Hilfe-Anleitung
   - Notfallnummern (sichtbar!)
   - AED (wann sinnvoll?)
4. Lagerung:
   - Kühl, trocken, dunkel
   - Kinder-sicher
   - Ablaufdaten checken
   - Entsorgung (Altmedikamente)
5. Wissen:
   - Wann zum Arzt?
   - Dosierung berechnen
   - Wechselwirkungen
   - Notfall-Symptome
6. Erweiterte Ausstattung:
   - Blutdruckmessgerät
   - Blutzuckermessgerät
   - Sauerstoffsättigung (Pulse-Oximeter)
   - Fieberthermometer
7. Checkliste:
   - Monatlich: Ablaufdaten checken
   - Jährlich: Komplett-Check
   - Nach Gebrauch: Auffüllen

WARNUNG: Dies ersetzt keinen Arztbesuch bei ernsten Symptomen!
```

## Beispiel

**Input:** "4 Personen (2 Erwachsene, 2 Kinder 5+8), keine Vorerkrankungen, Stadt"

**Output:**
- Basis: Paracetamol Kinder + Erwachsene, Ibuprofen, Wundspray
- Kinder: Fiebersaft, Nasenspray, Zeckenzange
- Erste-Hilfe: Komplett-Set mit Anleitung
- Lagerung: Medikamentenschrank im Bad (kühl)

## Variationen

- Für Reisen: "Reiseapotheke zusammenstellen"
- Für Sportler: "Sport-Verletzungen-Apotheke"
- Für Senioren: "Hausapotheke für Ältere"
- Für Camping: "Outdoor-Apotheke"
