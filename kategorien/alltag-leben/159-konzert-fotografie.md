# Konzert/Event-Fotografie

## Prompt

```
Rolle: Musik- und Event-Fotograf mit 10 Jahren Erfahrung
Kontext: Ich möchte Konzert- oder Event-Fotografie machen
Level: [ANFÄNGER / FORTGESCHRITTEN / PROFI]
Equipment: [SMARTPHONE / KOMPAKT / DSLR / SPIEGELLOS]
Zugang: [PUBLIKUM / FOTOGRAFENGRABEN / BACKSTAGE / AKKREDITIERT]
Event-Typ: [KLEINER CLUB / ARENA / FESTIVAL / PRIVAT]
Ziel: [HOBBY / PORTFOLIO / VERKAUF / SOCIAL MEDIA]
Aufgabe: Tipps für bessere Konzertfotos

Ausgabe:
1. Technik:
   - Kamera-Einstellungen (Manuell-Modus)
   - ISO (wie hoch ist okay?)
   - Verschlusszeit (Bewegung einfrieren)
   - Blende (Tiefenschärfe)
   - Weißabgleich (Lichtwechsel)
2. Licht:
   - Bühnenlicht nutzen
   - Bei schlechtem Licht
   - Farblicht handhaben
   - Blitz (nie auf der Bühne!)
3. Komposition:
   - Regel der Dritte
   - Blickkontakt zum Künstler
   - Publikum einbeziehen
   - Dynamische Winkel
4. Verschiedene Perspektiven:
   - Frontal von der Bühne
   - Seitlich (Silhouetten)
   - Aus dem Publikum
   - Detail-Aufnahmen (Instrumente)
5. Rechtliches:
   - Veranstalter-Regeln
   - Bildrechte (Künstler)
   - Publikum (Einverständnis?)
   - Veröffentlichung
6. Nachbearbeitung:
   - RAW entwickeln
   - Schwarz-Weiß (für Dramatik)
   - Noise Reduction
   - Lightroom-Workflow
7. Karriere:
   - Portfolio aufbauen
   - Akkreditierungen bekommen
   - Künstler kontaktieren
   - Verkauf (Redaktionen, Agenturen)
```

## Beispiel

**Input:** "Fortgeschritten, spiegellose Kamera, Fotografengraben, kleiner Club, Portfolio"

**Output:**
- Einstellungen: 1/250s, f/2.8, ISO 1600-6400
- Licht: Bühnenlicht nutzen, Spot-Metering
- Aufnahmen: 3 Songs Regel beachten
- Nachbearbeitung: Lightroom, Schwarz-Weiß für Drama

## Variationen

- Für Anfänger: "Erstes Konzert fotografieren (Publikum)"
- Für Festivals: "Festival-Fotografie Guide"
- Für Club: "Club-Fotografie (dunkel, eng)"
- Für Veranstalter: "Als Veranstalter Fotos organisieren"
