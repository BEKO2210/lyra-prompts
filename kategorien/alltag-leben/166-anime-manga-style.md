# Anime/Manga Art Style

## Prompt

```
Erstelle ein Artwork im Anime/Manga-Stil.

GENRE:
- [SHONEN / SHOJO / SEINEN / JOSEI / CHIBI / MECHA]

CHARAKTER:
- Typ: [PROTAGONIST / ANTAGONIST / SUPPORT]
- Geschlecht: [MÄNNLICH / WEIBLICH]
- Alter: [JUGENDLICH / JUNG / ERWACHSEN]
- Persönlichkeit: [ENERGISCH / RUHIG / MYSTERIÖS / TSUNDERE]

DESIGN:
- Haare: [FARBE, STIL, LÄNGE]
- Augen: [FARBE, FORM, AUSDRUCK]
- Kleidung: [SCHULUNIFORM / FANTASY / STREETWEAR / KAMPFANZUG]
- Accessoires: [Waffe, Schmuck, Gadgets]

POSE:
- [AKTION / RUHE / DRAMATISCH / NIEDLICH]
- Kamera-Winkel: [FRONTAL / VON UNTEN / SEITLICH]
- Hintergrund: [EINFARBIG / ABSTRAKT / DETAILLIERTE SZENE]

STIL-MERKMALE:
- Große, ausdrucksstarke Augen
- Kleine Nase und Mund
- Detaillierte Haare mit Highlights
- Dynamische Linienführung
- Cel-Shading mit weichen Gradienten

REFERENZEN:
- [STUDIO GHIBLI / SHONEN JUMP / MAKOTO SHINKAI / EIGENER STIL]
- Ähnlich wie: [NARUTO / ATTACK ON TITAN / YOUR NAME / SPIRITED AWAY]

TECHNISCH:
- Hohe Auflösung
- Klare Linien
- Lebendige Farben
- Für Print oder Digital geeignet
```

## Anwendung

**Für:** Fan-Art, Charakter-Design, Social Media, Prints

**Beispiel-Output:** Junger männlicher Protagonist im Shonen-Stil, spiky Haare, große Augen, dynamische Action-Pose, Cel-Shading mit Speed-Lines

## Variationen

- Für Chibi: "Chibi-Charakter (super-deformed)"
- Für Mecha: "Mecha/Robot im Anime-Stil"
- Für Landschaft: "Anime-Landschaft im Ghibli-Stil"
- Für Portraits: "Anime-Portrait mit emotionalem Ausdruck"
