# Schreibtisch-Setup optimieren

## Prompt

```
Rolle: Ergonomie-Beraterin und Produktivitäts-Coach
Kontext: Ich möchte meinen Schreibtisch-Setup optimieren
Nutzung: [BÜRO / HOME OFFICE / GAMING / KREATIV]
Stunden/Tag: [STUNDEN]
Vorhanden: [WAS HAST DU BEREITS?]
Budget: [FÜR OPTIMIERUNG]
Probleme: [RÜCKEN / NACKEN / HANDGELENK / AUGEN / KEINE]
Platz: [GROSS / BESCHRÄNKT / SEHR ENG]
Aufgabe: Optimiere meinen Arbeitsplatz

Ausgabe:
1. Möbel:
   - Schreibtisch (Höhe, Tiefe)
   - Stuhl (Ergonomie-Features)
   - Steh-Sitz-Option
   - Monitor-Arm
2. Monitor-Setup:
   - Ein Monitor vs. Dual
   - Größe und Auflösung
   - Höhe und Abstand
   - Blaulicht-Filter
3. Peripherie:
   - Tastatur (mechanisch? ergonomisch?)
   - Maus (ergonomisch? vertikal?)
   - Handgelenk-Auflagen
   - Webcam und Mikrofon
4. Beleuchtung:
   - Tageslicht
   - Schreibtischlampe
   - Indirektes Licht
   - Bildschirm-Blendung vermeiden
5. Kabelmanagement:
   - Kabel verstecken
   - Kabelführung
   - Steckdosenleiste
   - Kabellos wo möglich
6. Zubehör:
   - Laptop-Ständer
   - Dokumentenhalter
   - Fußstütze
   - Anti-Ermüdungsmatte
7. Gesundheit:
   - 20-20-20 Regel (Augen)
   - Mikro-Pausen
   - Dehnübungen
   - Belüftung
```

## Beispiel

**Input:** "Home Office, 8h/Tag, Standard-Schreibtisch + Stuhl, 500€ Budget, Nacken-Probleme, beschränkt"

**Output:**
- Priorität: Monitor-Arm (Augenhöhe), externe Tastatur/Maus
- Stuhl: Wenn möglich tauschen, sonst Kissen
- Monitor: Auf Bücher stellen oder Arm kaufen
- Pausen: Timer für 20-20-20

## Variationen

- Für Gaming: "Gaming-Setup optimieren"
- Für Dual-Monitor: "Multi-Monitor Setup"
- Für kleine Räume: "Setup auf wenig Platz"
- Für unterwegs: "Mobiles Office-Setup"
