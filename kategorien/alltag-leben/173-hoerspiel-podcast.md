# Hörspiel/Podcast produzieren

## Prompt

```
Rolle: Audio-Produzentin mit Hörspiel- und Podcast-Erfahrung
Kontext: Ich möchte ein Hörspiel oder Podcast produzieren
Format: [HÖRSPIEL / NARRATIV-PODCAST / INTERVIEW-PODCAST / DOKUMENTARISCH]
Genre: [KRIMI / FANTASY / COMEDY / BILDUNG / GESCHICHTE]
Länge: [MINUTEN PRO FOLGE]
Häufigkeit: [EINMALIG / SERIE / WÖCHENTLICH]
Budget: [NIEDRIG / MITTEL / HOCH]
Team: [ALLEIN / MIT SPRECHERN / MIT TEAM]
Aufgabe: Erstelle einen Produktionsplan

Ausgabe:
1. Drehbuch:
   - Struktur (Hörspiel vs. Podcast)
   - Dialog schreiben
   - Geräuschbeschreibungen
   - Sprecheranweisungen
2. Pre-Production:
   - Casting (Stimmen finden)
   - Aufnahme-Ort wählen
   - Equipment-Checkliste
   - Terminplanung
3. Aufnahme:
   - Mikrofon-Technik
   - Raumakustik
   - Mehrere Takes
   - Ambience aufnehmen
4. Post-Production:
   - Schnitt-Software (Audacity, Reaper, ProTools)
   - Dialog-Schnitt
   - Atmosphäre/Sounddesign
   - Musik (Copyright!)
5. Sounddesign:
   - Foley (Geräusche)
   - Effekte
   - Räumlichkeit (Reverb)
   - Musik-Untermalung
6. Veröffentlichung:
   - Hosting (Anchor, Podigee, etc.)
   - RSS-Feed
   - Plattformen (Spotify, Apple)
   - Marketing
7. Rechtliches:
   - Musikrechte
   - Sprecher-Verträge
   - GEMA/VG Wort
   - Impressum
```

## Beispiel

**Input:** "Hörspiel, Krimi, 30 Min, Serie 6 Folgen, mittel, mit Sprechern"

**Output:**
- Drehbuch: 3 Akt-Struktur pro Folge
- Casting: Hörbuch-Sprecher oder Theater
- Aufnahme: Mietstudio oder guter Raum
- Post: Reaper (kostengünstig), Freesound für Effekte
- Hosting: Podigee mit 20€/Monat

## Variationen

- Für Interview: "Interview-Podcast produzieren"
- Für Solo: "Solo-Podcast ohne Schnitt"
- Für Fiction: "Fiction-Podcast Serie"
- Für Bildung: "Bildungs-Podcast konzipieren"
