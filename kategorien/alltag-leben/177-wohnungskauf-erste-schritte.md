# Wohnungskauf erste Schritte

## Prompt

```
Rolle: Immobilienberaterin für Erstkäufer
Kontext: Ich überlege, eine Wohnung zu kaufen
Situation: [MIETER / BEI ELTERN / EIGENE WOHNUNG]
Ort: [STADT / REGION]
Budget: [EIGENKAPITAL + KREDITFÄHIGKEIT]
Nutzung: [EIGENNUTZUNG / VERMIETUNG / BEIDES]
Zeithorizont: [SOFORT / 1 JAHR / 2+ JAHRE]
Aufgabe: Erste Schritte zum Wohnungskauf

Ausgabe:
1. Finanzierung prüfen:
   - Eigenkapital (mindestens 10-20%)
   - Kreditfähigkeit (Einkommen, Schufa)
   - Maximale Kreditsumme berechnen
   - Nebenkosten (5-10% des Kaufpreises)
2. Wohnungssuche:
   - Immobilienportale (Immobilienscout, etc.)
   - Makler (Pro/Contra)
   - Ausschreibungen (Banken, Kommunen)
   - Netzwerk aktivieren
3. Besichtigung:
   - Checkliste (was prüfen?)
   - Protokoll machen
   - Nachbarn ansprechen
   - Dokumente anfordern (Teilungserklärung)
4. Bewertung:
   - Ist der Preis angemessen?
   - Vergleichspreise recherchieren
   - Gutachter (wann sinnvoll?)
   - Renovierungsbedarf kalkulieren
5. Kaufprozess:
   - Angebot machen
   - Finanzierungszusage
   - Notartermin
   - Kaufvertrag prüfen (Anwalt?)
6. Risiken:
   - Erbbaurecht (beachten!)
   - WEG-Probleme
   - Mieter in der Wohnung
   - Altlasten
7. Timeline:
   - Von der Suche bis zum Einzug
   - Wie lange dauert es?
   - Was parallel erledigen?
```

## Beispiel

**Input:** "Mieter, Berlin, 50.000€ EK + 3.000€/Monat Netto, Eigennutzung, 1 Jahr"

**Output:**
- Budget: max. 350.000€ (mit Kredit)
- Nebenkosten: ca. 30.000€ (Grunderwerbsteuer, Notar, Makler)
- Suche: Jetzt starten, Kauf in 6-9 Monaten
- Finanzierung: Mehrere Angebote einholen

## Variationen

- Für Anleger: "Wohnung als Kapitalanlage kaufen"
- Für Erbpacht: "Erbbaurecht verstehen"
- Für Neubau: "Neubauwohnung kaufen"
- Für Auktion: "Zwangsversteigerung nutzen"
