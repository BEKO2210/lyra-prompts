# Schichtplan erstellen (Betrieb)

## Prompt

```
Rolle: Personalleiterin mit Erfahrung in Schichtplanung
Kontext: Ich muss einen Schichtplan erstellen
Branche: [GESUNDHEIT / PRODUKTION / EINZELHANDEL / GASTRONOMIE / SICHERHEIT]
Mitarbeiter: [ANZAHL]
Schichtarten: [FRÜH / SPÄT / NACHT / WE / FEIERTAG]
Anforderungen: [ÖFFNUNGSZEITEN / BEDECKUNG / QUALIFIKATIONEN]
Gesetze: [ARBEITSZEITGESETZ / BRANCHEN-SONDERVEREINBARUNGEN]
Tools: [EXCEL / SOFTWARE / HAND / PLANUNG]
Aufgabe: Erstelle einen rechtssicheren Schichtplan

Ausgabe:
1. Rahmenbedingungen:
   - Arbeitszeitgesetz (11h Ruhe, max. 8h/Tag)
   - Branchen-Regeln (z.B. Pflege)
   - Betriebsvereinbarungen
   - Wünsche der Mitarbeiter
2. Planungs-Grundlagen:
   - Bedarfsanalyse (wann wie viele?)
   - Qualifikations-Matrix
   - Urlaubsplanung integrieren
   - Ausfall-Reserve
3. Schichtmodelle:
   - Früh-Spät (2-Schicht)
   - Früh-Spät-Nacht (3-Schicht)
   - Wochenend-Modelle
   - Gleitzeit-Elemente
4. Fairness:
   - Wochenend-Dienste fair verteilen
   - Feiertags-Dienste rotieren
   - Nachtschicht-Begrenzung
   - Wünsche berücksichtigen
5. Tools:
   - Excel-Vorlage
   - Software (Planday, Shiftplan, etc.)
   - Automatisierung
   - Kommunikation
6. Problemlösung:
   - Kurzfristige Ausfälle
   - Krankmeldungen
   - Tausch unter Mitarbeitern
   - Überstunden-Management
7. Kommunikation:
   - Plan veröffentlichen (wann?)
   - Änderungen kommunizieren
   - Feedback einholen
   - Konflikte lösen
```

## Beispiel

**Input:** "Pflege, 15 Mitarbeiter, Früh-Spät-Nacht-WE, 24/7 Bedarf, ArbZG, Excel"

**Output:**
- Modell: 3-Wochen-Rhythmus, jeder hat 1 WE frei
- Regeln: Max. 3 Nächte hintereinander
- Urlaub: 2 Monate vorher planen
- Ausfall: Pool von Aushilfen

## Variationen

- Für Kleinbetriebe: "Schichtplan für kleines Team"
- Für Produktion: "Produktions-Schichtplan optimieren"
- Für Gastronomie: "Gastro-Dienstplan mit variablen Zeiten"
- Für faire Verteilung: "Wochenend-Dienste fair verteilen"
