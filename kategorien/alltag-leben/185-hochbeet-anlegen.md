# Hochbeet anlegen und bepflanzen

## Prompt

```
Rolle: Gemüse-Gärtnerin mit Urban Gardening-Erfahrung
Kontext: Ich möchte ein Hochbeet anlegen
Standort: [GARTEN / TERRASSE / BALKON]
Größe: [MAßE ODER QM]
Bauen oder kaufen: [SELBER BAUEN / KAUFEN / UNKLAR]
Boden: [HOCHBEETERDE / EIGENE MISCHUNG]
Erfahrung: [ANFÄNGER / FORTGESCHRITTEN]
Aufgabe: Plane und lege ein Hochbeet an

Ausgabe:
1. Standort:
   - Sonne (mindestens 6h)
   - Wasserzugang
   - Windgeschützt
   - Zugänglichkeit
2. Bau:
   - Material (Holz, Stein, Metall)
   - Höhe (60-80cm optimal)
   - Breite (max. 120cm für Erreichbarkeit)
   - Länge (nach Platz)
   - Unterbau (Vlies, Gitter)
3. Befüllung:
   - Schichtung (Totholz, Grünschnitt, Kompost, Erde)
   - Höhenverhältnisse
   - Erde (Hochbeeterde, Pflanzerde)
   - Anmischen (Sand, Kompost)
4. Bepflanzung:
   - Gemüse nach Saison
   - Mischkultur (was passt zusammen?)
   - Fruchtfolge
   - Hochstamm-Obst
5. Pflege:
   - Gießen (Hochbeet trocknet schneller!)
   - Düngen (organisch)
   - Unkraut jäten
   - Schädlinge
6. Saisonales:
   - Frühling: Was pflanzen?
   - Sommer: Pflege, Ernte
   - Herbst: Wintergemüse, Abdeckung
   - Winter: Schutz, Planung
7. Troubleshooting:
   - Erde sackt ab
   - Zu trocken/nass
   - Wenig Ertrag
   - Schädlinge
```

## Beispiel

**Input:** "Garten, 2x1m, selber bauen, Hochbeeterde kaufen, Anfänger"

**Output:**
- Bau: Holz (Lärche/Kiefer), 80cm hoch, Vliesboden
- Befüllung: Unten Totholz, dann Kompost, oben Hochbeeterde
- Pflanzen: Tomaten, Salat, Kräuter (Mischkultur)
- Pflege: Täglich gießen, wöchentlich düngen

## Variationen

- Für Balkon: "Balkon-Hochbeet"
- Für Winter: "Hochbeet überwintern"
- Für Schädlinge: "Hochbeet Schädlinge bekämpfen"
- Für Erde: "Hochbeet-Erde selbst mischen"
