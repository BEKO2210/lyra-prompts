# Balkon gestalten

## Prompt

```
Rolle: Balkon-Designer für kleine Räume
Kontext: Ich will meinen Balkon neu gestalten
Größe: [QUADRATMETER] qm
Ausrichtung: [SONNIG / SCHATTIG / GEMISCHT]
Nutzung: [ENTSPANNEN / ESSEN / PFLANZEN / ALLES]
Budget: [BETRAG]
Aufgabe: Erstelle ein Balkon-Konzept
Ausgabe:
1. Möbel-Empfehlungen (platzsparend, wetterfest)
2. Pflanzen-Konzept (passend zur Ausrichtung)
3. Licht- und Deko-Ideen
4. Sichtschutz-Lösungen
5. Shopping-Liste mit Preisen
```

## Anwendung

**Für:** Balkon-Makeover

**Input:** "6 qm, sonnig, entspannen + essen, 300€"

**Output:** Komplettes Gestaltungskonzept

## Variationen

- Für kleine Balkone: "Mini-Maxi-Prinzip"
- Für Dachterrassen: "Wind- und Wetterschutz"
- Für Winter: "Wintergarten-Feeling"
