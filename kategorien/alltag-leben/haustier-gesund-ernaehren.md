# Haustier gesund ernähren

## Prompt

```
Rolle: Tierernährungsberater für Haustiere
Kontext: Ich will mein Haustier gesünder ernähren
Tier: [HUND / KATZE / ANDERES]
Alter: [JAHRE]
Gewicht: [KG / ZUSTAND: normal / übergewichtig / untergewichtig]
Aktivität: [WENIG / NORMAL / SEHR AKTIV]
Gesundheit: [GESUND / ALLERGIEN / ERKRANKUNGEN]
Aufgabe: Erstelle einen Ernährungsplan
Ausgabe:
1. Täglicher Futterbedarf (Menge, Nährstoffe)
2. Futterarten-Vergleich (Trocken, Nass, Barf, selbstgemacht)
3. Gesunde Leckerlies und Belohnungen
4. Tabu-Lebensmittel (was auf keinen Fall?)
5. Übergangsplan (von altem zu neuem Futter)
```

## Anwendung

**Für:** Gesunde Tierernährung

**Input:** "Hund, 3 Jahre, 5kg übergewichtig, wenig aktiv, gesund"

**Output:** Maßgeschneiderter Futterplan

## Variationen

- Für Welpen: "Wachstum, Entwicklung"
- Für Senioren: "Gelenke, Verdauung"
- Für Allergiker: "Eliminationsdiät, Auslöser"
