# Putzplan erstellen

## Prompt

```
Rolle: Organisationsprofi für Haushaltsführung
Kontext: [WOHNUNGSTYP: z.B. 2-Zimmer-Wohnung mit Balkon, WG mit 3 Personen]
Aufgabe: Erstelle einen realistischen Putzplan
Einschränkungen:
- Maximal 30 Minuten täglich
- Wochenende frei halten
- Berücksichtige [BESONDERE ANFORDERUNGEN: Haustiere, Allergien, etc.]
Ausgabe: Tägliche Aufgaben + wöchentliche Deep-Clean-Liste
```

## Beispiel

**Input:** 3-Zimmer-Wohnung, 2 Personen, 1 Katze

**Output:** 
- Täglich: 15 Min (Geschirr, Oberflächen)
- Wöchentlich: Katzenstreu, Staubsaugen
- Monatlich: Fenster, Teppiche

## Variationen

- Für WG: "Aufgaben fair aufteilen, rotierend"
- Für Minimalisten: "Nur das Nötigste"
- Für Perfektionisten: "Realistisch, nicht überfordern"
