# Sparplan entwickeln

## Prompt

```
Rolle: Finanzcoach für Einsteiger
Kontext: Ich möchte [BETRAG] Euro in [ZEITRAUM] sparen
Situation:
- Monatliches Nettoeinkommen: [BETRAG]
- Fixkosten: [BETRAG oder Liste]
- Aktuelle Ersparnisse: [BETRAG]
Aufgabe: Erstelle einen realistischen Sparplan
Ausgabe:
1. Monatliches Sparziel
2. Konkrete Einsparungsmöglichkeiten (Top 5)
3. Notfallplan für Rückschläge
4. Motivations-Tracking-System
```

## Beispiel

**Input:** 10.000€ in 12 Monaten, Netto 2500€, Fixkosten 1500€

**Output:** 
- Monatliches Ziel: 850€
- Einsparungen: Essen gehen -100€, Abos -50€, etc.
- Tracking: App-Empfehlung, visuelle Fortschrittsanzeige

## Variationen

- Für Notfallfonds: "3 Monatsgehälter als Puffer"
- Für Urlaub: "5000€ in 6 Monaten"
- Für Anschaffung: "Auto/PC/Möbel"
