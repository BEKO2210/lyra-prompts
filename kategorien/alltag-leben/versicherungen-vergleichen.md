# Versicherungen vergleichen

## Prompt

```
Rolle: Versicherungsexperte für Privatkunden
Kontext: Ich will meine Versicherungen überprüfen und optimieren
Aktuelle Situation:
- Alter: [XX]
- Familienstand: [LEDIG / VERHEIRATET / KINDER]
- Wohnung/Haus: [MIETE / EIGEN]
- Beruf: [BERUF]
- Aktuelle Versicherungen: [LISTE oder "unbekannt"]
Aufgabe: Analysiere meinen Versicherungsbedarf
Ausgabe:
1. Pflichtige vs. empfohlene vs. optionale Versicherungen
2. Checkliste: Was habe ich? Was fehlt? Was ist überflüssig?
3. Sparpotenziale (Doppelversicherungen, zu hohe Deckungssummen)
4. Prioritäten-Liste (was zuerst ändern?)
```

## Anwendung

**Für:** Versicherungs-Check-up

**Input:** "30, ledig, Miete, Bürojob, Kfz + Haftpflicht"

**Output:** Analyse mit Sparpotenzialen

## Variationen

- Für Familien: "Kind abgesichert?"
- Für Selbstständige: "Berufsunfähigkeit wichtig"
- Für Hausbesitzer: "Gebäude + Inhalt"
