# Wocheneinkauf planen

## Prompt

```
Rolle: Einkaufsstratege für effiziente Wochenplanung
Kontext: Ich will meinen Wocheneinkauf optimieren
Haushalt: [PERSONENANZAHL] Personen
Budget: [BETRAG] Euro pro Woche
Diet: [OMNIVOR / VEGETARISCH / VEGAN]
Zeitfenster: [WIE VIEL ZEIT HABE ICH?]
Aufgabe: Erstelle einen Wocheneinkaufsplan
Ausgabe:
1. Wochenplan mit 7 Abendessen (inkl. Resteverwertung)
2. Einkaufsliste nach Supermarkt-Abteilungen sortiert
3. Budget-Aufteilung (Fleisch/Gemüse/Grundnahrungsmittel)
4. Meal-Prep-Tipps für den Sonntag
5. Einkaufsroute (welcher Laden wann?)
```

## Anwendung

**Für:** Effizienten Wocheneinkauf

**Input:** "2 Personen, 80€, omnivor, 2 Stunden"

**Output:** Kompletter Wochenplan mit Einkaufsliste

## Variationen

- Für Singles: "Portionen anpassen, weniger Abfall"
- Für Familien: "Kinderfreundliche Gerichte"
- Für Sparfüchse: "Angebote nutzen, saisonal kaufen"
