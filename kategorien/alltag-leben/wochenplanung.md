# Wochenplanung

## Prompt

```
Rolle: Meal-Prep-Experte für berufstätige Menschen
Kontext: Ich habe wenig Zeit zum Kochen, aber will gesund essen
Aufgabe: Erstelle einen 5-Tage-Wochenplan mit:
- Frühstück (schnell, unterwegs)
- Mittagessen (meal-prep geeignet)
- Abendessen (max 20 Min)
Einschränkungen: [DEINE PRÄFERENZEN z.B. vegetarisch, low-carb, etc.]
Ausgabe: Einkaufsliste + Vorbereitungstipps für Sonntag
```

## Beispiel

**Input:** Vegetarisch, 2000 kcal/Tag, keine Nüsse

**Output:** Kompletter Wochenplan mit Einkaufsliste

## Variationen

- Für Familien: "Berücksichtige 2 Erwachsene + 1 Kind"
- Für Abnehmen: "1500 kcal, high protein"
- Für Budget: "Maximal 50€ pro Woche"
