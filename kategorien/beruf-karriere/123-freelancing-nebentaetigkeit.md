# Freelancing als Nebentätigkeit starten

## Prompt

```
Rolle: Business-Coach für Nebentätigkeiten und Freelancing
Kontext: Ich möchte neben meinem Hauptjob freiberuflich arbeiten
Fähigkeiten: [WAS KANN ICH ANBIETEN?]
Hauptjob: [ANGESTELLT / BEAMTER / SCHON FREIBERUFLICH]
Zeit: [STUNDEN PRO WOCHE]
Ziel: [ZUSATZEINKOMMEN / SPAREN FÜR X / TESTEN FÜR HAUPTBERUF]
Branche: [IT / DESIGN / BERATUNG / TEXT / SONSTIGES]
Aufgabe: Starte mein Freelancing richtig

Ausgabe:
1. Rechtliches:
   - Gewerbe vs. freiberuflich (was gilt für mich?)
   - Nebentätigkeitsgenehmigung (brauche ich sie?)
   - Steuern (Einkommensteuer, USt.)
   - Sozialversicherung (GKV, Rente)
   - Scheinselbstständigkeit vermeiden
2. Finanzielles:
   - Stundensatz berechnen
   - Angebotspreise kalkulieren
   - Rechnungen schreiben
   - Einnahmen-Überschuss-Rechnung
3. Erste Kunden:
   - Netzwerk aktivieren
   - Freelance-Plattformen (Upwork, Fiverr, etc.)
   - Social Media Präsenz
   - Referenzen sammeln
4. Zeitmanagement:
   - Hauptjob + Freelancing balancieren
   - Überlastung vermeiden
   - Familie/Freizeit schützen
   - Wann skalieren/aufhören?
5. Verträge:
   - AGB erstellen
   - Projektverträge
   - Datenschutz (DSGVO)
   - Haftpflichtversicherung
6. Steuer-Optimierung:
   - Werbungskosten
   - Home-Office absetzen
   - Equipment abschreiben
   - Steuerberater ja/nein?
7. Exit-Strategien:
   - Zum Hauptberuf machen?
   - Wieder aufhören?
   - GmbH gründen?
```

## Beispiel

**Input:** "Webdesign, Angestellt, 10h/Woche, Zusatzeinkommen, IT"

**Output:**
- Recht: Gewerbe anmelden, Arbeitgeber informieren
- Steuer: EÜR, ggf. Kleinunternehmer
- Stundensatz: 60-80€/h (je nach Erfahrung)
- Plattformen: Fiverr für erste Kunden
