# Feedbackgespräch führen

## Prompt

```
Rolle: Führungscoach für konstruktive Gespräche
Kontext: Ich muss ein Feedbackgespräch mit [MITARBEITER/KOLLEGE] führen
Thema: [LEISTUNG / VERHALTEN / ENTWICKLUNG]
Situation: [KONKRETE BESCHREIBUNG]
Ziel: [WAS WILL ICH ERREICHEN?]
Aufgabe: Bereite das Gespräch vor
Ausgabe:
1. Gesprächsstruktur (Anfang, Mitte, Ende)
2. Konkrete Formulierungen (Ich-Botschaften)
3. Beispiele aus der Praxis (Situation-Verhalten-Auswirkung)
4. Mögliche Reaktionen und Antworten
5. Nachbereitung (Dokumentation, Follow-up)
```

## Anwendung

**Für:** Professionelle Feedbackgespräche

**Input:** "Mitarbeiter, Verhalten, oft zu spät, Pünktlichkeit verbessern"

**Output:** Komplette Gesprächsvorbereitung

## Variationen

- Für positives Feedback: "Leistung würdigen"
- Für schwierige Gespräche: "Kritik wertschätzend verpacken"
- Für Entwicklungsgespräche: "Zukunft planen"
