# Gehaltsverhandlung vorbereiten

## Prompt

```
Rolle: Verhandlungsexperte für Gehaltsgespräche
Kontext: Ich habe bald mein Jahresgespräch
Aktuelle Situation:
- Position: [TITEL]
- Aktuelles Gehalt: [BETRAG]
- Unternehmen: [GRÖSSE, BRANCHE]
- Meine Leistungen: [3 KONKRETE BEISPIELE]
Marktdaten: [WENN BEKANNT, sonst: "Recherche erforderlich"]
Aufgabe: Bereite mich auf das Gespräch vor
Ausgabe:
1. Argumentationskette (Top 3 Punkte)
2. Konkrete Gehaltsforderung mit Begründung
3. Einwände des Chefs + Gegenargumente
4. Alternativen, wenn Gehalt nicht möglich (Benefits, Weiterbildung)
```

## Beispiel

**Input:** Junior Developer, 45k€, Mittelständler, 3 erfolgreiche Projekte

**Output:** 
- Argumentation: Marktwert + Leistungen + Verantwortung
- Forderung: 52k€ (+15%)
- Einwände: "Budget knapp" → Gegenargument: ROI meiner Arbeit

## Variationen

- Für Neueinstellung: "Einstiegsgehalt verhandeln"
- Für Beförderung: "Mehr Verantwortung = mehr Gehalt"
- Für Remote: "Standortunabhängige Vergütung"
