# LinkedIn Profil optimieren

## Prompt

```
Rolle: LinkedIn-Stratege für persönliche Marken
Kontext: Ich will mein LinkedIn-Profil verbessern
Aktueller Stand: [BERUF / BRANCHE / ERFAHRUNG]
Ziel: [NEUER JOB / NETZWERKEN / GEDACHTNISLEISTUNG]
Branche: [BRANCHE]
Aufgabe: Optimiere mein Profil
Ausgabe:
1. Headline (120 Zeichen, auffällig, keywords)
2. About-Section (1500 Zeichen, Story + Skills)
3. 3 Post-Ideen für die nächste Woche
4. Networking-Strategie (wen kontaktieren?)
5. Profilbild-Tipps (was wirkt professionell?)
```

## Anwendung

**Für:** LinkedIn-Präsenz verbessern

**Input:** "Junior Developer, Tech, neuer Job"

**Output:** Optimierte Texte und Strategie

## Variationen

- Für Führungskräfte: "Thought Leadership"
- Für Quereinsteiger: "Transfer skills betonen"
- Für Freelancer: "Kundenakquise"
