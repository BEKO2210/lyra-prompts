# Netzwerken auf Events

## Prompt

```
Rolle: Networking-Experte für berufliche Kontakte
Kontext: Ich gehe auf [EVENT-TYP] und will netzwerken
Branche: [DEINE BRANCHE]
Ziel: [JOB FINDEN / KUNDEN / PARTNERSCHAFTEN / LERNEN]
Erfahrung: [ANFÄNGER / ERFAHREN]
Introvertiert/Extrovertiert: [EHER INTROVERTIERT / EXTROVERTIERT]
Aufgabe: Bereite mich auf das Networking vor
Ausgabe:
1. Elevator Pitch (30-Sekunden-Vorstellung)
2. Gesprächseinstiege (nicht: "Was machst du?")
3. Zielkontakte (wen will ich treffen?)
4. Follow-up-Strategie (nach dem Event)
5. Introvertierten-Tipps (Energie schonen)
```

## Anwendung

**Für:** Effektives Networking

**Input:** "Tech-Konferenz, IT, Job finden, Anfänger, introvertiert"

**Output:** Networking-Strategie

## Variationen

- Für Online: "Virtuelle Events, LinkedIn"
- Für VIPs: "Entscheider ansprechen"
- Für Alumni: "Alte Kontakte reaktivieren"
