# Wasserfarb-Portrait

## Prompt

```
Erstelle ein Portrait im Aquarell-Stil.

MOTIV:
- [PERSON / TIER / FANTASIE-WESEN]
- Ausdruck: [FREUNDLICH / NACHDENKLICH / TRAURIG / Mysteriös]
- Blick: [DIREKT / SEITLICH / NACH UNTEN]

STIL:
- Aquarell-Technik: [NASS IN NASS / TROCKEN / MIXED]
- Detail-Level: [SKIZZENHAFT / DETAILLIERT / HYPERREALISTISCH]
- Kanten: [WEICH / DEFINIERT / GEMISCHT]

FARBSCHEMA:
- Hauttöne: [WARM / KÜHL / PORZELLAN / DUNKEL]
- Akzente: [FARBE FÜR HAARE / KLEIDUNG]
- Hintergrund: [WEIß / FARBLICH / ABSTRAKT / NATÜRLICH]

TECHNIK:
- Transparente Farbschichten
- Sichtbare Pinselstriche
- Farbverläufe und Blooms
- Papier-Textur sichtbar
- Lichteinfall von [RICHTUNG]

ATMOSPHÄRE:
- [ROMANTISCH / MELANCHOLISCH / FRISCH / TRAUMHAFT]
- Stimmung: [LEICHT / DRAMATISCH]

REFERENZEN:
- Agnes Cecile (fließend, emotional)
- Silvia Pelissero (dunkel, dramatisch)
- [EIGENE REFERENZ]

TECHNISCH:
- Hochauflösend für Print
- Aquarell-Papier Textur
- Natürliche Farbpalette
```

## Anwendung

**Für:** Kunst-Prints, Geschenke, Buchillustrationen, Social Media

**Beispiel-Output:** Frauenportrait mit sanftem Lächeln, warme Hauttöne, fließende Haare in Blau und Lila, weißer Hintergrund mit Farbspritzern

## Variationen

- Für Tiere: "Aquarell Tierportrait (Hund, Katze, Pferd)"
- Für Paare: "Aquarell Doppelportrait"
- Für Kinder: "Aquarell Kinderportrait"
- Für Landschaft: "Aquarell Landschaft mit Figuren"
