# Charakter-Design für Spiele/Animation

## Prompt

```
Erstelle ein professionelles Charakter-Design für Spiele oder Animation.

CHARAKTER:
- Name: [NAME]
- Rolle: [PROTAGONIST / ANTAGONIST / NPC / BEGLEITER]
- Genre: [FANTASY / SCI-FI / REALISTISCH / CARTOON / HORROR]
- Stimmung: [HELDENHAFT / DUNKEL / COMIC / MYSTERIÖS]

AUSSEHEN:
- Alter: [JAHRE]
- Geschlecht: [M/W/D]
- Körperbau: [SCHLANK / ATHLETISCH / STÄMMIG / ÜBERGewicht]
- Größe: [KLEIN / DURCHSCHNITT / GROSS]
- Besondere Merkmale: [NARBEN / TATTOOS / PROTHESEN / OHREN / HÖRNER]

KLEIDUNG:
- Stil: [RÜSTUNG / STREETWEAR / HISTORISCH / FUTURISTISCH]
- Farben: [FARBSCHEMA]
- Accessoires: [Waffen, Schmuck, Gadgets]
- Funktionalität: [PRAKTISCH / DEKORATIV / STATUSSYMBOL]

PERSÖNLICHKEIT:
- Traits: [3-5 EIGENSCHAFTEN]
- Hintergrund: [KURZ]
- Motivation: [WAS TREIBT DEN CHARAKTER?]

DESIGN-ANFORDERUNGEN:
- Silhouette erkennbar
- Lesbarkeit in klein
- Turnaround (Front, Seite, Rücken)
- Expression-Sheet (Emotionen)
- Pose-Sheet (Aktionen)

STIL:
- [REALISTISCH / STYLIZED / CEL-SHADED / PIXEL ART]
- Detail-Level: [LOW POLY / MID / HIGH]
- Ziel-Engine/Plattform: [UNITY / UNREAL / MOBILE / FILM]

REFERENZEN:
- [ÄHNLICHE CHARAKTERE / KÜNSTLER]
```

## Anwendung

**Für:** Game-Development, Animation, Concept Art, Indie-Studios

**Beispiel-Output:** Weiblicher Fantasy-Krieger mit Riesenschwert, auffällige Silhouette durch Cape, markante Narbe, Rüstung mit kulturellen Details
