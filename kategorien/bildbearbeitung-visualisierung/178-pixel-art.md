# Pixel Art erstellen

## Prompt

```
Erstelle Pixel Art im Retro-Gaming-Stil.

STIL:
- [8-BIT / 16-BIT / 32-BIT / MODERN PIXEL ART]
- Ära: [NES / SNES / GAME BOY / ARCADE / MODERN INDIE]

MOTIV:
- [CHARAKTER / LANDSCHAFT / OBJEKT / SCENE / UI ELEMENT]
- Beschreibung: [DETAILLIERTE BESCHREIBUNG]

FARBSCHEMA:
- [LIMITIERT (wie Game Boy) / ERWEITERT / VOLL]
- Primärfarben: [FARBEN]
- Palette: [ORIGINAL NES / CUSTOM / MONOCHROM]

DETAILS:
- Größe: [PIXEL DIMENSIONEN]
- Perspektive: [TOP-DOWN / SIDE / ISOMETRIC]
- Animation: [STATISCH / ANIMIERT]

STIL-MERKMALE:
- Sichtbare einzelne Pixel
- Limited color palette
- Dithering (ja/nein)
- Outlines (ja/nein)
- Anti-aliasing (nein bei echtem Pixel Art)

REFERENZEN:
- [STARDEW VALLEY / UNDERTALE / CELESTE / FINAL FANTASY / ZELDA]
- Ähnlich wie: [KÜNSTLER/STIL]

TECHNISCH:
- Transparent background (ja/nein)
- Für Game-Engine (Unity, Godot)
- Sprite-Sheet (falls animiert)
- Integer scaling
```

## Anwendung

**Für:** Indie-Games, Retro-Designs, Icons, Animationen

**Beispiel-Output:** 16-Bit Charakter-Sprite, 32x32 Pixel, limitierte Farbpalette, Walk-Animation, transparent, im Stil von Stardew Valley

## Variationen

- Für Landschaften: "Pixel Art Landschaft/Background"
- Für UI: "Pixel Art UI-Elemente"
- Für Portraits: "Pixel Art Charakter-Portraits"
- Für Items: "Pixel Art Inventory-Items"
