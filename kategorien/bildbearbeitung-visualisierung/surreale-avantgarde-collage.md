# Surreale Avantgarde Collage

## Prompt

```
Verwandle das Motiv in das zentrale Element einer surrealen Avantgarde-Collage im Stil moderner Mixed-Media-Editorial-Illustration. Das Porträt wird mit unerwarteten Elementen kombiniert – architektonische Fragmente, Naturformen, abstrakte Texturen, Vintage-Papier.

Die Komposition ist schichtweise aufgebaut: das Gesicht als Hauptelement, umgeben von und durchdrungen von surrealen Objekten – vielleicht ein Schmetterling als Auge, Wolken als Haar, architektonische Säulen als Hals. Die Farbpalette ist ungewöhnlich aber harmonisch.

Texturen von altem Papier, Zeitungsausschnitten, Farbspritzern und handgezeichneten Elementen verleihen Tiefe. Die Stimmung ist traumhaft, intellektuell und visuell fesselnd – wie ein Plakat für eine Kunstausstellung oder ein Titelbild für ein alternatives Magazin.
```

## Anwendung

**Für:** Künstlerische Projekte, Editorial-Design, kreative Porträts

**Input:** Ein Foto einer Person

**Output:** Surreale Collage

## Variationen

- Für Dada: "Absurd, zufällig, provokant"
- Für Retro: "50er Jahre, Vintage, Nostalgie"
- Für Futuristisch: "Technologie, Fragmentierung, Geschwindigkeit"
