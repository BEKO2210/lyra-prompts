# Vintage-Foto-Restauration

## Prompt
```
Rolle: Professioneller Foto-Restaurator und Digitalkünstler
Kontext: Ich habe ein altes Foto [BESCHREIBUNG/BILD HOCHLADEN], das restauriert werden soll
Zustand des Fotos: [VERBLASST/BESCHÄDIGT/KRATZER/FARBVERLUST]
Entstehungszeit: [JAHRZEHNT/JAHR]
Gewünschtes Ergebnis: [NATÜRLICH/FARBIG/KÜNSTLERISCH]
Aufgabe: Erstelle eine Restaurierungs-Anleitung
Einschränkungen:
- Respektiere historische Authentizität
- Schritt-für-Schritt für verschiedene Skill-Levels
- Open-Source oder gängige Software
Ausgabe: Schritt-für-Schritt Workflow, benötigte Tools (GIMP/Photoshop/AI-Tools), Techniken für spezifische Schäden, Farbkorrektur-Guide, Export-Einstellungen für Archivierung
```

## Anwendung
**Input:**
- Foto: Schwarz-Weiß Familienfoto aus den 1950ern
- Zustand: Kratzer, Flecken, Kontrastverlust
- Ziel: Natürliche Wiederherstellung
- Software: Photoshop verfügbar

**Output:** AI liefert: Scan-Empfehlungen (600dpi), Staub/Kratzer-Entfernung (Healing Brush), Kontrast-Anpassung per Levels, selektive Schärfung, Schwarz-Weiß-Abstimmung

## Variationen
- Für KI-Restauration: "MyHeritage/Remini vs manuelle Methode"
- Für Farbgebung: "KI-Farbe vs historisch recherchierte Farben"
- Für beschädigte Bereiche: "Rekonstruktionstechniken für fehlende Teile"
