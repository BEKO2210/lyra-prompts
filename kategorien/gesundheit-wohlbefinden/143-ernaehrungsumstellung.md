# Ernährungsumstellung planen

## Prompt

```
Rolle: Ernährungsberaterin
Kontext: Ich möchte meine Ernährung umstellen
Ziel: [ABNEHMEN / ZUNEHMEN / GESÜNDER ESSEN / SPORTLICHE LEISTUNG / KRANKHEIT]
Aktuelle Ernährung: [OMNIVOR / VEGETARISCH / VEGAN / FLEXIBEL]
Einschränkungen: [ALLERGIEN / UNVERTRÄGLICHKEITEN / RELIGIÖS]
Kochkenntnisse: [ANFÄNGER / FORTGESCHRITTEN / GEÜBT]
Zeitbudget: [MINUTEN PRO TAG FÜR ESSEN]
Budget: [NIEDRIG / MITTEL / HOCH]
Aufgabe: Erstelle einen realistischen Umstellungsplan

Ausgabe:
1. Ziel-Analyse:
   - Realistisches Zielgewicht/-zusammensetzung
   - Zeitrahmen (gesund: 0,5kg/Woche)
   - Kalorienbedarf berechnen
   - Makro-Verteilung
2. Ernährungsstil wählen:
   - Mediterran, Low-Carb, Vegan, etc.
   - Was passt zu meinem Leben?
   - Flexibel vs. streng
3. Einkaufs-Strategie:
   - Wochenplanung
   - Einkaufsliste Templates
   - Meal Prep (Vorkochen)
   - Budget-Tipps
4. Rezepte:
   - 7 Frühstücks-Ideen
   - 7 Mittagessen (schnell)
   - 7 Abendessen
   - Snacks (gesund)
5. Herausforderungen:
   - Heißhunger-Attacken
   - Essen unterwegs
   - Soziale Situationen
   - Motivationstiefs
6. Tracking:
   - App-Empfehlungen
   - Ohne App tracken?
   - Fortschritts-Messung
   - Anpassungen
7. Langfristigkeit:
   - 80/20 Regel
   - Cheat Meals
   - Gewohnheiten etablieren
   - Wartung nach Zielerreichung
```

## Beispiel

**Input:** "Gesünder essen + leicht abnehmen, omnivor, keine Einschränkungen, fortgeschritten, 45 Min/Tag, mittel"

**Output:**
- Ziel: -5kg in 3 Monaten
- Stil: Mediterran mit Portionenkontrolle
- Meal Prep: Sonntags für 3 Tage vorkochen
- Tracking: MyFitnessPal oder Cronometer

## Variationen

- Für Sportler: "Ernährung für Muskelaufbau"
- Für Veganer: "Vegane Ernährungsumstellung"
- Für Allergiker: "Ernährung bei [Allergie]"
- Für Familien: "Gesunde Familienküche"
