# Jogging/Running für Anfänger

## Prompt

```
Rolle: Laufcouch mit 10+ Jahren Erfahrung
Kontext: Ich möchte mit dem Laufen beginnen
Fitness-Level: [SEHR WENIG BEWEGUNG / ETWAS SPORT / REGELMÄßIG]
Ziel: [5KM / 10KM / ABNEHMEN / FITNESS / HALBMARATHON]
Gesundheit: [GESUND / ÜBERGEWICHT / GELENKPROBLEME / HERZ]
Zeit: [WIE OFT PRO WOCHE / MINUTEN PRO SESSION]
Ort: [DRAUßEN / LAUFBAND / BEIDES]
Aufgabe: Erstelle einen sicheren Einstiegsplan

Ausgabe:
1. Vorbereitung:
   - Laufschuhe (was ist wichtig?)
   - Kleidung (Layer-Prinzip)
   - Technik (Grundlagen)
   - Aufwärmen und Dehnen
2. 8-Wochen-Plan:
   - Woche 1-2: Gehen mit kurzen Lauf-Intervallen
   - Woche 3-4: Intervall-Training
   - Woche 5-6: Längere Läufe
   - Woche 7-8: Kontinuierliches Laufen
3. Technik:
   - Laufschritt (Cadence)
   - Atmung (Rhythmus)
   - Haltung
   - Arm-Schwung
4. Verletzungsprävention:
   - Anfang langsam!
   - 10%-Regel (Steigerung)
   - Regeneration
   - Warnsignale (Schmerzen)
5. Motivation:
   - Apps (Strava, Nike Run Club)
   - Musik/Podcasts
   - Lauftreffs
   - Ziele setzen
6. Ernährung:
   - Vor dem Lauf
   - Während des Laufs
   - Nach dem Lauf
   - Hydration
7. Fortschritt:
   - Wann 5km schaffen?
   - Nächste Ziele
   - Wettkampf-Anmeldung?
```

## Beispiel

**Input:** "Wenig Bewegung, 5km Ziel, gesund, 3x/Woche 30 Min, draußen"

**Output:**
- Woche 1: 30 Sek laufen, 2 Min gehen (wiederholen)
- Woche 4: 2 Min laufen, 1 Min gehen
- Woche 8: 5km durchlaufen
- Schuhe: Fachhandel, Laufanalyse empfohlen

## Variationen

- Für Übergewicht: "Laufen mit Übergewicht"
- Für 10km: "Von 5km auf 10km"
- Für Halbmarathon: "Halbmarathon-Trainingsplan"
- Für Laufband: "Laufband-Training für Anfänger"
