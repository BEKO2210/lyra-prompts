# Stressmanagement im Beruf

## Prompt

```
Rolle: Betriebspsychologin und Stressmanagement-Coach
Kontext: Ich habe beruflichen Stress
Stressoren: [WORKLOAD / KOLLEGEN / CHEF / UNSICHERHEIT / KONFLIKTE]
Symptome: [SCHLAFSTÖRUNGEN / REIZBARKEIT / MÜDIGKEIT / KÖRPERLICH]
Dauer: [WIE LANGE SCHON?]
Arbeitsumfeld: [BÜRO / REMOTE / SCHICHT / KUNDE-KONTAKT]
Unterstützung: [KANN OFFEN SPRECHEN / MUSS SELBST LÖSEN]
Aufgabe: Entwickle ein Stressmanagement-Programm

Ausgabe:
1. Stress-Analyse:
   - Stressoren identifizieren (intern vs. extern)
   - Eustress vs. Distress
   - Persönliche Stress-Signale
   - Auslöser erkennen
2. Sofortmaßnahmen (akut):
   - 3-Minuten-Atemtechnik
   - Progressiver Muskelrelaxation
   - Mini-Pause
   - Cognitive Reframing
3. Tagesstruktur:
   - Morgen-Routine
   - Arbeitsphasen planen (Pomodoro)
   - Pausen einbauen (wirklich!)
   - Grenzen setzen (Arbeitszeiten)
4. Kommunikation:
   - "Nein" sagen
   - Delegieren
   - Erwartungen klären
   - Schwierige Gespräche
5. Work-Life-Balance:
   - Abschalten nach der Arbeit
   - Hobbys pflegen
   - Soziale Kontakte
   - Digitale Detox
6. Organisation:
   - Priorisierung (Eisenhower)
   - Zeitmanagement
   - To-Do-Listen (sinnvoll nutzen)
   - Multitasking vermeiden
7. Wann professionell:
   - Burnout-Symptome
   - Therapie-Angebote
   - Betriebsärztlich
   - Coaching
```

## Beispiel

**Input:** "Workload + Chef, Schlafstörungen + Reizbarkeit, seit 6 Monaten, Büro, kann offen sprechen"

**Output:**
- Analyse: Zu viele Projekte, unklare Prioritäten
- Sofort: Box-Breathing bei Stress-Momenten
- Struktur: Pomodoro, feste Feierabend-Zeit
- Kommunikation: Gespräch mit Chef über Prioritäten
