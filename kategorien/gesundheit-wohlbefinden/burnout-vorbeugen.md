# Burnout vorbeugen

## Prompt

```
Rolle: Betriebspsychologe für Stressprävention
Kontext: Ich fühle mich ausgebrannt oder will Burnout vermeiden
Situation: [BERUF / PRIVAT / BEIDES]
Warnzeichen: [MÜDIGKEIT / GLEICHGÜLTIGKEIT / LEISTUNGSABFALL]
Arbeitsumfeld: [BELASTEND / UNTERSTÜTZEND / GEMISCHT]
Aufgabe: Erstelle einen Präventionsplan
Ausgabe:
1. Frühwarnsystem (eigene Warnzeichen erkennen)
2. Grenzen setzen (Nein sagen, Arbeitszeit)
3. Erholungsphasen (Mikropausen, Wochenende, Urlaub)
4. Unterstützung finden (Therapie, Coaching, Gespräche)
5. Langfristige Strategie (Karriere, Lebensziele)
```

## Anwendung

**Für:** Burnout-Prävention und Erholung

**Input:** "Beruf, Müdigkeit+Gleichgültigkeit, belastend"

**Output:** Präventionsstrategie

## Variationen

- Für Führungskräfte: "Vorbild sein, Team schützen"
- Für Selbstständige: "Struktur, Routine, Auszeit"
- Für Pflegende: "Empathie-Fatigue, Selbstfürsorge"
