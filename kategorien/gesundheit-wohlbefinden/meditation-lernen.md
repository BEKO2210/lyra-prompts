# Meditation lernen

## Prompt

```
Rolle: Meditationslehrer für Anfänger
Kontext: Ich will Meditation lernen
Grund: [STRESS / SCHLAF / FOKUS / NEUGIER]
Zeit: [MINUTEN] pro Tag
Ort: [ZU HAUSE / UNTERWEGS / BÜRO]
Erfahrung: [NOCH NIE / EIN WENIG]
Aufgabe: Erstelle einen Einstiegsplan
Ausgabe:
1. Meditationstechniken (Atmung, Body Scan, Walking)
2. Tägliche Routine (wann, wie lange, wo)
3. Apps und Ressourcen (Headspace, Calm, etc.)
4. Häufige Hindernisse (Gedanken, Unruhe, Schlaf)
5. Fortschritt messen (wie merke ich Erfolge?)
```

## Anwendung

**Für:** Meditationseinstieg

**Input:** "Stress, 10 Min, zu Hause, noch nie"

**Output:** Schritt-für-Schritt Lernplan

## Variationen

- Für Skeptiker: "Wissenschaftliche Basis"
- Für Spirituelle: "Tiefe Praxis, Retreats"
- Für Kinder: "Spielerisch, kurz"
