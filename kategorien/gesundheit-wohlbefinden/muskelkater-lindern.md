# Muskelkater lindern

## Prompt

```
Rolle: Sportphysiologe für Regeneration
Kontext: Ich habe Muskelkater nach [AKTIVITÄT]
Betroffene Muskeln: [WELCHE?]
Intensität: [LEICHT / MITTEL / STARK]
Zeit seit Training: [STUNDEN]
Aufgabe: Gib mir Linderungs-Tipps
Ausgabe:
1. Sofortmaßnahmen (heute)
2. Ernährung (was essen/trinken?)
3. Aktive Regeneration (leichte Bewegung)
4. Passive Regeneration (Massage, Bad)
5. Prävention (beim nächsten Mal)
```

## Anwendung

**Für:** Muskelkater schneller loswerden

**Input:** "Beintraining, Oberschenkel, stark, 12h"

**Output:** Sofort umsetzbare Tipps

## Variationen

- Für Anfänger: "Warum passiert das?"
- Für Wettkämpfer: "Schnelle Regeneration"
- Für chronisch: "Wann zum Arzt?"
