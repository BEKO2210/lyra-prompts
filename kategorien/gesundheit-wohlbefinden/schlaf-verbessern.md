# Schlaf verbessern

## Prompt

```
Rolle: Schlafberater
Kontext: Ich habe Probleme mit [EINSCHLAFEN / DURCHSCHLAFEN / QUALITÄT]
Aktuelle Situation:
- Schlafenszeit: [UHRZEIT]
- Aufstehzeit: [UHRZEIT]
- Vor dem Schlafen: [ROUTINE / BILDSCHIRME / ESSEN]
- Morgens: [MÜDE / FIT / VARIIERT]
Aufgabe: Optimiere meinen Schlaf
Ausgabe:
1. Abend-Routine (30-60 Min vor Schlafen)
2. Schlafumgebung (Licht, Temperatur, Geräusche)
3. Morgens-Routine (für bessere Müdigkeit am Abend)
4. 3 konkrete Änderungen (starte mit der wichtigsten)
5. Tracking-Methode
```

## Beispiel

**Input:** Einschlafprobleme, 23:00-07:00, Handy bis spät, morgens müde

**Output:** 
- Routine: Lesen statt Handy, 22:30 Start
- Umgebung: 18°C, Verdunkelung, White Noise
- Änderung: Handy aus dem Schlafzimmer verbannen

## Variationen

- Für Schichtarbeiter: "Irreguläre Zeiten managen"
- Für Eltern: "Mit Baby/Kind schlafen"
- Für Ängste: "Gedanken vor dem Schlafen beruhigen"
