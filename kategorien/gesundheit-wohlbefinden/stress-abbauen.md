# Stress abbauen

## Prompt

```
Rolle: Stressmanagement-Coach
Kontext: Ich fühle mich überlastet
Auslöser: [KONKRETE SITUATION oder ALLGEMEIN]
Symptome: [SCHLAF / REIZBARKEIT / KONZENTRATION / KÖRPERLICHES]
Verfügbare Zeit für Maßnahmen: [MINUTEN pro Tag]
Aufgabe: Erstelle einen Entlastungsplan
Ausgabe:
1. Sofortmaßnahme (für akute Momente)
2. Tägliche Mini-Routine (5-10 Min)
3. Wöchentliche Auszeit (länger, planbar)
4. Grenzen setzen (Was kann ich ablehnen?)
5. Unterstützung finden (Professionell / Sozial)
```

## Beispiel

**Input:** Arbeit + Privat, Schlafstörungen, Reizbarkeit, 15 Min/Tag

**Output:** 
- Sofort: 4-7-8 Atmung, kaltem Wasser ins Gesicht
- Täglich: 10 Min Meditation oder Spaziergang
- Wöchentlich: Ein Abend nur für mich

## Variationen

- Für Prüfungsstress: "Lernen + Entspannung balancieren"
- Für Jobstress: "Work-Life-Grenzen"
- Für Familienstress: "Zeit für sich selbst finden"
