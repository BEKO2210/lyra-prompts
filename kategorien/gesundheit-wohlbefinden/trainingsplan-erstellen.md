# Trainingsplan erstellen

## Prompt

```
Rolle: Fitnesstrainer für Alltagssportler
Kontext: Ich will fitter werden
Aktueller Stand: [ANFÄNGER / WIEDEREINSTEIGER / AKTIV]
Ziel: [ABNEHMEN / MUSKELAUFBAU / AUSDAUER / BEWEGlichkeit / ALLGEMEIN FIT]
Verfügbare Zeit: [MINUTEN] pro Tag, [TAGE] pro Woche
Equipment: [NUR Körpergewicht / Hanteln / Fitnessstudio / Outdoor]
Einschränkungen: [VERLETZUNGEN, GESUNDHEITLICHES]
Aufgabe: Erstelle einen Trainingsplan
Ausgabe:
1. Wochenübersicht (Welcher Tag was?)
2. Konkrete Übungen (mit Alternativen)
3. Progression (Wann steigern?)
4. Ernährungshinweise (falls relevant)
5. Motivations-System (Tracking, Belohnungen)
```

## Beispiel

**Input:** Anfänger, Muskelaufbau, 30 Min/Tag, 4x/Woche, Hanteln zuhause

**Output:** 
- Push/Pull/Legs/Full Body Split
- Übungen: Liegestütze, Kniebeugen, Rudern
- Progression: +1 Wiederholung pro Woche

## Variationen

- Für Abnehmen: "Cardio + Kraft kombiniert"
- Für Läufer: "5k, 10k, Halbmarathon"
- Für Zuhause: "Kein Equipment nötig"
