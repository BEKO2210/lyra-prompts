# Beziehung auf Distanz (Fernbeziehung)

## Prompt

```
Rolle: Paartherapeutin mit Erfahrung in Fernbeziehungen
Kontext: Wir führen eine Fernbeziehung oder planen eine
Distanz: [KM / ZEITZONE]
Dauer: [BEFRISTET / UNBEFRISTET]
Häufigkeit Treffen: [WIE OFT?]
Kommunikation: [VIDEO / TELEFON / TEXT / MIX]
Zukunft: [PLAN ZUSAMMENZUZIEHEN / UNKLAR]
Aufgabe: Tipps für eine gesunde Fernbeziehung

Ausgabe:
1. Kommunikation:
   - Feste Zeiten vs. spontan
   - Video-Calls (Qualität vs. Quantität)
   - Text-Nachrichten (Erwartungen klären)
   - Zeitverschiebung handhaben
2. Vertrauen:
   - Eifersucht managen
   - Transparenz vs. Kontrolle
   - Unabhängigkeit bewahren
   - Gemeinsame Ziele
3. Treffen planen:
   - Wie oft? (realistisch)
   - Wer fährt?
   - Qualität der Zeit
   - Abschiede handhaben
4. Alltag teilen:
   - Virtuelle Dates
   - Gleichzeitig Filme schauen
   - Online-Spiele
   - Gemeinsame Projekte
5. Konflikte:
   - Streit auf Distanz
   - Missverständnisse vermeiden
   - Wann persönlich klären?
   - Versöhnung
6. Zukunft planen:
   - Enddatum definieren
   - Zusammenziehen planen
   - Kompromisse finden
   - Wann aufgeben?
7. Warnsignale:
   - Wachsende Distanz (emotional)
   - Kommunikation bricht ab
   - Andere Personen
   - Ungleiches Engagement
```

## Beispiel

**Input:** "400km, 1 Jahr befristet, alle 2-3 Wochen, Video täglich, Plan zu zusammenziehen"

**Output:**
- Kommunikation: Täglich 30 Min Video, Text zwischendurch
- Treffen: Abwechselnd fahren, lange Wochenenden
- Dates: Online-Kochabende, parallel Netflix
- Zukunft: Monatlich Check-in zum Plan

## Variationen

- Für international: "Internationale Fernbeziehung"
- Für verheiratet: "Fernbeziehung als Ehepaar"
- Für neu: "Neue Beziehung auf Distanz"
- Für Ende: "Fernbeziehung beenden"
