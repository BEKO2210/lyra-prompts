# Beziehungspflege

## Prompt

```
Rolle: Beziehungscoach für langfristige Partnerschaften
Kontext: Ich will meine Beziehung zu [PERSON] stärken
Beziehungsstatus: [NEU / ETABLIERT / KRISENHAFT]
Herausforderung: [ZEITMANGEL / KOMMUNIKATION / ROUTINE]
Ziel: [MEHR NÄHE / VERTRAUEN / QUALITÄTSZEIT]
Aufgabe: Erstelle einen Beziehungsplan
Ausgabe:
1. Wöchentliche Ritualvorschläge (kleine Gesten)
2. Monatliche Aktivitäten (gemeinsame Erlebnisse)
3. Kommunikationsübungen (aktives Zuhören, Wertschätzung)
4. Konfliktlösungsstrategien (fair streiten)
5. Langfristige Ziele (gemeinsame Vision)
```

## Anwendung

**Für:** Beziehung aktiv pflegen

**Input:** "Partner, etabliert, Zeitmangel, mehr Nähe"

**Output:** Konkreter Beziehungsplan

## Variationen

- Für Fernbeziehung: "Virtuelle Nähe schaffen"
- Für Familie: "Eltern, Geschwister stärken"
- Für Freundschaft: "Alte Freundschaften beleben"
