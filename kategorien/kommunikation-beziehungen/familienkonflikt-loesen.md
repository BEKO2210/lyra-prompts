# Familienkonflikt lösen

## Prompt

```
Rolle: Familientherapeut für konstruktive Konfliktlösung
Kontext: Es gibt einen Konflikt in meiner Familie um [THEMA]
Betroffene: [WER IST INVOLVIERT?]
Dauer: [SEIT WANN?]
Versuche bisher: [WAS HABT IHR SCHON PROBIERT?]
Ziel: [WAS WOLLEN ALLE ERREICHEN?]
Aufgabe: Finde einen Lösungsweg
Ausgabe:
1. Perspektivenwechsel (jeder erzählt seine Sicht)
2. Gemeinsame Interessen (was verbindet euch?)
3. Konkrete Lösungsvorschläge (3 Optionen)
4. Familienregeln (für die Zukunft)
5. Nachsorge (wie bleibt es friedlich?)
```

## Anwendung

**Für:** Familiäre Konflikte lösen

**Input:** "Erbe, Geschwister, 2 Jahre, wenig gesprochen, faire Lösung"

**Output:** Lösungsstrategie

## Variationen

- Für Generationen: "Eltern, Kinder, Enkel"
- Für Scheidung: "Trennung, Kinder, Umgang"
- Für Alltag: "Haushalt, Pflichten, Respekt"
