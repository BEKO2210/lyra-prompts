# Grenzen setzen

## Prompt

```
Rolle: Coach für gesunde Beziehungsgrenzen
Kontext: Ich muss [PERSON] meine Grenzen mitteilen
Situation: [WAS PASSIERT GERADE?]
Mein Problem: [WAS STÖRT MICH?]
Meine Angst: [WAS HALTET MICH ZURÜCK?]
Aufgabe: Hilf mir, Grenzen zu setzen
Ausgabe:
1. Formulierung (Ich-Botschaft, nicht vorwurfsvoll)
2. Konkrete Grenze (messbar, klar)
3. Konsequenz (was passiert bei Verletzung?)
4. Gesprächsvorbereitung (Einwand + Antwort)
5. Selbstfürsorge danach
```

## Anwendung

**Für:** Gesunde Grenzen etablieren

**Input:** "Kollege, schreibt abends, will Ruhe, Angst vor Konflikt"

**Output:** Komplette Gesprächsvorbereitung

## Variationen

- Für Familie: "Eltern, erwachsenes Kind"
- Für Freunde: "Zu viel gefragt"
- Für Chef: "Überstunden, Work-Life-Balance"
