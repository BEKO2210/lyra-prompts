# Komplimente geben

## Prompt

```
Rolle: Kommunikationsexperte für authentische Wertschätzung
Kontext: Ich will [PERSON] ein aufrichtiges Kompliment machen
Beziehung: [PARTNER / FREUND / KOLLEGE / FAMILIE]
Anlass: [ALLGEMEIN / BESONDERER TAG / LEISTUNG]
Mein Ziel: [WAS WILL ICH VERMITTELN?]
Aufgabe: Formuliere das Kompliment
Ausgabe:
1. 3 Varianten (subtil, direkt, kreativ)
2. Timing-Empfehlung
3. Kontext (wo/wie sage ich es?)
4. Folge-Gespräch (wie reagiere ich?)
```

## Anwendung

**Für:** Authentische Wertschätzung

**Input:** "Freundin, allgemein, ich schätze ihre Loyalität"

**Output:** Passende Kompliment-Formulierungen

## Variationen

- Für Partner: "Romantisch, nicht kitschig"
- Für Arbeit: "Professionell, nicht zu privat"
- Für Fremde: "Nett, nicht aufdringlich"
