# Schwierige Nachricht formulieren

## Prompt

```
Rolle: Kommunikationsexperte für sensible Themen
Kontext: Ich muss [PERSON] sagen, dass [SCHWIERIGE NACHRICHT]
Beziehung: [FAMILIE / FREUND / PARTNER / KOLLEGE]
Mein Gefühl: [WIE FÜHLE ICH MICH?]
Wichtig: [WAS SOLL AM ENDE ÜBRIG BLEIBEN?]
Aufgabe: Formuliere die Nachricht
Ausgabe:
1. 3 Varianten (direkt, diplomatisch, emotional)
2. Timing-Empfehlung (Wann am besten?)
3. Kanalwahl (Persönlich, Telefon, Text?)
4. Vorbereitung auf die Reaktion
```

## Beispiel

**Input:** Freundin, kann nicht zur Hochzeit kommen, traurig, Beziehung erhalten

**Output:** 
- Direkt: "Ich muss dir leider sagen..."
- Diplomatisch: "Ich habe lange überlegt..."
- Emotional: "Es bricht mir das Herz..."

## Variationen

- Für Absagen: "Nein sagen ohne zu verletzen"
- Für Trennungen: "Beziehung beenden"
- Für Kritik: "Konstruktives Feedback"
