# Smalltalk meistern

## Prompt

```
Rolle: Kommunikationstrainer für Networking
Kontext: Ich bin auf [ANLASS] und muss Smalltalk machen
Situation: [KENNE NIEMAND / CHEF DA / VERLEGEN]
Ziel: [NETZWERKEN / SYMPATHISCH WIRKEN / INFORMATIONEN]
Dauer: [WIE LANGE DAUERT DAS EVENT?]
Aufgabe: Gib mir Smalltalk-Strategien
Ausgabe:
1. Einstiegsfragen (nicht: "Was machst du?")
2. Gesprächsthemen (sicher, interessant, aktuell)
3. Aktives Zuhören (Signalwörter, Nachfragen)
4. Ausstieg (elegant beenden, Kontakt halten)
5. Follow-up (LinkedIn, E-Mail, Treffen)
```

## Anwendung

**Für:** Bessere Smalltalk-Fähigkeiten

**Input:** "Firmen-Event, kenne niemanden, sympathisch wirken, 2 Stunden"

**Output:** Praktische Smalltalk-Taktiken

## Variationen

- Für Introvertierte: "Energie schonen, Qualität vor Quantität"
- Für Online: "Virtuelles Networking"
- Für Fremdsprache: "Einfache Phrasen, Verständnis"
