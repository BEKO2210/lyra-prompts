# YouTube Kanal starten

## Prompt

```
Rolle: YouTube-Strategin mit mehreren erfolgreichen Kanälen
Kontext: Ich möchte einen YouTube-Kanal starten
Nische: [THEMA/BEREICH]
Zielgruppe: [WER SOLL ZUSCHAUEN?]
Format: [TUTORIALS / VLOG / REVIEWS / UNTERHALTUNG / BILDUNG]
Verfügbare Zeit: [STUNDEN PRO WOCHE]
Budget: [FÜR EQUIPMENT]
Ziel: [HOBBY / NEBENEINKOMMEN / HAUPTEINKOMMEN]
Aufgabe: Erstelle einen kompletten Kanal-Launch-Plan

Ausgabe:
1. Kanal-Positionierung:
   - Einzigartiger Winkel (warum sollte jemand MIR zuschauen?)
   - Wertversprechen für Zuschauer
   - Namensideen (10 Vorschläge)
   - Branding (Logo, Banner, Thumbnail-Stil)
2. Content-Strategie:
   - Erste 20 Video-Ideen (mit Suchvolumen-Einschätzung)
   - Content-Pillars (3-4 Hauptthemen)
   - Upload-Rhythmus (realistisch)
   - Evergreen vs. Trending Content
3. Equipment-Setup (nach Budget):
   - Kamera-Optionen (Smartphone bis DSLR)
   - Audio (Mikrofon-Empfehlungen)
   - Beleuchtung
   - Schnitt-Software
4. Video-Produktion:
   - Drehbuch-Vorlage
   - Aufnahme-Workflow
   - Schnitt-Checkliste
   - Thumbnail-Design
5. SEO und Discovery:
   - Keyword-Recherche
   - Titel-Formeln
   - Beschreibungen
   - Tags
6. Wachstumsstrategie:
   - First 1000 Subscribers Plan
   - Community-Aufbau
   - Kollaborationen
   - Cross-Promotion
7. Monetarisierung:
   - YouTube Partner Programm
   - Sponsorships
   - eigene Produkte
   - Zeitplan (wann welche Einnahmequelle?)
```

## Beispiel

**Input:** "Kochen für Studenten, 18-25 Jahre, Tutorials, 10h/Woche, 500€ Budget, Nebeneinkommen"

**Output:**
- Name-Vorschläge: "Studentenküche", "Kochmuffel", "Budget Bites"
- Erste Videos: "5 Gerichte für unter 5€", "Ein-Pfannen-Pasta"
- Equipment: Smartphone + Lavalier-Mikrofon + Ringlicht
- Upload: 1x/Woche, sonntags

## Variationen

- Für Gaming: "Gaming-YouTube Kanal aufbauen"
- Für Business: "B2B YouTube als Marketing-Tool"
- Für bestehende Kanäle: "Stagnierenden Kanal reaktivieren"
- Für Shorts: "YouTube Shorts Strategie"
