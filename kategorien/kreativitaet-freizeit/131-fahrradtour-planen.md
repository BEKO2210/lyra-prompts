# Fahrrad-Tour planen

## Prompt

```
Rolle: Radreise-Experte mit mehreren Europa-Durchquerungen
Kontext: Ich plane eine Fahrradtour
Art: [TAGESAUSFLUG / WOCHENENDE / URLAUB / MEHRTÄGIG]
Strecke: [VON NACH / REGION]
Distanz pro Tag: [KM]
Fitness-Level: [ANFÄNGER / FORTGESCHRITTEN / SPORTLICH]
Unterkunft: [ZELT / HOTEL / PENSION / FREUNDE]
Fahrradtyp: [RENNRAD / MTB / TREKKING / E-BIKE]
Aufgabe: Plane meine Radtour

Ausgabe:
1. Routenplanung:
   - Tools (Komoot, Strava, Bikemap)
   - Streckenprofil (Höhenmeter)
   - Straßen vs. Radwege
   - Sehenswürdigkeiten unterwegs
   - Alternativ-Routen bei schlechtem Wetter
2. Tagesplanung:
   - Realistische Kilometer pro Tag
   - Pausen-Planung
   - Verpflegung unterwegs
   - Tankstellen (für E-Bike)
3. Ausrüstung:
   - Packliste (Werkzeug, Ersatzteile)
   - Bekleidung (Wetter-layers)
   - Navigation (Handy vs. GPS)
   - Sicherheit (Helm, Licht, Reflektoren)
4. Unterkunft:
   - Campingplätze finden
   - Hotels für Radfahrer
   - Bett & Bike Angebote
   - Last-Minute-Alternativen
5. Verpflegung:
   - Kalorienbedarf berechnen
   - Wasser-Versorgung
   - Restaurants unterwegs
   - Notverpflegung
6. Notfall-Plan:
   - Platten reparieren
   - Mechanische Probleme
   - Wetterumschwung
   - Abbruch-Optionen
7. Training:
   - Vorbereitung (wenn nötig)
   - Stretching
   - Regeneration
```

## Beispiel

**Input:** "4 Tage, Bodensee-Radweg, 60-80km/Tag, fortgeschritten, Hotels, Trekkingrad"

**Output:**
- Route: Konstanz → Überlingen → Friedrichshafen → Lindau
- Hotels: Bett & Bike zertifiziert buchen
- Ausrüstung: 2 Schläuche, Multitool, Regenjacke
- Verpflegung: 2.500 kcal/Tag planen

## Variationen

- Für Anfänger: "Erste Mehrtagestour planen"
- Für E-Bike: "E-Bike Tour mit Ladestationen"
- Für MTB: "MTB-Trail-Tour planen"
- Für Fernradweg: "Deutschland-Tour oder EuroVelo"
