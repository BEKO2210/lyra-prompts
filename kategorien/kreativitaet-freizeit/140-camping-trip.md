# Camping-Trip planen

## Prompt

```
Rolle: Outdoor-Experte mit 20 Jahren Camping-Erfahrung
Kontext: Ich plane einen Camping-Trip
Art: [ZELT / WOHNWAGEN / CAMPINGBUS / GLAMPING]
Region: [WO?]
Dauer: [TAGE]
Begleitung: [ALLEIN / PAAR / FAMILIE / FREUNDE]
Erfahrung: [ANFÄNGER / FORTGESCHRITTEN / EXPERTE]
Komfort-Anspruch: [RUSTIKAL / BALANCIERT / KOMFORTABEL]
Aufgabe: Plane den perfekten Camping-Trip

Ausgabe:
1. Zielauswahl:
   - Campingplatz vs. Wildcampen (Rechtslage!)
   - Reservierung nötig?
   - Ausstattung des Platzes
   - Umgebung (Wandern, See, Stadt)
2. Ausrüstung:
   - Zelt (Größe, Saison)
   - Schlafsystem (Schlafsack, Isomatte)
   - Koch-Setup
   - Licht, Werkzeug, Erste Hilfe
   - Packliste (nach Wetter)
3. Verpflegung:
   - Mahlzeiten planen
   - Kühlung (Kühlbox, passive Kühlung)
   - Wasser (Mitbringen, aufbereiten)
   - Camping-Rezepte
4. Sicherheit:
   - Wetter beobachten
   - Notfall-Plan
   - Erste-Hilfe-Kurs (empfohlen)
   - Ticks, Schlangen, gefährliche Tiere
5. Aktivitäten:
   - Wanderrouten
   - Wassersport
   - Lagerfeuer (Regeln!)
   - Spiele (bei Regen)
6. Nachhaltigkeit:
   - Leave No Trace Prinzipien
   - Müllentsorgung
   - Feuer machen (verantwortungsvoll)
   - Natur schützen
7. Checkliste:
   - Vor der Abfahrt
   - Ankunft am Platz
   - Abreise
```

## Beispiel

**Input:** "Zelt, Schwarzwald, 5 Tage, Paar, Anfänger, balanciert"

**Output:**
- Platz: Campingplatz mit Sanitär, reservieren
- Zelt: 3-Personen-Kuppelzelt
- Essen: Frühstück selbst, abends Grillen
- Ausrüstung: Komplett-Liste mit Checkliste

## Variationen

- Für Familien: "Camping mit Kindern"
- Für Wildcampen: "Wildcampen in Skandinavien"
- Für Winter: "Winter-Camping Guide"
- Für Festivals: "Festival-Camping optimieren"
