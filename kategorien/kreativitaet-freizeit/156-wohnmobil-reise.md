# Wohnmobil/Campervan-Reise planen

## Prompt

```
Rolle: Vanlife-Experte mit 50.000+ km Wohnmobil-Erfahrung
Kontext: Ich plane eine Wohnmobil/Camper-Reise
Fahrzeug: [EIGENES / GEMIETET / CAMPERVAN / WOHNMOBIL]
Region: [EUROPA / DEUTSCHLAND / SKANDINAVIEN / MEDITERRAN]
Dauer: [WOCHENENDE / 1 WOCHE / 2+ WOCHEN]
Reisebegleitung: [ALLEIN / PAAR / FAMILIE / FREUNDE]
Komfort-Level: [RUSTIKAL / BALANCIERT / LUXURIÖS]
Aufgabe: Plane die perfekte Camper-Reise

Ausgabe:
1. Fahrzeug-Vorbereitung:
   - Checkliste vor Abfahrt
   - Technische Kontrolle
   - Versicherungen (Ausland?)
   - Mieten: Was beachten?
2. Routenplanung:
   - Tagesetappen (realistisch)
   - Stellplätze vs. Campingplätze
   - Apps und Karten (Park4Night, etc.)
   - Flexibilität einplanen
3. Stellplätze:
   - Kostenlos (Wildcampen - Rechtslage!)
   - Campingplätze (Komfort vs. Natur)
   - Stellplätze (preiswert)
   - Reservierung nötig?
4. Ausrüstung:
   - Küche (Kochen im Camper)
   - Strom (Solar, Batterie, Landstrom)
   - Wasser (Tanken, sparen)
   - Bad (WC, Dusche)
5. Verpflegung:
   - Einkaufen unterwegs
   - Kühlen (Kompressor vs. Absorber)
   - Einfache Camper-Rezepte
   - Restaurants (Budget)
6. Sicherheit:
   - Nachts sicher schlafen
   - Wertgegenstände
   - Erste Hilfe
   - Pannenhilfe
7. Destinationen:
   - Geheimtipps
   - Touristenfallen vermeiden
   - Saisonale Besonderheiten
   - Grenzübertritte (Dokumente)
```

## Beispiel

**Input:** "Gemietet, Norwegen, 2 Wochen, Paar, balanciert"

**Output:**
- Route: Oslo → Bergen → Fjorde → Lofoten
- Stellplätze: Mix aus Natur-Stellplätzen und Campingplätzen
- Ausrüstung: Kühltasche, Campingkocher, Outdoor-Kleidung
- Budget: 150€/Tag (inkl. Miete, Diesel, Plätze)

## Variationen

- Für Familien: "Camper-Reise mit Kindern"
- Für Europa: "Europa-Rundreise im Camper"
- Für Winter: "Winter-Camping im Wohnmobil"
- Für Digital Nomaden: "Als Digital Nomad im Van arbeiten"
