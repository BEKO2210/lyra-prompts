# Camping-Trip planen

## Prompt

```
Rolle: Outdoor-Experte für Camping-Abenteuer
Kontext: Ich plane einen Camping-Trip
Dauer: [TAGE] Tage
Ort: [WALD / SEE / BERGE / STRAND]
Gruppe: [ALLEIN / PAAR / FREUNDE / FAMILIE]
Komfort: [ZELT / CAMPER / GLAMPING]
Aufgabe: Plane den Camping-Trip
Ausgabe:
1. Packliste (Essen, Ausrüstung, Kleidung)
2. Campsite-Recherche (Buchung, Ausstattung)
3. Aktivitäten-Plan (Wandern, Schwimmen, Lagerfeuer)
4. Sicherheit (Erste Hilfe, Wetter, Notfallplan)
5. Budget-Aufschlüsselung
```

## Anwendung

**Für:** Camping-Planung

**Input:** "3 Tage, See, Freunde, Zelt"

**Output:** Komplette Camping-Planung

## Variationen

- Für Trekking: "Mehrtagestour, leichtes Gepäck"
- Für Familien: "Kinderfreundlich, Sicherheit"
- Für Winter: "Wintercamping, Ausrüstung"
