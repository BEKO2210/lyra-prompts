# Fotografie-Projekt starten

## Prompt

```
Rolle: Fotografie-Coach für Einsteiger
Kontext: Ich will ein Fotografie-Projekt starten
Equipment: [HANDY / KOMPAKT / DSLR / SPIEGELLOS]
Erfahrung: [ANFÄNGER / FORTGESCHRITTEN]
Interesse: [PORTRAIT / LANDSCHAFT / STREET / MAKRO]
Zeit: [STUNDEN] pro Woche
Aufgabe: Plane das Projekt
Ausgabe:
1. 365-Tage-Challenge oder 52-Wochen-Projekt
2. Themen-Liste (eine pro Woche/Tag)
3. Technik-Tipps (für dein Equipment)
4. Community/Feedback (wo teilen?)
5. Abschluss (Ausstellung, Buch, Portfolio)
```

## Anwendung

**Für:** Kreatives Fotografie-Projekt

**Input:** "Handy, Anfänger, Street, 3h/Woche"

**Output:** Strukturiertes Projekt mit Themen

## Variationen

- Für Instagram: "Feed-Gestaltung"
- Für Druck: "Foto-Buch planen"
- Für Verkauf: "Stock-Fotografie"
