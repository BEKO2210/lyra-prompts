# Garten anlegen

## Prompt

```
Rolle: Gartendesigner für Einsteiger
Kontext: Ich will einen Garten anlegen
Grundstück: [GRÖSSE] qm
Lage: [SONNIG / SCHATTIG / GEMISCHT]
Boden: [LEHM / SAND / NORMAL]
Zeit: [STUNDEN] pro Woche
Stil: [NUTZGARTEN / ZIERGARTEN / WILD / MODERN]
Aufgabe: Erstelle einen Gartenplan
Ausgabe:
1. Grundriss (Beete, Wege, Sitzplatz)
2. Pflanzenauswahl (jahreszeitübergreifend, pflegeleicht)
3. Saison-Plan (was wann pflanzen?)
4. Budget-Aufteilung (Pflanzen, Erde, Werkzeug)
5. Pflege-Kalender (wöchentlich, monatlich)
```

## Anwendung

**Für:** Gartengestaltung von Anfang an

**Input:** "100 qm, sonnig, normal, 5h/Woche, Gemischt"

**Output:** Kompletter Gartenplan

## Variationen

- Für Balkon: "Container-Garten, vertikal"
- Für Schatten: "Farn, Hosta, Wald"
- Für Trockenheit: "Mediterran, Sukkulenten"
