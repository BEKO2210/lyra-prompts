# Geschichte entwickeln

## Prompt

```
Rolle: Kreativschreib-Coach
Kontext: Ich will eine [KURZGESCHICHTE / BLOGPOST / SOCIAL MEDIA POST] schreiben
Thema: [ROHE IDEE]
Zielgruppe: [WER SOLL ES LESEN?]
Stimmung: [HUMORVOLL / NACHDENKLICH / SPANNEND / INSPIRIEREND]
Länge: [WORTZAHL oder "kurz/mittel/lang"]
Aufgabe: Entwickle die Story
Ausgabe:
1. Hook (Der erste Satz, der fesselt)
2. Struktur (Anfang, Mitte, Ende)
3. 3 konkrete Szenen/Sätze zum Weiterschreiben
4. Titel-Ideen (5 Varianten)
```

## Beispiel

**Input:** Blogpost über ersten Arbeitstag, junge Berufseinsteiger, humorvoll, 800 Wörter

**Output:** 
- Hook: "Ich stand in Unterwäsche vor meinem neuen Chef..."
- Struktur: Setup → Konflikt → Auflösung
- Szenen: Der Wecker, der Fahrstuhl, die Kantine

## Variationen

- Für Romane: "Plot und Charaktere entwickeln"
- Für Marketing: "Story für Marke finden"
- Für Präsentationen: "Narrative Struktur"
