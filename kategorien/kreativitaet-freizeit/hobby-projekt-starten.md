# Hobby-Projekt starten

## Prompt

```
Rolle: Hobby-Coach für Anfänger
Kontext: Ich will [HOBBY] ausprobieren
Erfahrung: [NOCH NIE / EIN WENIG / FRÜHER MAL]
Budget: [BETRAG] für Startausrüstung
Zeit: [STUNDEN] pro Woche
Ziel: [REINES VERGNÜGEN / ETWAS KONKRETES ERSTELLEN / SOZIALER ASPEKT]
Aufgabe: Begleite mich beim Einstieg
Ausgabe:
1. Minimal-Ausrüstung (was brauche ich wirklich?)
2. Erstes Projekt (klein, erreichbar, motivierend)
3. Lernressourcen (YouTube, Bücher, Kurse – kostenlos bis premium)
4. Gemeinschaft (Foren, Gruppen, lokale Treffen)
5. Nächster Schritt (konkret: Was mache ich heute?)
```

## Beispiel

**Input:** Gitarre, noch nie, 200€, 3h/Woche, Musik machen

**Output:** 
- Ausrüstung: Akustikgitarre, Plektren, Stimmgerät
- Erstes Projekt: 3 Akkorde, einfaches Lied
- Ressourcen: JustinGuitar (kostenlos)
- Community: Lokale Jam-Sessions

## Variationen

- Für Handwerk: "Holzbearbeitung, Nähen, etc."
- Für Sport: "Klettern, Yoga, etc."
- Für Kunst: "Malen, Fotografie, etc."
