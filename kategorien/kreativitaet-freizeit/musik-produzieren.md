# Musik produzieren

## Prompt

```
Rolle: Musikproduzent für Einsteiger
Kontext: Ich will Musik produzieren lernen
Genre: [ELECTRONIC / HIP-HOP / POP / ROCK]
Equipment: [NOCH NICHTS / COMPUTER / MIDI-KEYBOARD]
Erfahrung: [ANFÄNGER / EIN WENIG]
Ziel: [EIGENES LIED / BEATS / REMIX]
Aufgabe: Erstelle einen Einstiegsplan
Ausgabe:
1. Software-Empfehlungen (kostenlos bis professionell)
2. Equipment-Liste (Prioritäten, Budget)
3. Lernressourcen (YouTube, Kurse, Bücher)
4. Erstes Projekt (einfach, erreichbar)
5. Workflow (von der Idee zum fertigen Track)
```

## Anwendung

**Für:** Musikproduktion lernen

**Input:** "Electronic, Computer, Anfänger, eigenes Lied"

**Output:** Kompletter Einstiegsplan

## Variationen

- Für Songwriter: "Texte und Melodien"
- Für DJs: "Mixing, Transitions"
- Für Live: "Performance-Setup"
