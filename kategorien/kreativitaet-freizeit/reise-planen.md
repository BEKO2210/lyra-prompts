# Reise planen

## Prompt

```
Rolle: Reiseexperte für maßgeschneiderte Trips
Kontext: Ich will nach [ZIEL] fahren
Dauer: [TAGE]
Budget: [BETRAG] (ohne Anreise)
Reisestil: [BACKPACKER / KOMFORT / LUXUS / MIX]
Interessen: [z.B. Kultur, Natur, Essen, Party, Entspannung]
Einschränkungen: [MOBILITÄT, ESSEN, ZEIT]
Aufgabe: Plane den Trip
Ausgabe:
1. Tagesplan (Morgens, Mittags, Abends)
2. Unterkunftsempfehlungen (3 Preisklassen)
3. Insider-Tipps (nicht die typischen Touristenfallen)
4. Packliste (wetterangepasst)
5. Budget-Aufschlüsselung
```

## Beispiel

**Input:** Japan, 14 Tage, 2500€, Mix, Kultur + Essen, vegetarisch

**Output:** 
- Route: Tokyo → Kyoto → Osaka
- Unterkünfte: Ryokan, Capsule Hotel, Airbnb
- Insider: Lokale Izakayas, versteckte Tempel

## Variationen

- Für Städtetrips: "48 Stunden in..."
- Für Natur: "Wandern + Camping"
- Für Familien: "Kinderfreundlich"
