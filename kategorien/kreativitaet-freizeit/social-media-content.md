# Social Media Content

## Prompt

```
Rolle: Content-Stratege für authentische Präsenz
Kontext: Ich will auf [PLATTFORM: Instagram/LinkedIn/TikTok/etc.] posten
Thema: [WORUM GEHT ES?]
Meine Stimme: [PROFESSIONELL / PERSÖNLICH / HUMORVOLL / INSPIRIEREND]
Ziel: [MEHR FOLLOWER / POSITIONIERUNG / VERKAUF / AUSTAUSCH]
Aufgabe: Erstelle einen Post
Ausgabe:
1. Haupttext (mit Aufbau: Hook → Story → CTA)
2. 3 Hashtag-Optionen (unterschiedliche Reichweiten)
3. Bild-/Video-Idee
4. Beste Posting-Zeit
5. Antworten auf Kommentare (3 Vorlagen)
```

## Beispiel

**Input:** LinkedIn, Karrierewechsel, professionell aber persönlich, Positionierung

**Output:** 
- Hook: "Vor 3 Jahren hätte ich nie gedacht..."
- Story: Der Wendepunkt, die Entscheidung, das Ergebnis
- CTA: "Wer steht vor einer ähnlichen Entscheidung?"

## Variationen

- Für Instagram: "Visuell fokussiert"
- Für TikTok: "Kurz, unterhaltsam, trendig"
- Für Twitter/X: "Prägnant, kontrovers, thread-fähig"
