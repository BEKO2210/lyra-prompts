# Wochenend-Aktivität

## Prompt

```
Rolle: Lokaler Aktivitäten-Scout
Kontext: Ich habe ein [WOCHENENDE / FEIERTAG] frei
Ort: [DEIN STANDORT]
Begleitung: [ALLEIN / PARTNER / FREUNDE / FAMILIE]
Budget: [BETRAG]
Wetter: [AKTUELLE VORHERSAGE]
Stimmung: [AKTIV / ENTSPANNT / KULTURELL / SPONTAN]
Aufgabe: Finde die perfekte Aktivität
Ausgabe:
1. 3 Optionen (unterschiedliche Energielevel)
2. Zeitplan (inklusive Puffer)
3. Was mitnehmen
4. Plan B (bei schlechtem Wetter / ausgebucht)
```

## Beispiel

**Input:** Freiburg, Partner, 100€, sonnig, entspannt

**Output:** 
- Option 1: Schauinsland-Wanderung + Hüttenessen
- Option 2: Museumsbesuch + Café-Hopping
- Option 3: Thermalbad + entspanntes Dinner

## Variationen

- Für Regen: "Indoor-Aktivitäten"
- Für Spontane: "Last-Minute-Ideen"
- Für Budget: "Kostenlos bis 20€"
