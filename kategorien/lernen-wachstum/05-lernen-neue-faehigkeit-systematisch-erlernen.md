## Prompt

**Rolle:** Du bist ein Lernwissenschaftler und Skill-Acquisition-Experte, der evidence-basierte Methoden für effizientes Lernen anwendet.

**Kontext:** Der Nutzer möchte eine neue Fähigkeit erlernen (sprachlich, technisch, kreativ oder sportlich), hat aber begrenzte Zeit und möchte nicht ineffizient üben. Er sucht einen strukturierten Lernplan.

**Aufgabe:** Entwickle einen personalisierten Lernplan basierend auf:
- Deliberate Practice Prinzipien
- Spaced Repetition
- Progressive Overload
- Feedback-Loops

**Einschränkungen:**
- Realistisches Zeitbudget berücksichtigen
- Motivation durch kleine Erfolge aufrechterhalten
- Vermeidung von Überforderung und Burnout

**Ausgabe:** Ein strukturierter Lernplan mit konkreten Übungen, Zeitplan und Fortschrittsmetriken.

## Beispiel

**Input:**
"Ich möchte Python programmieren lernen. Kann täglich 45 Minuten investieren. Absolute Anfängerin, Ziel: eigenes kleines Datenanalyse-Tool bauen."

**Output:**

### Python Lernplan: 0 → Datenanalyse-Tool
**Zeithorizont:** 12 Wochen (ca. 50 Stunden effektives Lernen)

---

### Phase 1: Grundlagen (Woche 1-3)
**Ziel:** Syntax verstehen, erste Programme schreiben

**Lernmethode:**
- 30 Min aktives Coden (nicht nur lesen!)
- 15 Min Theorie/Video

**Wöchentliche Struktur:**
| Tag | Fokus | Übung |
|-----|-------|-------|
| Mo | Variablen & Datentypen | Mini-Rechner bauen |
| Di | Kontrollstrukturen | Zahlenratespiel |
| Mi | Funktionen | Wiederverwendbare Module |
| Do | Listen & Dictionaries | Kontaktverwaltung |
| Fr | Fehlerbehebung | Debug-Übungen |
| Sa | Projekt | Kleines CLI-Tool |
| So | Review | Wiederholung schwacher Bereiche |

**Fortschrittsmetrik:** 5 kleine Programme pro Woche erfolgreich ausführen.

---

### Phase 2: Anwendung (Woche 4-7)
**Ziel:** Libraries nutzen, Daten verarbeiten

**Fokus:**
- Pandas für Datenmanipulation
- Matplotlib für Visualisierung
- Erstes echtes Datenprojekt

**Deliberate Practice:**
- Jeden Tag eine neue Funktion aktiv anwenden
- Code auf GitHub dokumentieren
- Peer-Review in Community (z.B. Reddit r/learnpython)

---

### Phase 3: Projekt (Woche 8-12)
**Ziel:** Eigenes Datenanalyse-Tool

**Projekt-Idee:** Automatischer CSV-Report-Generator
- Daten einlesen
- Statistiken berechnen
- Diagramme erstellen
- PDF-Export

**Wochenziele:**
- Woche 8: Projektplanung & Setup
- Woche 9: Kernfunktionalität
- Woche 10: UI/UX
- Woche 11: Testing & Bugfixing
- Woche 12: Dokumentation & Showcase

---

### Motivations-System
- **Woche 3:** Belohnung nach Phase 1
- **Woche 7:** Portfolio-Eintrag
- **Woche 12:** Projekt vor Freunden/Familie präsentieren

### Ressourcen
- Automate the Boring Stuff (Buch)
- Corey Schafer YouTube
- LeetCode (später für Algorithmen)

## Variationen

**Variation 1 - Sprachenlernen:**
Fokus auf Input-Hypothese (Krashen), SRS mit Anki, Sprachtandem, Immersion-Techniken.

**Variation 2 - Musikalisches Instrument:**
Technik-Übungen, Ohrtraining, Repertoire-Aufbau, regelmäßige Mini-Auftritte als Ziele.

**Variation 3 - Öffentliches Sprechen:**
Toastmasters-Struktur, Video-Analyse, progressive Schwierigkeit (Spiegel → Familie → Gruppe → Stage).

**Variation 4 - Intensiv-Bootcamp (20h/Woche):**
Komprimierter Plan, Fokus auf Projekt-basiertes Lernen, weniger Theorie, mehr Bauen.
