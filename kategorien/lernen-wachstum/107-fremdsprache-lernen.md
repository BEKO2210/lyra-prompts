# Fremdsprache lernen - Effektive Methode

## Prompt

```
Rolle: Polyglott und Sprachlern-Coach (fließend in 5+ Sprachen)
Kontext: Ich möchte eine neue Sprache lernen
Sprache: [WELCHE?]
Aktueller Level: [ANFÄNGER / A1 / A2 / B1 / ETC.]
Ziel-Level: [KONVERSATION / FLIEßEND / GESCHÄFTLICH]
Zeitrahmen: [WIE VIELE MONATE?]
Tägliches Budget: [MINUTEN PRO TAG]
Hauptziel: [REISEN / BERUF / INTERESSE]
Vorherige Spracherfahrung: [ERSTE FREMDSPRACHE / MEHRFACH]
Aufgabe: Erstelle einen optimierten Lernplan

Ausgabe:
1. Methoden-Auswahl:
   - Input-Hypothese (Comprehensible Input)
   - SRS (Spaced Repetition)
   - Immersion-Techniken
   - Sprachaustausch
2. Material-Stack:
   - App (Anki/Duolingo/Babbel - was passt?)
   - Hörbuch/Podcast
   - YouTube-Kanäle
   - Buch/Lektüre
   - Tutor (wann sinnvoll?)
3. Täglicher Lernplan:
   - Morgens: [Aktivität, Minuten]
   - Unterwegs: [Aktivität, Minuten]
   - Abends: [Aktivität, Minuten]
4. Wochenplan:
   - Input (Hören/Lesen)
   - Output (Sprechen/Schreiben)
   - Wiederholung
   - Tutor-Session (optional)
5. Meilensteine:
   - Woche 4: Erste Sätze
   - Woche 12: Einfache Konversation
   - Woche 24: [ZIEL]
6. Motivation halten:
   - Gewohnheits-Tracking
   - Sprachpartner finden
   - Mini-Ziele setzen
   - Rückfälle vermeiden
7. Typische Fehler vermeiden:
   - Was funktioniert NICHT?
   - Zeitfallen
   - Methoden-Wechsel wann?
```

## Beispiel

**Input:** "Spanisch, Anfänger, Konversation, 6 Monate, 45 Min/Tag, Reisen, schon Englisch gelernt"

**Output:**
- Apps: Anki (Vokabeln) + Language Transfer (Konzepte)
- Input: Dreaming Spanish (YouTube)
- Täglich: 20 Min Anki, 20 Min Video, 5 Min laut sprechen
- Meilenstein: Nach 3 Monaten erste iTalki-Session

## Variationen

- Für beruflich: "Business-Englisch in 90 Tagen"
- Für Prüfung: "Goethe/Telc Prüfung vorbereiten"
- Für mehrere Sprachen: "Zwei Sprachen parallel lernen"
- Für Aufrechterhaltung: "Sprache nach Jahren wieder aktivieren"
