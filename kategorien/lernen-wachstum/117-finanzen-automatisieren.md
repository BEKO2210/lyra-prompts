# Persönliche Finanzen automatisieren

## Prompt

```
Rolle: Finanz-Coach für Automation und passives Vermögensmanagement
Kontext: Ich möchte meine Finanzen automatisieren
Einkommen: [NETTO PRO MONAT]
Fixkosten: [MIETE, VERSICHERUNGEN, ABOS]
Ziele: [NOTGROSCHEM / INVESTIEREN / SPARZIEL / SCHULDEN ABBAUEN]
Bank: [BANKNAME / MEHRERE BANKEN]
Technik-Affinität: [ANFÄNGER / FORTGESCHRITTEN]
Aufgabe: Erstelle ein automatisiertes Finanzsystem

Ausgabe:
1. Konten-Struktur:
   - Tagesgeld-Konto (Eingang)
   - Fixkosten-Konto
   - Spar/Invest-Konto
   - Spaß-Geld-Konto
   - Notgroschen-Konto
2. Automatisierungs-Workflow:
   - Gehaltseingang
   - Automatische Überweisungen (Wann? Wie viel?)
   - Daueraufträge einrichten
   - Apps/Tools für Überblick
3. 50/30/20 Regel anpassen:
   - 50% Bedürfnisse (anpassen an reale Fixkosten)
   - 30% Wünsche (Spaß-Geld)
   - 20% Sparen/Investieren
   - Oder alternative Aufteilung
4. Investitions-Automation:
   - Sparpläne (ETFs, Aktien)
   - Robo-Advisor vs. Selbst machen
   - Rebalancing-Strategie
   - Steuer-Optimierung
5. Überwachung:
   - Dashboard (Excel/App)
   - Monatlicher Check (10 Minuten)
   - Jährliches Review
   - Anpassung bei Gehaltsänderung
6. Notfall-Plan:
   - Notgroschen-Höhe berechnen
   - Wo parken?
   - Wann anbrechen?
7. Optimierung:
   - Abos prüfen (jährlich)
   - Versicherungen optimieren
   - Konditionen verhandeln
```

## Beispiel

**Input:** "3.500€ netto, 1.500€ Fixkosten, Ziele: Notgroschen + Investieren, Sparkasse + ING, fortgeschritten"

**Output:**
- Konten: Sparkasse (Eingang), ING (Spar/Invest)
- Automation: 1.500€ Fix, 800€ Spaß, 700€ Invest, 500€ Notgroschen
- ETFs: 2 Sparpläne (World + EM)
- Review: Monatlich 15 Min

## Variationen

- Für Selbstständige: "Unregelmäßiges Einkommen automatisieren"
- Für Paare: "Finanzen als Paar organisieren"
- Für Schuldner: "Schuldenabbau automatisieren"
- Für High-Earner: "Hohes Einkommen optimieren"
