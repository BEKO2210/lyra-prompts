# Immobilien als Kapitalanlage

## Prompt

```
Rolle: Immobilien-Investment-Beraterin
Kontext: Ich überlege, in Immobilien als Kapitalanlage zu investieren
Budget: [EIGENKAPITAL VERFÜGBAR]
Finanzierung: [BAR / KREDITFÄHIG / UNKLAR]
Ziel: [RENDITE / WERTSTEIGERUNG / RUHESTAND / STEUERN SPAREN]
Region: [WO SOLL GEKAUFT WERDEN?]
Erfahrung: [KEINE / EIGENE WOHNUNG / SCHON VERMIETET]
Zeitaufwand: [WENIG / MITTEL / VIEL]
Aufgabe: Bewerte meine Situation und erstelle einen Plan

Ausgabe:
1. Eignungs-Check:
   - Ist Immobilien-Investment für mich geeignet?
   - Risikoprofil
   - Liquidität
   - Alternative Investments vergleichen
2. Finanzierung:
   - Eigenkapital-Quote
   - Kreditfähigkeit prüfen
   - Zinsbindung
   - Tilgung
   - Nebenkosten (Grunderwerbsteuer, Notar, Makler)
3. Objektsuche:
   - Kriterien (Lage, Zustand, Mieter)
   - Rendite-Berechnung
   - Due Diligence
   - Makler vs. selbst suchen
4. Steuern:
   - AfA (Abschreibung)
   - Werbungskosten
   - Überschussrechnung
   - Steuerberater ja/nein?
5. Vermietung:
   - Mieter finden
   - Mietvertrag
   - Nebenkostenabrechnung
   - Instandhaltung
6. Verwaltung:
   - Selbstverwaltung vs. Hausverwaltung
   - Kosten der Verwaltung
   - Wann lohnt es sich?
7. Exit-Strategien:
   - Langfristig halten
   - Verkauf
   - Eigennutzung
   - Vererben
```

## Beispiel

**Input:** "100.000€ Eigenkapital, kreditfähig, Rendite + Wertsteigerung, Berlin/Brandenburg, keine Erfahrung, wenig Zeit"

**Output:**
- Finanzierung: 70% Fremdfinanzierung möglich
- Objekt: 2-3 Zimmer-Wohnung, vermietet
- Rendite: 3-4% nach Kosten
- Verwaltung: Unbedingt Hausverwaltung

## Variationen

- Für Anfänger: "Erste Immobilie als Kapitalanlage"
- Für Mehrfamilienhäuser: "MFH als Investment"
- Für Ausland: "Immobilien im Ausland kaufen"
- Für Commercial: "Gewerbeimmobilien investieren"
