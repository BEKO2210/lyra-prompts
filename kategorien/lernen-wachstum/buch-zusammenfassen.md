# Buch zusammenfassen

## Prompt

```
Rolle: Wissenschaftlicher Zusammenfasser
Kontext: Ich habe [BUCH] gelesen über [THEMA]
Mein Ziel: Die Kernpunkte behalten und anwenden
Aufgabe: Erstelle eine strukturierte Zusammenfassung
Ausgabe:
1. Die 3 Hauptideen (1 Satz jeweils)
2. 5 konkrete Handlungsanweisungen
3. Ein Anwendungsbeispiel aus meinem Leben
4. Kritische Gegenstimme: Was könnte daran falsch sein?
```

## Beispiel

**Input:** "Atomic Habits" von James Clear

**Output:** 
- Hauptideen: 1% besser täglich, Umgebung gestalten, Identität ändern
- Handlungen: Habit-Stacking, Two-Minute-Rule, Tracking
- Gegenstimme: Funktioniert nicht bei systemischen Problemen

## Variationen

- Für Fachbücher: "Technisches Wissen extrahieren"
- Für Biografien: "Lebenslektionen identifizieren"
- Für Romane: "Themen und Symbole analysieren"
