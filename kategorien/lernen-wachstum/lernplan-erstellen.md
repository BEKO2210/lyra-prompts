# Lernplan erstellen

## Prompt

```
Rolle: Lernmethodik-Experte
Kontext: Ich will [FÄHIGKEIT] lernen
Ausgangspunkt: [ANFÄNGER / FORTGESCHRITTEN / EXPERTE]
Verfügbare Zeit: [STUNDEN] pro Woche
Zeitrahmen: [z.B. 3 Monate bis Ziel]
Lernstil: [VISUELL / AUDITIV / PRAKTISCH / THEORETISCH]
Aufgabe: Erstelle einen Lernplan
Ausgabe:
1. Zerlegung in 5-7 Module
2. Ressourcen pro Modul (kostenlos + premium)
3. Wochenplan mit konkreten Aktivitäten
4. Fortschritts-Checks (Meilensteine)
5. Motivations-Strategien für Tiefs
```

## Beispiel

**Input:** Python Programmierung, Anfänger, 5h/Woche, 3 Monate, praktisch

**Output:** 
- Module: Grundlagen, Datenstrukturen, OOP, Projekte, etc.
- Ressourcen: freeCodeCamp, Automate the Boring Stuff, LeetCode
- Wochenplan: 2h Theorie, 3h Praxis

## Variationen

- Für Sprachen: "Immersion + Grammatik"
- Für Musik: "Instrument lernen"
- Für Zertifikate: "SAP/Scrum/AWS-Zertifizierung"
