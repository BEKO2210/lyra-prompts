# Mentoring finden

## Prompt

```
Rolle: Karriereberater für Mentoring-Beziehungen
Kontext: Ich suche einen Mentor für [BEREICH]
Mein Stand: [AKTUELLE POSITION / ERFAHRUNG]
Mein Ziel: [WAS WILL ICH ERREICHEN?]
Verfügbare Zeit: [STUNDEN] pro Monat
Aufgabe: Hilf mir, einen Mentor zu finden
Ausgabe:
1. Idealprofil Mentor (Wen suche ich?)
2. Suchstrategien (LinkedIn, Netzwerk, Programme)
3. Erste Kontaktaufnahme (Anschreiben-Template)
4. Mentoring-Gespräch (Agenda, Fragen)
5. Beziehung pflegen (Regelmäßigkeit, Gegenleistung)
```

## Anwendung

**Für:** Mentoring-Beziehung aufbauen

**Input:** "SAP Consulting, Junior Consultant, Senior werden, 4h/Monat"

**Output:** Strategie zum Finden und Halten eines Mentors

## Variationen

- Für Peer-Mentoring: "Gleichrangige Unterstützung"
- Für Reverse-Mentoring: "Von Jüngeren lernen"
- Für Gruppen: "Mastermind-Gruppe gründen"
