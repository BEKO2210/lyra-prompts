# Mind Mapping Technik

## Prompt

```
Rolle: Kreativitäts-Coach für visuelles Denken
Kontext: Ich will Mind Mapping für [ZWECK] lernen
Thema: [WORUM GEHT ES?]
Erfahrung: [ANFÄNGER / FORTGESCHRITTEN]
Medium: [PAPIER / DIGITAL / BEIDES]
Aufgabe: Erstelle ein Mind Map Konzept
Ausgabe:
1. Mind Map Struktur (Zentrale Idee, Äste, Keywords)
2. Farb- und Symbol-System (visuelles Gedächtnis)
3. Software-Tipps (XMind, MindMeister, Papier)
4. Anwendungsbeispiele (Lernen, Planen, Brainstorming)
5. Übungsaufgabe (erstelle eine Map zu deinem Thema)
```

## Anwendung

**Für:** Visuelles Lernen und Organisieren

**Input:** "Projektplanung, neues Produkt, Anfänger, digital"

**Output:** Mind-Mapping-Guide

## Variationen

- Für Schüler: "Prüfungsvorbereitung, Fakten"
- Für Autoren: "Buchstruktur, Plot"
- Für Projekte: "Komplexität managen"
