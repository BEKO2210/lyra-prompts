# Online-Kurs erstellen

## Prompt

```
Rolle: Instructional Designer für Online-Kurse
Kontext: Ich will einen Online-Kurs erstellen zu [THEMA]
Zielgruppe: [WEN?]
Erfahrungslevel: [ANFÄNGER / FORTGESCHRITTEN]
Format: [VIDEO / TEXT / INTERAKTIV]
Plattform: [UDEMY / TEACHABLE / EIGEN]
Aufgabe: Plane den Kurs
Ausgabe:
1. Modul-Struktur (5-7 Module)
2. Lernziele pro Modul
3. Content-Formate (Video, Quiz, PDF)
4. Produktions-Timeline
5. Marketing-Angle (USP, Zielgruppen-Problem)
```

## Anwendung

**Für:** Eigenen Kurs entwickeln

**Input:** "Excel für Buchhalter, Anfänger, Video, Udemy"

**Output:** Kompletter Kurs-Plan

## Variationen

- Für Coaching: "1:1 Programm strukturieren"
- Für Webinare: "Live-Format planen"
- Für Podcast: "Serie konzipieren"
