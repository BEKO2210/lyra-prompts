# Podcast-Empfehlungen

## Prompt

```
Rolle: Podcast-Kurator für lebenslanges Lernen
Kontext: Ich suche Podcasts zu [THEMA / INTERESSE]
Aktuelles Wissenslevel: [ANFÄNGER / FORTGESCHRITTEN]
Format-Präferenz: [INTERVIEWS / STORYTELLING / LEHRREICH]
Länge: [KURZ <30min / MITTEL / LANG]
Sprache: [DEUTSCH / ENGLISCH / BEIDES]
Aufgabe: Empfiehl Podcasts
Ausgabe:
1. Top 5 Podcasts (mit Begründung)
2. Einstiegsfolge pro Podcast
3. Hörgewohnheiten (wann hören? wie integrieren?)
4. Ähnliche Formate (YouTube, Bücher, Newsletter)
```

## Anwendung

**Für:** Neue Podcasts entdecken

**Input:** "Psychologie, Anfänger, Interviews, mittel, Deutsch"

**Output:** Kuratierte Liste mit Einstiegsfolgen

## Variationen

- Für Pendler: "30-Minuten-Formate"
- Für Sport: "Motivierende Podcasts"
- Für Einschlafen: "Ruhig, nicht zu spannend"
