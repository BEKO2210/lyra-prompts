# Pro-Contra-Analyse

## Prompt

```
Rolle: Unabhängiger Berater für schwierige Entscheidungen
Kontext: Ich muss entscheiden: [OPTION A] vs [OPTION B]
Meine Prioritäten (rangiert):
1. [WICHTIGSTES KRITERIUM]
2. [ZWEITWICHTIGSTES]
3. [DRITTWICHTIGSTES]
Meine Ängste: [WAS HALTET MICH ZURÜCK?]
Aufgabe: Hilf mir strukturiert entscheiden
Ausgabe:
1. Pro/Contra-Liste (gewichtet nach meinen Prioritäten)
2. "Was wäre, wenn..."-Szenarien (3 Jahre)
3. Der eine Dealbreaker für jede Option
4. Empfehlung mit Begründung
5. Erster konkreter Schritt
```

## Beispiel

**Input:** Jobwechsel vs. Bleiben, Priorität: Wachstum, Gehalt, Work-Life-Balance

**Output:** 
- Gewichtete Matrix mit Punkten
- Szenarien: Wo bin ich in 3 Jahren?
- Dealbreaker: Keine Weiterentwicklung vs. Unsicherheit

## Variationen

- Für Beziehungen: "Beziehung beenden oder kämpfen?"
- Für Umzug: "Stadt X oder Stadt Y?"
- Für Investitionen: "Kaufen oder warten?"
