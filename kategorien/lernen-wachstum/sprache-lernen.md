# Sprache lernen

## Prompt

```
Rolle: Sprachcoach für effektives Lernen
Kontext: Ich will [SPRACHE] lernen
Level: [ANFÄNGER / A2 / B1 / FORTGESCHRITTEN]
Ziel: [URLAUB / BERUF / FLIESSEND]
Verfügbare Zeit: [MINUTEN] pro Tag
Bis wann: [ZEITRAUM]
Aufgabe: Erstelle einen Sprachlernplan
Ausgabe:
1. Lernmaterialien (Apps, Bücher, Podcasts)
2. Tägliche Routine (Hören, Sprechen, Lesen, Schreiben)
3. Immersions-Strategien (Filme, Musik, Tandem)
4. Fortschritts-Checks (Meilensteine)
5. Motivationstipps für Frustrationsphasen
```

## Anwendung

**Für:** Strukturiertes Sprachenlernen

**Input:** "Spanisch, Anfänger, Urlaub, 30 Min, 6 Monate"

**Output:** Kompletter Lernplan mit Ressourcen

## Variationen

- Für Beruf: "Business-Vokabular priorisieren"
- Für Prüfung: "Test-Strategie, Zeitmanagement"
- Für Kinder: "Spielerisch, intuitiv"
