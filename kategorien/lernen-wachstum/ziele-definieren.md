# Ziele definieren

## Prompt

```
Rolle: Zielcoach für nachhaltige Veränderung
Kontext: Ich will [ALLGEMEINES ZIEL] erreichen
Aktueller Stand: [WO BIN ICH JETZT?]
Motivation: [WARUM IST MIR DAS WICHTIG?]
Hindernisse: [WAS HAT BISHER NICHT GEKLAPPT?]
Aufgabe: Formuliere SMART-Ziele
Ausgabe:
1. Das große Ziel (Vision)
2. 3 Zwischenziele (3 Monate, 6 Monate, 12 Monate)
3. Die nächste Mini-Aufgabe (heute/diese Woche)
4. Accountability-System
5. Erfolgsmetriken (Woran merke ich, dass ich auf Kurs bin?)
```

## Beispiel

**Input:** Fit werden, Couch Potato, Gesundheit für Familie, keine Zeit

**Output:** 
- Vision: 3x pro Woche Sport, gesunde Ernährung
- Zwischenziele: 1x/Woche → 2x/Woche → 3x/Woche
- Heute: Sportschuhe kaufen, 10 Minuten Spaziergang

## Variationen

- Für Karriere: "Beförderung in 2 Jahren"
- Für Finanzen: "Schuldenfrei in 18 Monaten"
- Für Beziehungen: "Mehr Zeit mit Partner"
