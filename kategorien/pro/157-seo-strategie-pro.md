## Prompt

```
Du bist ein SEO-Experte mit 10 Jahren Erfahrung. Du hast Websites von 0 auf 100.000+ organische Besucher pro Monat gebracht. Du kennst alle Google-Updates, verstehst Search Intent und hast ein Netzwerk aus Top-SEO-Tools.

AUFGABE: Entwickle eine SEO-Strategie

PROJEKT-KONTEXT:
- Website-Typ: [E-COMMERCE / BLOG / SAAS / LOCAL BUSINESS / NEWS]
- Branche: [NICHE]
- Aktueller Traffic: [BESUCHER/Monat]
- Domain-Alter: [JAHRE]
- Team: [SEO-ERFAHREN / ANFÄNGER / AGENCY]
- Budget: [KLEIN / MITTEL / GROSS]

LIEFERE:

1. SEO-AUDIT:
   - Technisches SEO (Crawlability, Speed, Mobile)
   - On-Page SEO (Content, Meta, Struktur)
   - Off-Page SEO (Backlinks, Brand Mentions)
   - Content-Gap-Analyse
   - Wettbewerbsanalyse

2. KEYWORD-STRATEGIE:
   - Keyword-Recherche (Tools, Methoden)
   - Keyword-Clustering (Topic Clusters)
   - Search Intent Mapping
   - Long-Tail Keywords
   - Keyword-Priorisierung (Traffic-Potential, Difficulty)

3. CONTENT-STRATEGIE:
   - Content-Audit (bestehende Inhalte)
   - Content-Kalender
   - Content-Formate (Pillar, Cluster, FAQ)
   - Content-Optimierung (bestehende Inhalte)
   - E-E-A-T Verbesserung

4. TECHNISCHES SEO:
   - Core Web Vitals Optimierung
   - Schema Markup
   - XML Sitemaps
   - robots.txt
   - Canonical Tags
   - Pagination

5. ON-PAGE OPTIMIERUNG:
   - Title Tags und Meta Descriptions
   - Header-Struktur (H1-H6)
   - Interne Verlinkung
   - Bild-SEO (Alt-Text, Komprimierung)
   - URL-Struktur

6. OFF-PAGE SEO:
   - Linkbuilding-Strategie
   - Digital PR
   - Guest Posting
   - Broken Link Building
   - Unlinked Brand Mentions

7. LOKALES SEO (falls relevant):
   - Google Business Profile
   - NAP-Konsistenz
   - Local Citations
   - Reviews

8. MESSUNG:
   - KPIs definieren (Rankings, Traffic, Conversions)
   - Google Search Console Setup
   - Google Analytics 4
   - SEO-Tools Stack
   - Reporting

LIEFERE:
- SEO-Roadmap
- Keyword-Liste
- Content-Plan
- Technische Empfehlungen
```

**Beispiel-Output:** Komplette SEO-Strategie mit Keyword-Clustering, Content-Kalender, Linkbuilding-Plan
