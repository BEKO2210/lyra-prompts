# Finanzmodell erstellen

## Prompt

```
Rolle: Investment Banking Analyst + FP&A Director (CFA, ex-Goldman Sachs)
Kontext: Ich brauche ein Finanzmodell für [ZWECK]:
Unternehmensdaten:
- Branche: [SEKTOR]
- Phase: [STARTUP / WACHSTUM / REIF]
- Rechtsform: [GMBH / AG / UG / etc.]
Historische Finanzen (letzte 3 Jahre):
- Umsatz: [JAHR 1, 2, 3]
- EBITDA: [JAHR 1, 2, 3]
- Cashflow: [JAHR 1, 2, 3]
- Bilanz-Kennzahlen: [EIGENKAPITAL, FK, UMLAUFVERMÖGEN]
Marktdaten:
- Marktwachstum: [% p.a.]
- Marktanteil aktuell: [%]
- Adressierbare Marktgröße (TAM/SAM/SOM)
Planungshorizont: [3/5/10 JAHRE]
Szenarien: [BEST/BASE/WORST CASE nötig?]
Spezielle Anforderungen:
- [FUNDING RUNDE / BUDGETPLANUNG / MA / BÖRSENGANG]
- [WÄHRUNGSRISIKEN / ZINSÄNDERUNGEN / INFLATION]
Aufgabe: Erstelle ein professionelles Finanzmodell
Einschränkungen:
- Audit-fähige Struktur (alle Formeln nachvollziehbar)
- Sensitivitätsanalyse eingebaut
- Konservative Annahmen dokumentiert
- Deutsche GAAP / IFRS konform
Ausgabe:
1. Modell-Struktur (Sheet-Übersicht)
2. Annahmen-Sheet (alle Inputs zentral)
3. P&L Forecast (Jahre 1-5/10)
4. Cashflow Forecast (direkte Methode)
5. Bilanz-Projektion
6. KPI-Dashboard:
   - Wachstumsraten
   - Margen (Brutto, EBITDA, Netto)
   - Cash Conversion Cycle
   - ROI, ROE, ROIC
   - Break-Even-Analyse
7. Sensitivitätsanalyse (Tornado-Diagramm-Beschreibung)
8. Szenario-Vergleich (Best/Base/Worst)
9. Valuation (falls relevant: DCF, Multiples, VC-Method)
10. Excel-Formeln für komplexe Berechnungen
```

## Anwendung

**Für:** CFOs, Investoren, Gründer, M&A-Teams

**Input:** "Series A Fundraising, SaaS, €1M ARR, 120% NRR, 20% EBITDA Margin, 5-Jahres-Plan"

**Output:** Investor-reifes Finanzmodell

## Variationen

- Für Startups: "Seed-Stage Financial Model"
- Für M&A: "Due Diligence Financial Analysis"
- Für Budget: "Annual Budget & Variance Analysis"
- Für LBO: "Leveraged Buyout Model"
