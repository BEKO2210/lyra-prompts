# Verhandlungsstrategie komplex

## Prompt

```
Rolle: Chief Negotiation Officer (ehemaliger Verhandlungsführer für Fortune 500 Deals)
Kontext: Ich führe eine [VERHANDLUNGSART] mit folgenden Parametern:
Gegenüber: [FIRMA/PERSON + deren Hintergrund, Stil, bekannte Taktiken]
Verhandlungsgegenstand: [WAS WIRD VERHANDELT?]
Meine Position:
- Must-Haves: [NICHT VERHANDELBAR]
- Nice-to-Haves: [WÜNSCHENSWERT]
- Walk-Away-Punkt: [ABSOLUTE GRENZE]
- BATNA (Best Alternative): [PLAN B]
Deren vermutete Position: [WAS KÖNNTE IHNEN WICHTIG SEIN?]
Zeitdruck: [WER HAT DRUCK?]
Beziehungsebene: [LANGFRISTIG vs. ONE-OFF, Vertrauenslevel]
Vorgeschichte: [FRÜHERE VERHANDLUNGEN, Konflikte, Erfolge]
Aufgabe: Entwickle eine umfassende Verhandlungsstrategie
Einschränkungen:
- Berücksichtige kulturelle Unterschiede (falls international)
- Plane für 3 Verhandlungsrunden
- Antizipiere emotionale Manipulationstaktiken
- Halte Reputation langfristig im Blick
Ausgabe:
1. Verhandlungsziele (primär, sekundär, verdeckt)
2. Gegenüber-Profil (Motivation, Druckpunkte, Schwächen)
3. Taktischer Plan:
   - Eröffnung (Anchor, Framing)
   - Mittelphase (Concession-Management, Logrolling)
   - Endphase (Closing-Techniken, Deadline-Taktik)
4. Szenario-Planung:
   - Best Case (ihre Zugeständnisse)
   - Realistic Case (Kompromiss)
   - Worst Case (Breakdown-Strategie)
5. Konkrete Formulierungen für kritische Momente
6. Body Language & psychologische Triggers
7. Dokumentations-Checkliste (was muss schriftlich festgehalten werden)
8. Post-Verhandlung: Implementierungsplan
```

## Anwendung

**Für:** Enterprise Sales, M&A, Lizenzen, Großprojekte

**Input:** "Enterprise-Software-Lizenz, SAP als Gegenüber, €500k Budget, Must-Have: API-Zugriff, Walk-Away: €750k"

**Output:** Komplette Verhandlungsdokumentation

## Variationen

- Für Gehaltsverhandlungen: "C-Level Package negotieren"
- Für Investoren: "Term Sheet Optimierung"
- Für Lieferanten: "Strategisches Sourcing"
- Für Krisensituationen: "Notfall-Verhandlung"
