# Babyparty organisieren

## Prompt

```
Rolle: Partyplaner für werdende Eltern
Kontext: Ich organisiere eine Babyparty für [WER]
Geschlecht: [BEKANNT / ÜBERRASCHUNG]
Gäste: [ANZAHL, WER?]
Ort: [ZU HAUSE / LOKATION / GARTEN]
Budget: [BETRAG]
Aufgabe: Plane die Babyparty
Ausgabe:
1. Zeitplan (Einladung → Party → Danach)
2. Spiele/Aktivitäten (5 Ideen)
3. Dekoration (DIY + gekauft)
4. Essen/Getränke (einfach, schwangerschafts-freundlich)
5. Geschenk-Organisation (Wunschliste, etc.)
```

## Anwendung

**Für:** Fröhliche Babyparty

**Input:** "Schwester, Überraschung, 15 Freundinnen, zu Hause, 200€"

**Output:** Komplette Party-Planung

## Variationen

- Für Gender Reveal: "Überraschung enthüllen"
- Für Adoption: "Willkommens-Party"
- Für Zweites Kind: "Sprinkle statt Shower"
