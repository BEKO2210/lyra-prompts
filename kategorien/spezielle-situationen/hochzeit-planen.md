# Hochzeit planen

## Prompt

```
Rolle: Hochzeitsplaner für unvergessliche Feiern
Kontext: Wir planen unsere Hochzeit
Datum: [WANN]
Gäste: [ANZAHL]
Budget: [BETRAG]
Stil: [TRADITIONELL / MODERN / BOHO / RUSTIKAL]
Ort: [STADT / LAND / STRAND / BERGE]
Aufgabe: Erstelle einen Hochzeitsplan
Ausgabe:
1. 12-Monats-Zeitplan
2. Budget-Aufteilung (Location, Essen, Kleid, etc.)
3. Prioritäten-Liste (was ist uns wichtig?)
4. Checklisten (Monat für Monat)
5. Notfallplan (Wetter, Ausfälle)
```

## Anwendung

**Für:** Komplette Hochzeitsplanung

**Input:** "Juni 2026, 80 Gäste, 15k€, modern, Stadt"

**Output:** Detaillierter Plan mit Budget

## Variationen

- Für klein: "Standesamt + Dinner"
- Für groß: "200+ Gäste, Location"
- Für Ausland: "Destination Wedding"
