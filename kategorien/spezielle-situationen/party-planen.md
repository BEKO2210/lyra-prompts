# Geburtstag/Party planen

## Prompt

```
Rolle: Eventplaner für private Feiern
Kontext: Ich plane [GEBURTSTAG / PARTY / FEIER] für [WER]
Gäste: [ANZAHL, ALTER, VERHÄLTNIS]
Ort: [ZU HAUSE / LOKATION / OUTDOOR]
Budget: [BETRAG]
Stil: [ENTSANNT / AUSGELASSEN / ELEGANT / THEMENPARTY]
Aufgabe: Plane die Feier
Ausgabe:
1. Zeitplan (Vorher, Während, Nachher)
2. Einkaufs-/Organisationsliste
3. Musik/Entertainment
4. Notfallplan (Wetter, Ausfälle, zu viel/zu wenig Gäste)
5. Nachbereitung (Danken, Aufräumen, Fotos)
```

## Beispiel

**Input:** 30. Geburtstag, 20 Personen, Zuhause, 300€, ausgelassen

**Output:** 
- Zeitplan: 2 Wochen vorher Einkauf, 1 Tag vorher Vorbereitung
- Liste: Getränke, Snacks, Deko
- Musik: Spotify-Playlist, Lautsprecher

## Variationen

- Für Kinder: "Kindergeburtstag"
- Für Überraschung: "Überraschungsparty organisieren"
- Für kleines Budget: "Kostenlos feiern"
