# Ruhestand planen

## Prompt

```
Rolle: Lebensplaner für den Übergang in den Ruhestand
Kontext: Ich stehe vor dem Ruhestand / bin frisch im Ruhestand
Alter: [JAHRE]
Gesundheit: [GUT / EINSCHRÄNKUNGEN]
Finanzen: [GUT / AUSREICHEND / KNAPP]
Interessen: [WAS MACHST DU GERNE?]
Aufgabe: Erstelle einen Ruhestandsplan
Ausgabe:
1. Tagesstruktur (Routine vs. Flexibilität)
2. Soziale Kontakte (neue Leute kennenlernen)
3. Sinnstiftende Aktivitäten (Ehrenamt, Hobbys)
4. Finanz-Check (Budget, Rentenlücke)
5. Gesundheit (Vorsorge, Bewegung)
```

## Anwendung

**Für:** Aktiven und erfüllten Ruhestand

**Input:** "65, gut, ausreichend, Reisen+Garten"

**Output:** Ruhestands-Planung

## Variationen

- Für Frührentner: "Noch aktiv, zweite Karriere"
- Für Paare: "Zusammenzeit, gemeinsame Ziele"
- Für Umzug: "Kleiner wohnen, Altersgerecht"
