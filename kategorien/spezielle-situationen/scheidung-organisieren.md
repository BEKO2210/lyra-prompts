# Scheidung organisieren

## Prompt

```
Rolle: Scheidungsberater für strukturierte Trennungen
Kontext: Ich stehe vor einer Scheidung/Trennung
Dauer Beziehung: [JAHRE]
Gemeinsame Kinder: [JA / NEIN]
Gemeinsame Vermögen: [WOHNUNG / HAUS / KONTO / AUTO]
Aktuelle Situation: [FRIEDLICH / SPANNEND]
Aufgabe: Erstelle einen Organisationsplan
Ausgabe:
1. Checkliste (Rechtsanwalt, Wohnung, Finanzen)
2. Zeitplan (was wann erledigen?)
3. Dokumenten-Liste (was brauche ich?)
4. Unterstützung (Beratung, Freunde, Therapie)
5. Zukunftsplanung (neuer Start, Ziele)
```

## Anwendung

**Für:** Strukturierte Scheidung

**Input:** "10 Jahre, ja, Wohnung+Auto, friedlich"

**Output:** Unterstützender Organisationsplan

## Variationen

- Für schwierige Trennungen: "Rechtsschutz, Dokumentation"
- Für Kinder: "Umgangsrecht, Kindeswohl"
- Für Internationale: "Recht, Staatsangehörigkeit"
