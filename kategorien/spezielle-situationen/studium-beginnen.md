# Studium beginnen

## Prompt

```
Rolle: Studienberater für Erstsemester
Kontext: Ich fange ein Studium an
Fach: [STUDIENGANG]
Uni/FH: [NAME / ORT]
Wohnort: [ZU HAUSE / WOHNHEIM / EIGENE WOHNUNG]
Finanzierung: [BAFÖG / ELTERN / JOB]
Aufgabe: Bereite mich aufs Studium vor
Ausgabe:
1. Vorab-Checkliste (Einschreibung, Bücher, Wohnung)
2. Erste-Woche-Plan (Orientierung, Kennenlernen)
3. Semesterplanung (Vorlesungen, Lernen, Prüfungen)
4. Finanzierung optimieren (Nebenjob, Stipendien)
5. Netzwerk aufbauen (Fachschaft, Hochschulgruppen)
```

## Anwendung

**Für:** Studienstart vorbereiten

**Input:** "BWL, Uni Hamburg, Wohnheim, BAföG"

**Output:** Komplette Startvorbereitung

## Variationen

- Für Zweitstudium: "Vorherige Erfahrung nutzen"
- Für Berufstätige: "Berufsbegleitend, Zeitmanagement"
- Für Internationale: "Visum, Sprache, Integration"
