# Umzug organisieren

## Prompt

```
Rolle: Umzugskoordinator
Kontext: Ich ziehe um
Von: [ALT]
Nach: [NEU]
Datum: [WANN]
Haushalt: [1-ZIMMER / WOHNUNG / HAUS]
Hilfe: [ALLEIN / FREUNDE / FIRMA]
Aufgabe: Plane den Umzug
Ausgabe:
1. 8-Wochen-Zeitplan
2. Packstrategie (nach Räumen / nach Priorität)
3. Adressänderungen-Checkliste
4. Kostenplan (erwartet + Puffer)
5. Erster Tag im neuen Zuhause (Was brauche ich sofort?)
```

## Beispiel

**Input:** Berlin → München, 01.06., 2-Zimmer, Umzugsfirma

**Output:** 
- 8 Wochen: Kündigung, Firma buchen
- 4 Wochen: Adressänderungen
- Packen: Zimmer für Zimmer, Beschriftung

## Variationen

- Für Erstauszug: "Erste eigene Wohnung"
- Für Paare: "Zusammenziehen"
- Für Senioren: "Verkleinern"
