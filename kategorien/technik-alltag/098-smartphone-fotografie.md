# Smartphone-Fotografie meistern

## Prompt

```
Rolle: Mobiler Fotografie-Experte
Kontext: Ich möchte bessere Fotos mit meinem Smartphone machen
Gerät: [IPHONE / SAMSUNG / PIXEL / ANDERES]
Haupt-Einsatzzweck: [REISEN / PORTRAITS / PRODUKTE / ALLTAG]
Kenntnisstand: [ANFÄNGER / FORTGESCHRITTEN]
Aufgabe: Gib mir eine komplette Smartphone-Fotografie-Ausbildung

Ausgabe:
1. Grundlagen (die 80% des Ergebnisses):
   - Licht verstehen (golden hour, Schatten)
   - Komposition (Drittel-Regel, Führungslinien)
   - Fokus und Belichtung manuell steuern
   - Stabilität (Haltung, Stativ-Alternativen)
2. Kamera-Features meines Handys:
   - Welche Modi gibt es?
   - Wann welchen nutzen?
   - Pro/Raw-Modus erklärt
3. Szenario-Guides:
   - Portraits (Menschen & Haustiere)
   - Landschaften
   - Essen/Produkte
   - Nachtaufnahmen
   - Action/Sport
4. Editing-Workflow:
   - Beste Apps (kostenlos + bezahlt)
   - Standard-Presets erstellen
   - Schneller Workflow
5. Organisation:
   - Fotos verwalten (Backup!)
   - Beste Bilder finden
   - Cloud-Strategie
6. Übungsplan:
   - 30-Tage-Challenge mit täglichen Aufgaben
```

## Beispiel

**Input:** "iPhone 15, Reise-Fotos, Anfänger"

**Output:**
- Features: Portrait-Modus für Menschen, Action-Modus für Bewegung
- Apps: Lightroom Mobile (kostenlos), Snapseed
- Challenge: Tag 1 = golden hour, Tag 2 = Symmetrie, etc.

## Variationen

- Für Instagram: "Instagram-taugliche Fotos erstellen"
- Für Verkauf: "Produktfotos mit dem Handy"
- Für Videos: "Smartphone-Videografie Basics"
- Für ältere Handys: "Gute Fotos mit älterem Smartphone"
