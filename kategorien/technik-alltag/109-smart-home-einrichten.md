# Smart Home einrichten

## Prompt

```
Rolle: Smart Home Integrator mit Zertifizierungen
Kontext: Ich möchte mein Zuhause smart machen
Wohnung/Haus: [GRÖSSE, MIETE/EIGEN]
Budget: [GESAMTBUDGET]
Technik-Affinität: [ANFÄNGER / FORTGESCHRITTEN]
Prioritäten: [KOMFORT / SICHERHEIT / ENERGIESPAREN / UNTERHALTUNG]
Bestehende Geräte: [ALEXA / GOOGLE / APPLE / GAR NICHTS]
Mieteinschränkungen: [KEINE BOHRER / KEINE ÄNDERUNGEN / EIGENTUM]
Aufgabe: Erstelle einen Smart Home Plan

Ausgabe:
1. Ökosystem-Wahl:
   - Vergleich: Alexa vs. Google vs. Apple HomeKit vs. openHAB
   - Was passt zu meinen Geräten?
   - Zukunftssicherheit
2. Phasen-Plan (nach Priorität):
   - Phase 1: Basis (Licht, Steckdosen)
   - Phase 2: Komfort (Heizung, Jalousien)
   - Phase 3: Sicherheit (Kamera, Tür)
   - Phase 4: Entertainment
3. Produkt-Empfehlungen:
   - Licht: Philips Hue vs. Alternativen
   - Heizung: Thermostate
   - Sicherheit: Kameras, Türschlösser
   - Sensoren: Bewegung, Temperatur, Luftfeuchtigkeit
4. Automationen:
   - "Guten Morgen" Routine
   - "Gute Nacht" Routine
   - Anwesenheits-Simulation
   - Energie-Sparen-Modus
5. Installation:
   - Was kann ich selbst?
   - Was braucht Elektriker?
   - Miet-wohnung-freundliche Lösungen
6. Sicherheit & Datenschutz:
   - Netzwerk absichern
   - Cloud vs. Local
   - Hersteller-Vertrauen
7. Budget-Aufschlüsselung:
   - Pro Phase
   - Spar-Tipps
   - Wo lohnt Premium?
```

## Beispiel

**Input:** "80qm Mietwohnung, 1000€ Budget, Anfänger, Komfort + Energiesparen, Alexa vorhanden, keine Bohrer"

**Output:**
- Ökosystem: Alexa (passt zu vorhandenem)
- Phase 1: Smarte Steckdosen, Lampen
- Phase 2: Smarte Heizthermostate
- Produkte: TP-Link Kasa, Philips Hue, Tado

## Variationen

- Für Eigenheim: "Komplett-Smart-Home für Neubau"
- Für Mieter: "Smart Home ohne Spuren zu hinterlassen"
- Für Sicherheit: "Smart Home als Alarmanlage"
- Für Energie: "Smart Home für maximale Energieeinsparung"
