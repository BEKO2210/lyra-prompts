# 3D-Druck für Anfänger

## Prompt

```
Rolle: 3D-Druck-Experte mit eigener Maker-Space-Erfahrung
Kontext: Ich möchte mit 3D-Druck beginnen
Budget: [FÜR DRUCKER]
Verwendungszweck: [HOBBY / PROTOTYPEN / FUNKTIONSTEILE / DEKO]
Technik-Level: [ANFÄNGER / FORTGESCHRITTEN]
Platz: [SCHREIBTISCH / WERKSTATT / KELLER]
Aufgabe: Erstelle einen Einstiegs-Guide

Ausgabe:
1. Drucker-Auswahl:
   - FDM vs. Resin (wann was?)
   - Empfohlene Einsteiger-Drucker
   - Bausatz vs. fertig
   - Wichtige Features
2. Setup:
   - Erster Aufbau
   - Leveling (manuell vs. automatisch)
   - Erster Testdruck
   - Kalibrierung
3. Software:
   - Slicer (Cura, PrusaSlicer)
   - Einstellungen erklärt
   - Support-Material
   - Infill-Strategien
4. Materialien:
   - PLA (Anfänger-freundlich)
   - PETG (stärker)
   - ABS (temperaturbeständig)
   - Spezial-Filamente
5. Modelle finden:
   - Thingiverse, Printables
   - Kaufen vs. gratis
   - Eigene Modelle (Tinkercad, Fusion 360)
6. Troubleshooting:
   - Warping
   - Stringing
   - Layer-Shifts
   - First-Layer-Probleme
7. Sicherheit:
   - Lüftung (besonders Resin!)
   - Brandgefahr
   - Kinder im Haushalt
   - Entsorgung
```

## Beispiel

**Input:** "500€ Budget, Hobby + Funktionsteile, Anfänger, Werkstatt"

**Output:**
- Drucker: Creality Ender 3 V2 oder Prusa Mini+
- Material: Start mit PLA
- Software: Cura (kostenlos)
- Erste Prints: Benchy (Test), Handy-Ständer, Werkzeughalter

## Variationen

- Für Resin: "Resin-Druck für Miniaturen"
- Für Funktionsteile: "Funktionale Teile designen und drucken"
- Für Design: "3D-Modellierung für Druck"
- Für Business: "3D-Druck als Business"
