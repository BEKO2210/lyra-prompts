## Prompt

```
Rolle: IT-Support-Spezialist für PC-Wartung und Performance-Optimierung
Kontext: Mein Computer ist langsam geworden und ich möchte ihn gründlich aufräumen
Mein System:
- Betriebssystem: [WINDOWS 10/11 / MACOS / LINUX]
- Alter des PCs: [JAHRE]
- Hauptproblem: [z.B. langsamer Start, Speicher voll, Programme hängen]
- Festplatte: [SSD / HDD, SPEICHERPLATZ frei/gesamt]
- Technische Kenntnisse: [ANFÄNGER / ETWAS / FORTGESCHRITTEN]
Aufgabe: Erstelle einen Aufräum- und Optimierungsplan für meinen Computer
Einschränkungen:
- Keine Datenverluste riskieren (Backup zuerst!)
- Reihenfolge: sicher → effektiv → fortgeschritten
- Kostenlose Tools bevorzugen
Ausgabe: Schritt-für-Schritt-Anleitung nach Priorität mit erwarteter Verbesserung
```

## Anwendung

**Für:** Alle deren Computer langsam geworden ist

**Input:** "Windows 11, 3 Jahre alt, 20GB von 256GB frei, Start dauert 3 Minuten, Anfänger"

**Output:** 10-Schritte-Plan von "Autostart ausmisten" bis "Festplatte bereinigen" mit konkreten Anleitungen

## Variationen

- **Notfall-Speedup:** Die 3 wichtigsten Maßnahmen für sofort mehr Tempo
- **Mac aufräumen:** Spezielle Tipps für macOS
- **Speicherplatz finden:** Große Dateien finden und sicher löschen
- **Browser-Reinigung:** Chrome/Firefox schneller machen
