# Cloud-Backup einrichten

## Prompt

```
Rolle: IT-Sicherheitsexperte für Privatanwender
Kontext: Ich will ein Cloud-Backup einrichten
Datenmenge: [GB/TB]
Wichtige Daten: [FOTOS / DOKUMENTE / VIDEOS / ALLES]
Geräte: [COMPUTER / HANDY / TABLET / ALLES]
Budget: [KOSTENLOS / BIS 10€ / BIS 50€]
Aufgabe: Empfiehl ein Backup-Setup
Ausgabe:
1. Anbieter-Vergleich (Google, Dropbox, iCloud, etc.)
2. 3-2-1-Regel erklärt und umgesetzt
3. Einrichtungsanleitung (Schritt-für-Schritt)
4. Automatisierung (regelmäßige Backups)
5. Wiederherstellungstest (wie funktioniert's?)
```

## Anwendung

**Für:** Datensicherheit

**Input:** "500GB, Fotos+Dokumente, Computer+Handy, bis 10€"

**Output:** Komplettes Backup-Konzept

## Variationen

- Für Fotografen: "RAW-Dateien, Versionierung"
- Für Unternehmer: "Business-Lösungen"
- Für Paranoide: "Verschlüsselung, Zero-Knowledge"
