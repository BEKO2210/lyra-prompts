# Dateien sortieren

## Prompt

```
Rolle: Digital-Organizer
Kontext: Mein [COMPUTER / HANDY / CLOUD] ist ein Chaos
Hauptprobleme: [ZU VIELE DATEIEN / KEINE STRUKTUR / DUPLIKATE / NICHTS WIEDERFINDEN]
Wichtige Kategorien: [ARBEIT / PRIVAT / FOTOS / DOKUMENTE]
Aufgabe: Erstelle ein Ordnungssystem
Ausgabe:
1. Ordnerstruktur (maximal 3 Ebenen)
2. Namenskonvention (einheitlich)
3. Aufräum-Plan (30 Min pro Tag für eine Woche)
4. Backup-Strategie
5. Pflege-Routine (wöchentlich/monatlich)
```

## Beispiel

**Input:** Computer, keine Struktur, Duplikate, Arbeit + Privat

**Output:** 
- Struktur: 01_Arbeit, 02_Privat, 03_Medien
- Namen: YYYY-MM-DD_Projekt_Beschreibung
- Plan: Tag 1 Downloads, Tag 2 Desktop, etc.

## Variationen

- Für Fotos: "Chronologisch + Events"
- Für Cloud: "Google Drive / Dropbox optimieren"
- Für E-Mails: "Inbox Zero"
