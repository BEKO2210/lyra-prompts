## Prompt

```
Rolle: IT-Sicherheitsberater spezialisiert auf Datensicherung für Privatpersonen
Kontext: Ich habe keine richtige Backup-Strategie und möchte meine wichtigen Daten schützen
Meine Daten:
- Wichtigste Daten: [z.B. Fotos, Dokumente, Steuern, Musik, Projekte]
- Datenmenge: [z.B. 500GB Fotos, 50GB Dokumente]
- Geräte: [z.B. PC, Laptop, Smartphone, Tablet]
- Aktuelles Backup: [KEINS / USB-STICK / CLOUD / EXTERNE FESTPLATTE]
- Budget: [monatlich für Cloud oder einmalig für Hardware]
Aufgabe: Erstelle eine 3-2-1-Backup-Strategie für meine persönlichen Daten
Einschränkungen:
- 3-2-1-Regel erklären und umsetzen
- DSGVO-konforme Cloud-Anbieter bevorzugen
- Automatische Backups einrichten (kein manuelles Kopieren)
Ausgabe: Backup-Plan mit Anbieter-Vergleich, Einrichtungsanleitung und Testplan
```

## Anwendung

**Für:** Alle die ihre Daten vor Verlust schützen wollen

**Input:** "200GB Fotos, 30GB Dokumente, Windows-PC + iPhone, kein Backup, 5€/Monat Budget"

**Output:** 3-2-1-Plan: Lokale Festplatte + Cloud (pCloud/iCloud) + automatische Sync-Einrichtung

## Variationen

- **Foto-Sicherung:** Alle Fotos von Smartphone und Kamera sicher archivieren
- **NAS-Einsteiger:** Eigenen Netzwerkspeicher einrichten statt Cloud
- **Notfall-Recovery:** Was tun wenn die Festplatte stirbt?
- **Familien-Backup:** Zentrale Datensicherung für die ganze Familie
