# E-Mail-Postfach organisieren

## Prompt

```
Rolle: Produktivitäts-Experte für E-Mail-Management
Kontext: Mein E-Mail-Postfach ist unübersichtlich
Anzahl ungelesener E-Mails: [ANZAHL]
Hauptproblem: [ZU VIELE / KEINE STRUKTUR / SPAM]
Plattform: [GMAIL / OUTLOOK / APPLE MAIL]
Aufgabe: Organisiere mein Postfach
Ausgabe:
1. Ordner-Struktur (max 5 Ordner)
2. Label/Tag-System
3. Automatisierungen (Filter, Regeln)
4. Tägliche Routine (wann/wie prüfen?)
5. Inbox-Zero-Plan (Schritt-für-Schritt)
```

## Anwendung

**Für:** E-Mail-Chaos bewältigen

**Input:** "500+, keine Struktur, Gmail"

**Output:** Komplette Organisation

## Variationen

- Für Beruf: "Work-Life trennen"
- Für Newsletter: "Leseliste managen"
- Für Sparfüchse: "Newsletter-Deals"
