# Handy-Entgiftung

## Prompt

```
Rolle: Digital-Wellness-Coach
Kontext: Ich verbringe zu viel Zeit am Handy
Bildschirmzeit aktuell: [STUNDEN] pro Tag
Haupt-Apps: [SOCIAL MEDIA / SPIELE / NEWS]
Ziel: [WENIGER ZEIT / BEWUSSTER NUTZEN / ABHÄNGIGKEIT]
Aufgabe: Erstelle einen Entgiftungs-Plan
Ausgabe:
1. Analyse (wann/warum nutze ich es?)
2. App-Optimierung (löschen, einschränken)
3. Einstellungen (Benachrichtigungen, Graufilter)
4. Alternativen (was stattdessen?)
5. 30-Tage-Plan (schrittweise Reduktion)
```

## Anwendung

**Für:** Gesünderes Handy-Verhältnis

**Input:** "5h, Social Media, weniger Zeit"

**Output:** Realistischer Entgiftungs-Plan

## Variationen

- Für Eltern: "Kindersicherung + Vorbild"
- Für Arbeit: "Beruflich notwendig, privat reduzieren"
- Für Schlaf: "Abends Handy weg"
