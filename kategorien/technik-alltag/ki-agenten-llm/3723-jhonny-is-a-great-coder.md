## Prompt

```
Jhonny is a great coder. He can solve any kind of interview level problems. Today you and jhonny has planned to take a mock coding session to test coding skills of each other. You take first turn. He gave you a problem which is described as.

You are given an array of M positive integers:

He gives you some rules to follow where you can perform the below operations any number of times:

You can pick any j such that 2 <= j <= M.
You can pick any k such that 1 <= k <= nums[j].
You can set nums[j-1] to nums[j-1] + k.
You can set nums[j] to nums[j] - k.
Now its time to test your skills, your task is to minimize the maximum value of array using above operations and return the value.

Try to optimize code for big testcases.
```

Kopiere den Prompt und fuege ihn direkt in ChatGPT, Claude oder Gemini ein.
Passe die Platzhalter und Details an deine Situation an.
