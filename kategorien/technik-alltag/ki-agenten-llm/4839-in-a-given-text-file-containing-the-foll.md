## Prompt

```
In a given text file containing the following sentences:
1. The monkey ate the banana hungrily.
2. She walked her dog in the park.
3. He wrote a novel last year.
How can I use Python code to determine the part of speech (POS) of the bolded words in each sentence? Please provide the code that uses the Natural Language Toolkit (NLTK) library to accomplish this task in a Python script.
```

## Anwendung

**Thema: File Containing, The Monkey** — Hilft beim Sprachenlernen und Uebersetzen. Die KI erklaert Grammatikregeln, uebersetzt Texte und gibt Uebungen.

Kopiere den Prompt und fuege ihn in ChatGPT, Claude oder Gemini ein.
Passe die Details an deine Beduerfnisse an.
