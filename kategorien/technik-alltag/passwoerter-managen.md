# Passwörter managen

## Prompt

```
Rolle: Sicherheitsberater für Nicht-Techniker
Kontext: Ich will meine Online-Sicherheit verbessern
Aktuell: [GLEICHES PASSWORT / WENIGE VARIANTEN / POST-ITS / NICHTS]
Geräte: [COMPUTER / HANDY / TABLET]
Aufgabe: Erstelle ein sicheres, einfaches System
Ausgabe:
1. Passwort-Manager Empfehlung (kostenlos + premium)
2. Umzugsplan (alte Passwörter sicher ändern)
3. 2FA einrichten (wo unbedingt nötig)
4. Notfallplan (Was wenn ich ausgesperrt bin?)
5. Zukunft: Wie bleibe ich sicher?
```

## Beispiel

**Input:** Gleiches Passwort überall, Computer + Handy

**Output:** 
- Manager: Bitwarden (kostenlos) oder 1Password
- Umzug: Top 10 wichtigste Accounts zuerst
- 2FA: Bank, E-Mail, Social Media

## Variationen

- Für Anfänger: "Einfachster Einstieg"
- Für Fortgeschrittene: "Selbst gehostete Lösungen"
- Für Unternehmer: "Team-Passwort-Management"
