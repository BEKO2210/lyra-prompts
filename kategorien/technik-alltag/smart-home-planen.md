# Smart Home planen

## Prompt

```
Rolle: Smart-Home-Berater für Einsteiger
Kontext: Ich will mein Zuhause smart machen
Wohnung: [1-ZIMMER / WOHNUNG / HAUS]
Budget: [BETRAG]
Priorität: [BELEUCHTUNG / HEIZUNG / SICHERHEIT / KOMFORT]
System: [ALEXA / GOOGLE / APPLE / EGAL]
Aufgabe: Erstelle einen Smart-Home-Plan
Ausgabe:
1. Prioritäten-Liste (was zuerst?)
2. Geräte-Empfehlungen (Lampen, Thermostate, etc.)
3. Kompatibilitätscheck (funktioniert alles zusammen?)
4. Installations-Guide (DIY vs. Profi)
5. Automationen (Routinen, Szenen)
```

## Anwendung

**Für:** Smart-Home-Einstieg

**Input:** "2-Zimmer, 500€, Beleuchtung, Alexa"

**Output:** Schritt-für-Schritt Smart-Home-Plan

## Variationen

- Für Mieter: "Ohne Umbau, Steckdosen-Lösungen"
- Für Hausbesitzer: "Installation, Elektriker"
- Für Sparfüchse: "Günstige Alternativen"
