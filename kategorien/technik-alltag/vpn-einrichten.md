# VPN einrichten

## Prompt

```
Rolle: IT-Sicherheitsexperte für Privatanwender
Kontext: Ich will ein VPN für [ZWECK] einrichten
Nutzen: [PRIVATSPHÄRE / STREAMING / ÖFFENTLICHES WLAN / ARBEIT]
Geräte: [COMPUTER / HANDY / ROUTER / ALLES]
Budget: [KOSTENLOS / BIS 5€ / BIS 15€]
Technik-Level: [ANFÄNGER / FORTGESCHRITTEN]
Aufgabe: Empfiehl und richte ein VPN ein
Ausgabe:
1. Anbieter-Vergleich (nach deinen Kriterien)
2. Einrichtungsanleitung (Schritt-für-Schritt)
3. Automatisierung (Auto-Connect, Kill-Switch)
4. Test (funktioniert es? IP-Leak-Test)
5. Tipps (wann nutzen? was beachten?)
```

## Anwendung

**Für:** Sicheres und privates Internet

**Input:** "Privatsphäre+Streaming, Computer+Handy, bis 5€, Anfänger"

**Output:** VPN-Einrichtung

## Variationen

- Für Reisen: "Ausländisches Streaming, Zensur"
- Für Arbeit: "Remote-Zugriff, Firmennetzwerk"
- Für Torrent: "Anonymität, keine Logs"
