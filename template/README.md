# Lyra Prompt Template

## Anleitung

1. Kopiere `TEMPLATE.md`
2. Benenne um zu: `dein-titel.md`
3. Ersetze alle `[PLATZHALTER]`
4. Speichern
5. In die passende Kategorie verschieben

## Kategorien-Übersicht

| Ordner | Für Prompts über... |
|--------|---------------------|
| `alltag-leben/` | Haushalt, Kochen, Finanzen, Planung |
| `beruf-karriere/` | Bewerbung, Gehalt, Meetings, Karriere |
| `lernen-wachstum/` | Skills, Ziele, Bücher, Entscheidungen |
| `kommunikation-beziehungen/` | Gespräche, Konflikte, Networking |
| `kreativitaet-freizeit/` | Schreiben, Reisen, Hobbys |
| `gesundheit-wohlbefinden/` | Fitness, Schlaf, Mental Health |
| `technik-alltag/` | Organisation, Sicherheit, Digital |
| `spezielle-situationen/` | Umzug, Feiern, Lebensereignisse |
| `bildbearbeitung-visualisierung/` | KI-Bilder, Midjourney, DALL-E |
| `pro/` | Premium-Content (PIN-geschützt) |

## Beispiel-Frontmatter

```yaml
---
titel: Rezept anpassen
kategorie: Alltag & Leben
unterkategorie: Küche & Kochen
tags: [kochen, rezept, meal-prep]
erstellt: 2026-02-21
plattformen: [ChatGPT, Claude]
---
```

## Naming Convention

- Kleinbuchstaben
- Bindestriche statt Leerzeichen
- Keine Sonderzeichen
- Beispiel: `rezept-anpassen.md`

## Tags

Wähle 3-5 relevante Tags aus:
- Alltag: kochen, haushalt, finanzen, planung
- Beruf: bewerbung, karriere, gehalt, meeting
- Lernen: lernen, skill, ziele, buch
- Kommunikation: gespräch, konflikt, beziehung
- Kreativ: schreiben, reise, hobby, foto
- Gesundheit: fitness, schlaf, stress, ernährung
- Technik: digital, sicherheit, organisation
- Spezial: umzug, feier, hochzeit
- Bild: midjourney, dalle, portrait, kunst
